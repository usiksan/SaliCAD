color

Содержание=>Скрипт 3d=>Функции 3d=>color

Формат вызова

Color color( Float red, Float green, Float blue, Float alpha )

Описание

Эта функция строит цвет по его компонентам. Каждый компонент (красный, зеленый, синий, альфа-канал) представлен собственным параметром от 0 до 1.0.

Параметры

  • red - красная компонента 0-1.0
  • green - зеленая компонента 0-1.0
  • blue - голубая компонента 0-1.0
  • alpha - альфа-канал (прозрачность) 0-1.0, где 0 - полностью прозрачный, 1.0 - совсем непрозрачный

Пример применения

bodyColor1 = color( 0.2, 0.1, 0.4, 1.0 )
