faceListWall

Content=>Script 3d=>Functions 3d=>faceListWall

Prototype

FaceList faceListWall( Face src, Face dst, Bool close )

Description

This function builds walls between the src and dst surfaces. The number of points in the source surfaces must be the same. In fact, the dst surface is usually obtained by some transformation of the src surface. The close parameter determines whether to build a wall between the end and start points.

Parameters

  • src - source surface (floor)
  • dst - target surface (ceiling)
  • close - when true, a wall is built between the last and first points, otherwise, this wall is not built and the figure turns out to be open

Example

color1 = selectColor("#a00000")

c1 = faceCircleSide( 3, 6, matrix1 )
c2 = faceDuplicateShift( c1, 4 )

h1 = faceListWall( c1, c2, true )

partModel = model( color1, color1, color1, h1, matrix1 )
