Contents=>Modes=>Part specific modes=>Show-hide pads

Tool bar:

Show-hide pin pads.

SaliCAD system for each pin can specify a description of the pad corresponding to that pin. Based on this description the graphical editors SaliCAD build a picture of the pin pads. When the pad display is enabled, its image is displayed along with the output. When the pad display is disabled, the pads are not visible, i.e. the pins are shown without pads.
