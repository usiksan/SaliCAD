Вставка фрагмента символа

Содержание=>Режимы=>Специальные режимы символа=>Вставка фрагмента символа

Панель инструментов:

Меню: Символ|Вставка фрагмента символа

Курсор:

Служит для вставки библиотечных символов при создании новых символов. Таким образом, библиотечные символы могут служить основой или составными частями нового символа.

1. В диалоге "Ввод символов" (см.Диалоговое окно "Ввод символов") выберите символ для вставки.

2. Укажите точку вставки фрагмента символа. При этом копия символа вставляется в текущий символ.

3. Повторяйте шаг 2 для вставки нескольких копий фрагмента.

4. Для прекращения ввода данной копии символа и возврата к выбору символа нажмите правую кнопку мыши.

5. Повторяйте шаги 1-4 для ввода различных фрагментов символов в текущий символ.

  • Указание точки вставки осуществляется нажатием левой кнопки мыши или вручную.
  • Прекращение ввода осуществляется нажатием правой кнопки мыши.

"Разумный" режим отсутствует.
