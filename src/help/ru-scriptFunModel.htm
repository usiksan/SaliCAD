model

Содержание=>Скрипт 3d=>Функции 3d=>model

Формат вызова

Model model( Color ambient, Color diffuse, Color specular, FaceList faces, Matrix transfer )

Описание

Эта функция строит модель на основе параметров.

Параметры

  • ambient - цвет модели
  • diffuse - цвет модели
  • specular - цвет модели
  • faces - набор поверхностей модели
  • transfer - матрица преобразования для первой копии модели

Пример применения

color1 = selectColor("#a00000")

c1 = faceListBox( 2, 3, -4 )

partModel = model( color1, color1, color1, c1, matrix1 )
