faceCurveXZ

Content=>Script 3d=>Functions 3d=>faceCurveXZ

Prototype

Face faceCurveXZ( Face src, Float radius, Float angleSrc, Float angleDst)

Description

This function performs a rotation in the XZ plane of the original surface. For the initial surface, the initial angle is set. For the resulting surface, the rotation radius and the resulting angle are set. This function is mainly used for constructing curved leads.

Parameters

  • src - initial surface
  • radius - turning radius from the center of mass of the initial surface
  • angleSrc - angle of the initial surface
  • angleDst - angle of the resulting surface

Example

#Example for rotation
color1 = selectColor("#800000")
color2 = selectColor("#008000")

c1 = faceCircle( 2, 10, matrix1 )

c2 = faceCurveXZ( c1, 4, 0, 35 )

partModel = model( color1, color1, color1, [c1], matrix1 )
partModel = model( color2, color2, color2, [c2], matrix1 )
