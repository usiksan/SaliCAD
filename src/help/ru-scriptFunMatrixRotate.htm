matrixRotate

Содержание=>Скрипт 3d=>Функции 3d=>matrixRotate

Формат вызова

Matrix matrixRotate( Matrix src, Float angleGrad, Float rotateX, Float rotateY, Float rotateZ )

Описание

Эта функция строит строит матрицу поворота исходной матрицы. На основании параметров angleGrad, rotateX, rotateY и rotateZ выполняется поворот исходной матрицы src. Поворот выполняется на угол равный angleGrad вокруг вектора, который определяется значения rotate.

Параметры

  • src - исходная матрица, к которой добавляется поворот
  • angleGrad - угол поворота в градусах вокруг вектора rotateX, rotateY, rotateZ
  • rotateX - x-компонент вектора поворота
  • rotateY - y-компонент вектора поворота
  • rotateZ - z-компонент вектора поворота

Пример применения

matrix2 = matrixRotate( matrix1, 90, 1, 0, 0 )

Строит новую матрицу путем добавления к исходной единичной матрице matrix1 поворота вокруг оси X на 90 градусов.
