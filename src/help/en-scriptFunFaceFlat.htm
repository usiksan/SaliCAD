faceFlat

Content=>Script 3d=>Functions 3d=>faceFlat

Prototype

Face faceFlat( Vertex origin, Float orientation, FloatList pairs )

Description

This function builds a polygon in the XY, YZ or XZ planes according to the given relative two-dimensional coordinates. The vertex of the beginning of the polygon is determined by the origin parameter. The orientation parameter specifies the plane in which the construction will be performed. The pairs parameter contains a list of pairs of relative coordinates of the vertices of the polygon. Each next vertex is defined as an offset relative to the previous one. The first pair defines the offset relative to the origin vertex. In order for a polygon to be used as a surface, it must be convex. If the polygon is non-convex, then it must be converted to a set of convex polygons using the function faceListSimplify.

Parameters

  • origin - the initial vertex of the polygon
  • orientation - the orientation code of the surface. 0 - XY plane, 1 - YZ plane, 2 - XZ plane
  • pairs - a list of pairs of relative coordinates. [ offx0,offy0,  offx1,offy1,  offx2,offy2 ... ]

Example

color1 = selectColor("#800000")

c1 = faceFlat( vertex(0,0,0), 0, [ 0,2, 1,1, 1,-1, 0,-2] )

partModel = model( color1, color1, color1, [c1], matrix1 )
