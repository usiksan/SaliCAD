Диалог Конструктор слоя

В этом диалоговом окне Вы можете сконструировать новый слой (см.слои).

Конструирование слоя возможно с помощью встроенных класса и модификатора, а также путем задания произвольного идентификатора.

Наиболее предпочтительным является использование встроенных класса и модификатора. Важно! Трассировочный слой может быть сконструирован только с помощью встроенного модификатора.

Чтобы сконструировать слой выберите класс слоя, а затем необязательный модификатор. Если Вы не хотите использовать модификатор, т.е. достаточно только класса, то в качестве модификатора выберите "---". Сконструированный таким образом идентификатор слоя отображается в поле ID слоя.

В поле Имя слоя отображается имя слоя, соответствующее текущему идентификатору. Вы можете задать наименование слоя на национальном языке. Это наименование сохранится на Вашем локальном компьютере и будет отображаться вместо идентификатора.

Чтобы создать произвольный идентификатор вместо встроенных класса и модификатора выберите в качестве класса Пользовательский ID, который расположен в конце списка классов.
