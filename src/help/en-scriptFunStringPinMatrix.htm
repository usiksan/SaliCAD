stringPinMatrix

Content=>Script 3d=>Functions 3d=>stringPinMatrix

Prototype

String stringPinMatrix( Float row, Float column )

Description

The function builds a textual representation of the matrix output number, for example, for bga from the row and column numbers. As a result, the rows are converted to letters, so 0 corresponds to A, 1 - B, etc.

Parameters

  • row - row number. As a result, the rows are converted to letters, so 0 corresponds to A, 1 - B, etc.
  • column - column number

Example

  pin = stringPinMatrix( 3, 5 )

This example will generate the pin number "D5"
