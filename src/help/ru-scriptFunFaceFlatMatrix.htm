faceFlatMatrix

Содержание=>Скрипт 3d=>Функции 3d=>faceFlatMatrix

Формат вызова

Face faceFlatMatrix( FloatList pairs, Matrix matrix )

Описание

Эта функция строит многоугольник на плоскости по заданным относительным двумерным координатам в плоскости XY. Вершина начала многоугольника определяется первой парой. Параметр pairs содержит список пар относительных координат вершин многоугольника. Каждая следующая вершина определяется как смещение относительно предыдущей. Первая пара определяет координаты XY первой вершины. Для того, чтобы многоугольник мог использоваться в качестве поверхности он должен быть выпуклым. Если многоугольник невыпуклый, то его необходимо преобразовать к набору выпуклых многоугольников с помощью функции faceListSimplify. Результирующий многоугольник может быть преобразован с помощью матрицы.

Параметры

  • pairs - пара, определяющая первую вершину и список пар относительных координат. [ org0,org0,  offx1,offy1,  offx2,offy2 ... ]
  • matrix - матрица преобразования результирующего многоугольника

Пример применения

color1 = selectColor("#800000")

c1 = faceFlatMatrix( [ 0,0, 0,2, 1,1, 1,-1, 0,-2], matrix1 )

partModel = model( color1, color1, color1, [c1], matrix1 )
