stringPinMatrix

Содержание=>Скрипт 3d=>Функции 3d=>stringPinMatrix

Формат вызова

String stringPinMatrix( Float row, Float column )

Описание

Функция строит текстовое представление номера матричного вывода, например для bga из номера ряда и колонки. В результате ряды преобразуются в буквы, так 0 соответствует A, 1 - B и т.д.

Параметры

  • row - номер ряда. В результате ряды преобразуются в буквы, так 0 соответствует A, 1 - B и т.д.
  • column - номер колонки

Пример применения

  pin = stringPinMatrix( 3, 5 )

Данный пример сформирует номер вывода "D5"
