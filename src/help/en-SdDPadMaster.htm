Dialog Pad Master

This dialog box contains the program settings grouped by pages.

  Page Colors


This dialog box contains a wizard for creating and editing pads description. see Pads

In the left part of the window - a list of parameters of the contact area, and on the right - preview area: in the section and top view.

When you change any parameter, the appearance of the site is automatically rebuilt, showing its actual appearance.

Name Description
Round platform or rectangular
 If this parameter is checked, it will be formed round area. Otherwise - rectangular.
Through pin pad or planar (smt) Through pin pad is designed for through pins and vias. It is characterized by the presence of holes and the absence of holes in the stencil for her. There are no holes in the planar pad, but there is a stencil for applying solder paste. If given the parameter is marked, then the Through pin pad will be formed. In otherwise - planar.
Pad center X The Pad is constructed symmetric with respect to the center. By default the center of the pad is the same as the pin position. Sometimes, depending on the pin configuration, pad needs to be shifted. This parameter sets the value such an offset along the x-axis.
site Center Y The Pad is constructed symmetric with respect to the center. By default the center of the pad is the same as the pin position. Sometimes, depending on the pin configuration, pad needs to be shifted. This parameter sets the value such an offset along the y-axis.
Pad diameter For a round pad, its diameter is specified here.
Pad width For a rectangular area, the width of the pad is set here. X-Axis.
Pad height For a rectangular area, the height of the pad is set here. Y Axis.
Solder mask threshold The area of absence of the soldering mask has the same contour as pad. The gap determines how much this area is larger (smaller), than a pad. In practice, this area is somewhat larger pad.
Stencil hole threshold The Stencil for solder paste consists of holes through which the paste is applied to the pad. The position and shape of the holes is typically coincide with the pads. This parameter determines how much this the area is smaller (larger) than the pad. In practice, the hole stencil slightly smaller than the pad.
Stencil holes array rows If the pad is too big, then a hole in a stencil divide on an array of small holes. This parameter shows the number of rows of holes in such an array.
Stencil holes array columns If the pad is too big, then a hole in a stencil divide on an array of small holes. This parameter shows the number of columns of holes in such an array.
Stencil holes array cell width If the pad is too big, then a hole in a stencil divide on an array of small holes. This parameter shows the width each of the small holes in such an array.
Stencil holes array cell height If the pad is too big, then a hole in a stencil divide on an array of small holes. This parameter shows the height each of the small holes in such an array.
Hole diameter Hole Diameter for part pin or vias.
