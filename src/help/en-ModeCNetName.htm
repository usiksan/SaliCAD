Insert net name

Contents=>Modes=>Sheet specific modes=>Insert net name

Tool bar:

Menu: Sheet|Insert net name

Cursor:

Serves for placing on the diagram the names of the chains. Change the names of circuits using this mode impossible. Instead, you must use select mode or chain insert mode for this purpose.

1. Specify the net whose name you want to insert

2. Setup net name properties

3. Specify the location of the net name

4. Repeat steps 1-3 until you have affixed the names of all the desired nets.

  • You can specify a point by clicking the left mouse button or manually.
  • The termination of the input is done by pressing the right mouse button.

Net name properties are set using the text properties bar.

"Smart mode" not present.
