faceRect

Content=>Script 3d=>Functions 3d=>faceRect

Prototype

Face faceRect( Float length, Float width, Matrix transfer )

Description

This function builds a surface in the form of a rectangle. The resulting rectangle can be transformed using a matrix.

Parameters

  • length - rectangle length X
  • width - rectangle width Y
  • transfer - final rectangle transformation

Example

color1 = selectColor("#800000")

r1 = faceRect( 2, 1, matrix1 )

partModel = model( color1, color1, color1, [r1], matrix1 )
