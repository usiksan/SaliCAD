faceListWalls

Содержание=>Скрипт 3d=>Функции 3d=>faceListWalls

Формат вызова

FaceList faceListWalls( FaceList floors, Bool close )

Описание

Эта функция строит стены между этажами, представленными набором поверхностей. Количество точек в поверхностях должно быть одинаковым. Параметр close определяет, нужно ли строить стену между конечной и начальной точками каждой поверхности.

Расположение этаже - произвольное, а не только горизонтальное. Поэтому с помощью данной функции могут быть получены тела вращения, как показано в примере.

Параметры

  • floors - список поверхностей этажей
  • close - когда true строится стена между последней и первой точками, иначе - эта стена не строится и фигура получается разомкнутой

Пример применения

color1 = selectColor("#a00000")

pin = faceListRotation( 0, 80, 20, matrix1, [ 0.5,0, 0.5,0, 0,1, -0.5,0 ] )

pin = faceListWalls( pin, false )

partModel = model( color1, color1, color1, pin, matrix1 )
