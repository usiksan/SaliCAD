Перенос идентификатора

Содержание=>Режимы=>Плата=>Перенос идентификатора

Панель инструментов:

Меню: Плата|Перенос идентификатора

Курсор:

Служит для переноса идентификаторов компонента в плате. Каждый идентификатор имеет положение по умолчанию. Оно задается относительно компонента, поэтому когда компонент переносится или поворачивается идентификатор переносится и поворачивается вместе с ним. Чтобы изменить положение идентификатора относительного компонента служит данный режим.

1. Укажите компонент, идентификатор которого нужно перенести

2. Установите атрибуты идентификатора

3. Укажите местоположение идентификатора

4. Повторяйте шаги 1-3 пока не будут расставлены все идентификаторы

  • Указание точки осуществляется нажатием левой кнопки мыши или вручную.
  • Прекращение ввода осуществляется нажатием правой кнопки мыши.

Атрибуты идентификатора задаются с помощью панели свойств текста.

"Разумный режим" отсутствует.
