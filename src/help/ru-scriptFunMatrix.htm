matrix

Содержание=>Скрипт 3d=>Функции 3d=>matrix

Формат вызова

Matrix matrix( Float angleGrad, Float rotateX, Float rotateY, Float rotateZ, Float offX, Float offY, Float offZ )

Описание

Эта функция строит строит матрицу поворота и перемещения. На основании параметров angleGrad, rotateX, rotateY и rotateZ выполняется поворот, а на основании параметров offX, offY и offZ - перемещение. Поворот выполняется на угол равный angleGrad вокруг вектора, который определяется значения rotate. Перемещение определяется вектором с координатами off.

Параметры

  • angleGrad - угол поворота в градусах вокруг вектора rotateX, rotateY, rotateZ
  • rotateX - x-компонент вектора поворота
  • rotateY - y-компонент вектора поворота
  • rotateZ - z-компонент вектора поворота
  • offX - x-компонент вектора перемещения
  • offY - y-компонент вектора перемещения
  • offZ - z-компонент вектора перемещения

Пример применения

matrix2 = matrix( 90, 1, 0, 0, 2, 0, 0 )

Выполняет поворот вокруг оси X на 90 градусов, а затем сдвигает на 2 по этой-же оси
