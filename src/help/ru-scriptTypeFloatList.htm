Тип набор чисел с плавающей точкой

Содержание=>Скрипт схем=>Тип набор чисел с плавающей точкой

Содержание=>Скрипт 3d=>Тип набор чисел с плавающей точкой

Несколько значений с плавающей точкой могут быть объединены в наборы или массивы. Эти наборы используются для задания двумерных координат, индексов и т.п.

Для создания наборов используется операция []. Например,

far1 = [ 1.0, 2.0, 900, -1]

При этом переменной far1 присваивается набор из четырех значений.

Набор может использоваться внутри другого набора, при этом все элементы первого набора копируются во второй.

far2 = [ 10.9, far1, -100]

В результате набор far2 будет состоять из шести значений: значение 10.9, затем элементы набора far1, затем -100.

Важно! Вставка не может быть в первую позицию, т.е.

far3 = [far1, 50] #Ошибка!

Для наборов не допускаются никакие операции. Сами наборы используются в вызовах функций и данные, размещаемые в таких наборах определяются соответствующим параметром функции.
