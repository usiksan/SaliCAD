Grid dialog

In this dialog defined parameters and properties graphics editor grid.

Step grid parameters group

On axiz X define grid cell size on axiz X

On axiz Y define grid cell size on axiz Y

If checked Sync X and Y, then cell size of axiz Y changed syncronous with axiz X

In list in the central part of dialog there are previous entered grids. You can select one of this list instead enter by handle.

Display grid if this checked, then in graphics editors display grid.

Align cursor if this checked, then mouse movement aligned to grid.
