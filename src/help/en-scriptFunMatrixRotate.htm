matrixRotate

Content=>Script 3d=>Functions 3d=>matrixRotate

Prototype

Matrix matrixRotate( Matrix src, Float angleGrad, Float rotateX, Float rotateY, Float rotateZ )

Description

This function builds builds the rotation matrix of the original matrix. Based on the parameters angleGrad, rotateX, rotateY and rotateZ, the rotation of the original src matrix is performed. Rotation is performed at an angle equal to angleGrad around the vector, which is determined by the value of rotate.

Parameters

  • src - the original matrix to which the rotation is added
  • angleGrad - rotation angle in degrees around the vector rotateX, rotateY, rotateZ
  • rotateX - x-component of the rotation vector
  • rotateY - y-component of the rotation vector
  • rotateZ - z-component of the rotation vector

Example

matrix2 = matrixRotate( matrix1, 90, 1, 0, 0 )

Builds a new matrix by adding 90 degrees rotation around the X axis to the original identity matrix matrix1.
