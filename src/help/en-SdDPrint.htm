Print Dialog

Use this dialog box to set print options and print the contents of the current editor.

The Printer field contains the name of the printer to be printed on. Clicking the Options button opens a dialog box where you can configure the settings for the current printer or select another printer from the list of available printers. When you install the appropriate driver, you can print to a pdf file.

The Print manage group of options allows you to specify the exact scale of the image output to print or to stretch the image to the entire sheet. By default, the image is stretched to the entire sheet.

The Orientation group determines the vertical or horizontal position of the picture. By default, the orientation is selected to fill the sheet as much as possible.

The Print area group allows you to select the rectangular area of the drawing to be printed. By default, the entire image is printed. If a part is selected in the picture, only the selected part can be printed. You can also specify the coordinates of the window whose contents will be printed. When you click Select, the print dialog box is hidden and the mode is enabled so that you can specify the dialog box directly in the image.

The group of other parameters contains:

  • copy count - here you can see the number of copies to be printed
  • zero line width - on the drawings of circuits and circuit boards some of the lines have the thickness equal to zero. When printing, the thickness of such lines is taken to be equal to the size of one point of the printer. For printers with high resolution, these lines may be too thin. In this field, you can specify the thickness for such lines in pixels (points) of the printer
  • color print - by default, the picture is printed in black and white. That is, any elements are printed in black, and the background remains white, even if the printer allows color printing. By setting this flag, you can get a color printout on a color printer
