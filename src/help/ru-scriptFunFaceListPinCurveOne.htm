faceListPinCurveOne

Содержание=>Скрипт 3d=>Функции 3d=>faceListPinCurveOne

Формат вызова

FaceList faceListPinCurveOne( Face src, Float firstLen, Float middleLen, Float radius, Float angle, Float sideCount )

Описание

Эта функция строит вывод по его основанию. На первом этапе основание вытягивается вверх на величину firstLen, затем загибается на угол angle по радиусу radius, при этом для загиба используются sideCount поверхностей для аппроксимации кривой и, наконец, происходит вытяжка на величину middleLen. Обычно эта функция используется для построения выводов, но может использоваться также и для построения других тел.

Параметры

  • src - исходная поверхность
  • firstLen - величина вытяжки вверх
  • middleLen - величина финальной вытяжки
  • radius - радиус загиба
  • angle - угол загиба
  • sideCount - количество сторон для аппроксимации кривой загиба

Пример применения

color1 = selectColor("#a00000")

c1 = faceRect( 0.2, 1, matrix1 )

pin = faceListPinCurveOne( c1, 2, 3, 1, 90, 10 )

partModel = model( color1, color1, color1, pin, matrix1 )
