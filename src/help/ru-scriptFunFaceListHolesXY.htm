faceListHolesXY

Содержание=>Скрипт 3d=>Функции 3d=>faceListHolesXY

Формат вызова

FaceList faceListHolesXY( Face src, FaceList holes )

Описание

Эта функция преобразует поверхность и отдельно взятые отверстия в набор поверхностей, которые представляет собой исходную поверхность со встроенными в нее отверстиями, например для формирования штыревой контактной площадки. Для корректной работы функции необходимо, чтобы исходная поверхность и отверстия располагались на одной плоскости XY. Кроме того, вершины исходной поверхности и отверстий должны иметь порядок обхода вершин по часовой стрелке.

Параметры

  • src - исходная поверхность
  • holes - список поверхностей-отверстий

Пример применения

color1 = selectColor("#a00000")

src = faceRect( 5, 5, matrix1 )
hole1 = faceRect( 1, 1, matrixTranslate( matrix1, -1, -1, 0 ) )
hole2 = faceRect( 1, 1, matrixTranslate( matrix1, 1, 1, 0 ) )
body = faceListHolesXY( src, [hole1,hole2] )

partModel = model( color1, color1, color1, body, matrix1 )
