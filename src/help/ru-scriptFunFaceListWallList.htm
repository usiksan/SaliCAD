faceListWallList

Содержание=>Скрипт 3d=>Функции 3d=>faceListWallList

Формат вызова

FaceList faceListWallList( FaceList src, FaceList dst, Bool close )

Описание

Эта функция строит стены между наборами поверхностей src и dst. Количество точек в соответствующих исходных поверхностях должно быть одинаковым, количество поверхностей в исходных наборах также должно быть одинаковым. Параметр close определяет, нужно ли строить стену между конечной и начальной точками каждой поверхности.

Параметры

  • src - набор исходных поверхностей (полы)
  • dst - набор целевых поверхностей (потолоки)
  • close - когда true строится стена между последней и первой точками каждой поверхности в наборе, иначе - эта стена не строится и фигуры получаются разомкнутыми

Пример применения

color1 = selectColor("#a00000")

cb1 = faceCircleSide( 3, 6, matrix1 )
ct1 = faceDuplicateShift( cb1, 4 )
cb2 = faceCircleSide( 3, 8, matrixTranslate( matrix1, 6, 0, 0 ) )
ct2 = faceDuplicateShift( cb2, 2 )

h1 = faceListWallList( [cb1,cb2], [ct1,ct2], true )

partModel = model( color1, color1, color1, h1, matrix1 )
