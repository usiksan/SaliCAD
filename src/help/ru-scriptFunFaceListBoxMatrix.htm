faceListBoxMatrix

Содержание=>Скрипт 3d=>Функции 3d=>faceListBoxMatrix

Формат вызова

FaceList faceListBoxMatrix( Float lenght, Float width, Float height, Matrix translate )

Описание

Эта функция строит параллелепипед с основанием на плоскости XY, а затем преобразует его с помощью матрицы.

Параметры

  • lenght - длина параллелепипеда (ось X)
  • width - ширина параллелепипеда (ось Y)
  • height - высота параллелепипеда (ось Z)
  • translate - матрица преобразования

Пример применения

color1 = selectColor("#a00000")

c1 = faceListBoxMatrix( 2, 3, -4, matrixRotate( matrix1, 45, 0, 1, 0 ) )

partModel = model( color1, color1, color1, c1, matrix1 )
