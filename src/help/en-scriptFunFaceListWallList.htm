faceListWallList

Content=>Script 3d=>Functions 3d=>faceListWallList

Prototype

FaceList faceListWallList( FaceList src, FaceList dst, Bool close )

Description

This function builds walls between sets of src and dst surfaces. The number of points in the corresponding source surfaces must be the same, the number of surfaces in the source sets must also be the same. The close parameter determines whether to build a wall between the end and start points of each surface.

Parameters

  • src - source list of surfaces (floors)
  • dst - набор целевых поверхностей (потолоки)
  • close - when true, a wall is built between the last and first points of each surface in the set, otherwise, this wall is not built and the figures are open

Example

color1 = selectColor("#a00000")

cb1 = faceCircleSide( 3, 6, matrix1 )
ct1 = faceDuplicateShift( cb1, 4 )
cb2 = faceCircleSide( 3, 8, matrixTranslate( matrix1, 6, 0, 0 ) )
ct2 = faceDuplicateShift( cb2, 2 )

h1 = faceListWallList( [cb1,cb2], [ct1,ct2], true )

partModel = model( color1, color1, color1, h1, matrix1 )
