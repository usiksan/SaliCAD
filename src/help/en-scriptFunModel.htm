model

Content=>Script 3d=>Functions 3d=>model

Prototype

Model model( Color ambient, Color diffuse, Color specular, FaceList faces, Matrix transfer )

Description

This function builds a model based on parameters.

Parameters

  • ambient - model color
  • diffuse - model color
  • specular - model color
  • faces - set of model surfaces
  • transfer - transformation matrix for the first copy of the model

Example

color1 = selectColor("#a00000")

c1 = faceListBox( 2, 3, -4 )

partModel = model( color1, color1, color1, c1, matrix1 )
