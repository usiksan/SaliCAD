Float list type

Content=>Schematic script=>Float list type

Content=>Script 3d=>Float list type

Multiple floating-point values can be combined into sets or arrays. These sets are used to set two-dimensional coordinates, indexes, etc.

The [] operation is used to create sets. For example,

far1 = [ 1.0, 2.0, 900, -1]

In this case, the variable var1 is assigned a set of four values.

A set can be used inside another set, while all elements of the first set are copied into the second.

far2 = [ 10.9, far1, -100]

В результате набор far2 будет состоять из шести значений: значение 10.9, затем элементы набора far1, затем -100.

Важно! Вставка не может быть в первую позицию, т.е.

far3 = [far1, 50] #Ошибка!

No operations are allowed for sets. The sets themselves are used in function calls and the data placed in such sets is determined by the corresponding function parameter.
