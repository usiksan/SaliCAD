faceRectSide

Содержание=>Скрипт 3d=>Функции 3d=>faceRectSide

Формат вызова

Face faceRectSide( Float lenght, Float width, Float sideCount, Matrix transfer )

Описание

Эта функция строит поверхность в виде прямоугольника состоящего из ломаной с указанным количеством ребер. Полученный прямоугольник можно преобразовать с помощью матрицы. Несмотря на количество ребер, фигура выглядит как прямоугольник.

Параметры

  • lenght - длина
  • width - ширина
  • sideCount - количество ребер многоугольника должно быть кратно 4-м
  • transfer - заключительное преобразование прямоугольника

Пример применения

color1 = selectColor("#800000")

r1 = faceRectSide( 2, 1, 40, matrix1 )

partModel = model( color1, color1, color1, [r1], matrix1 )
