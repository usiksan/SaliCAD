Вставка цепей

Содержание=>Режимы=>Специальные режимы схемы=>Вставка цепей

Панель инструментов:

Меню: Лист схемы|Вставить цепь

Курсор:

Вставляет цепь в лист схемы.

1. Укажите имя вновь вводимой цепи

2. Укажите/измените атрибуты линии

3. Укажите первую точку ломаной цепи

4. Укажите очередную точку ломаной цепи

5. Повторяйте шаги 2-4 для ввода всех сегментов ломаной цепи.

6. Для прекращения ввода цепи нажмите правую кнопку мыши.

Замечание: Если точка очередная точка цепи попадает на сегмент другой цепи, то выдается запрос на объединение двух цепей в одну. Если обе цепи имеют имена отличные от имен по умолчанию то дополнительно выдается диалог варианта объединения цепей.

Атрибуты и имя цепи задаются с помощью панели свойств цепи:

  • Указание точки осуществляется нажатием левой кнопки мыши или вручную.
  • Прекращение ввода осуществляется нажатием правой кнопки мыши.

"Разумный режим"

Для цепей имеется два уровня разумного режима.

На первом уровне при вводе первой точки разумный режим предлагает полное соединение в окрестностях курсора мыши. Стартовая точка может предлагаться как на неподключенном выводе, так и на цепи.

Пример "разумного режима" цепи. Система предлагает соединить вывод A с выводом B.

Пример "разумного режима" цепи. Система предлагает соединить сегмент существующей цепи A с выводом B.

На втором уровне при вводе очередной, но не первой точки цепи, эта точка предлагается на ближайшем свободном выводе при последнем движении курсора. Другими словами поиск свободного вывода ведется в направлении движения курсора мыши. При этом до данного вывода предлагается ломаная.

Средняя кнопка мыши или "пробел" - подтверждение "разумного" ввода.
