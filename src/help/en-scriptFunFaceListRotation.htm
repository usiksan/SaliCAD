faceListRotation

Content=>Script 3d=>Functions 3d=>faceListRotation

Prototype

FaceList faceListRotation( Float angleStart, Float angleStop, Float sideCount, Matrix translate, FloatList profilePairs )

Description

This function builds a set of rotated profiles based on the original profile. This function, together with faceListWalls, is used to build rotation surfaces. The initial profile is built on the XZ plane. Then a set of rotation points is formed by sequentially rotating the initial profile around the OZ axis by an angle equal to the angle step (the difference between the initial and final angles divided by the number of sides). The resulting set of points is transformed using a matrix.

Parameters

  • angleStart - the initial angle of the profile
  • angleStop - the final angle of the profile
  • sideCount - number of sides of the rotation approximation
  • translate - matrix of the resulting transformation
  • profilePairs - a set of pairs of values that define the vertices of the original profile. Each pair represents a vertex offset relative to the previous vertex

Example

color1 = selectColor("#a00000")

pin = faceListRotation( 0, 80, 20, matrix1, [ 0.5,0, 0.5,0, 0,1, -0.5,0 ] )

pin = faceListWalls( pin, false )

partModel = model( color1, color1, color1, pin, matrix1 )
