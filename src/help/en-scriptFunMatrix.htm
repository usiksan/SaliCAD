matrix

Content=>Script 3d=>Functions 3d=>matrix

Prototype

Matrix matrix( Float angleGrad, Float rotateX, Float rotateY, Float rotateZ, Float offX, Float offY, Float offZ )

Description

This function builds builds a rotation and displacement matrix. Based on the parameters angleGrard, rotateX, rotateY and rotateZ, rotation is performed, and based on the parameters offX, offY and offZ, movement is performed. Rotation is performed at an angle equal to angleGrad around the vector, which is determined by the value of rotate. The movement is determined by a vector with coordinates off.

Parameters

  • angleGrad - rotation angle in degrees around the vector rotateX, rotateY, rotateZ
  • rotateX - x-component of the rotation vector
  • rotateY - y-component of the rotation vector
  • rotateZ - z-component of the rotation vector
  • offX - x-component of the displacement vector
  • offY - y-component of the displacement vector
  • offZ - z-component of the displacement vector

Example

matrix2 = matrix( 90, 1, 0, 0, 2, 0, 0 )

Rotates around the X axis by 90 degrees, and then shifts by 2 along the same axis
