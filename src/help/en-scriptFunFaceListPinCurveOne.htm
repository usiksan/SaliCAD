faceListPinCurveOne

Content=>Script 3d=>Functions 3d=>faceListPinCurveOne

Prototype

FaceList faceListPinCurveOne( Face src, Float firstLen, Float middleLen, Float radius, Float angle, Float sideCount )

Description

This function builds the output based on its base. At the first stage, the base is pulled up by the value of firstLen, then it is bent by the angle angle along the radius radius, while sideCount surfaces are used for bending to approximate the curve and, finally, there is an extraction by the value of middleLen. Usually this function is used to build conclusions, but it can also be used to build other bodies.

Parameters

  • src - initial surface
  • firstLen - the value of the hood up
  • middleLen - the value of the final hood
  • radius - bend radius
  • angle - bend angle
  • sideCount - number of sides to approximate the bend curve

Example

color1 = selectColor("#a00000")

c1 = faceRect( 0.2, 1, matrix1 )

pin = faceListPinCurveOne( c1, 2, 3, 1, 90, 10 )

partModel = model( color1, color1, color1, pin, matrix1 )
