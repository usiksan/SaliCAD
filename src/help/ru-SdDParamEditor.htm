Диалог Редактор параметров

В этом диалоговом окне представлен список параметров объетка. Параметры можно добавлять, удалять и изменять их значения.

В центральной части диалога представлена таблица параметров. В левой колонке - наименование параметра, а в правой - значение параметра. Чтобы изменить значение параметра редактируйте соответствующую ячейку.

Добавить - при нажатии на эту кнопку выдается запрос на ввод имени параметра, после чего данный параметр добавляется в список.

По умолчанию - при нажатии на эту кнопку выдается диалог со списком параметров по умолчанию. Отметьте нужные параметры и при нажатии Ок они добавятся в список.

Копировать - служит для копирования параметров из другого объекта. При нажатии на эту кнопку выдается диалог поиска объекта. При выборе объекта и нажатии кнопки Ок параметры из него копируются в текущий объект замещая все исходные параметры.

Удалить - при нажатии на эту кнопку текущий выбранный параметр удаляется из списка.

Значение - позволяет выбрать значение из ряда. Ряд, минимальное и максимальное значения задаются параметрами. Эти параметры есть в списке по умолчанию.
