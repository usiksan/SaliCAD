selectPad

Content=>Script 3d=>Functions 3d=>selectPad

Prototype

String selectPad( String padString )

Description

This function is used for the dialog construction of the pin pad. If the cursor is on this function, pressing the F2 button causes the pin pad Wizard dialog box to be displayed. After constructing the site through a dialog, its text description is replaced in the script text.

Parameters

  • padString - text representation of the pin pad

Example

pin1Pad = selectPad( "c1d0.4m0.1" )
