For setup pin properties there is part pin properties bar:

  • You can specify a point by clicking the left mouse button or manually.
  • The termination of the input is done by pressing the right mouse button.

"Smart" mode.

When entering a pin location point, this point is suggested by the "smart" mode at the end of the nearest line segment.

When you enter the pin number and pin name location points, these points are suggested relative to the pin location point in the same way as the previous pin. This mode is particularly useful for serial pins where the relative position of the number and name is the same.

When you enter the number of the pin, the number of the previous pin is suggested with an index greater than 1, that is, if the previous number of the pin was, for example, A1, then the proposed number will be A2. This mode is convenient for sequential pins of part.

To enter pin name and pin number properties there is text properties bar.

Middle mouse button or "space" - confirmation of "smart" input.
