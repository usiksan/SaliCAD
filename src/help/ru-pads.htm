Площадки

В SaliCAD выводы корпусов характеризуются типом. Для привязки конкретному типу контактной площадки служит ассоциация контактных площадок. Она представляет собой таблицу соответствия типов выводов контактным площадкам. Такой подход позволяет изменять параметры контактных площадок в зависимости от технологии печатной платы, что дает гибкие возможности настройки.

Площадка конструируется системой SaliCAD автоматически на основе описания контактной площадки для каждого конкретного применения. Такое конструирование выполняется прозрачно для разработчика плат и ему не нужно совершать никаких дополнительных действий для этого. Например, для сквозного переходного отверстия для двухсторонней платы нужно сформировать две площадки: на верхнем и нижнем слоях. Для четырехслойной платы нужно сформировать уже четыре площадки. В системе SaliCAD вы описываете только одну площадку, а остальные формируются автоматически при необходимости.

Помимо собственно изображений на слоях трассировки площадки содержать описание отверстия (при необходимости), описание маски припоя и описание трафарета для печати паяльной пасты.

В текущей версии SaliCAD допускаются только отверстия круглой формы.

Паяльная маска повторяет контур контактной площадки с определенным зазором.

Отверстие трафарета для печати паяльной пасты в простейшем случае повторяет контур контактной площадки с определенным зазором. Если контактная площадка имеет большой размер, то рекомендуется разбить это большое отверстие на массив мелких отверстий. Описание контактной площадки позволяет такую возможность.
