faceEllipseSide

Content=>Script 3d=>Functions 3d=>faceEllipseSide

Prototype

Face faceEllipseSide( Float radiusX, Float radiusY, Float sideCount, Matrix transfer )

Description

This function constructs a surface in the form of an equilateral polygon inscribed in an ellipse. The number of sides of the polygon is determined by the parameter. The resulting polygon can be transformed using a matrix. This function is used to construct approximated ellipses. The greater the number of sides, the higher the approximation quality.

Параметры

  • radiusX - the radius X of the described ellipse around the polygon
  • radiusY - the radius Y of the described ellipse around the polygon
  • sideCount - side count
  • transfer -  final polygon transformation

Example

color1 = selectColor("#800000")

c1 = faceEllipseSide( 3, 2, 36, matrix1 )

partModel = model( color1, color1, color1, [c1], matrix1 )
