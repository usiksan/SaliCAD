faceListWallDoubleBevelXY

Содержание=>Скрипт 3d=>Функции 3d=>faceListWallDoubleBevelXY

Формат вызова

FaceList faceListWallDoubleBevelXY( Face src, Face dst, Float radiusBot, Float radiusTop, Float angleStepBot, Float angleStepTop, Float height )

Описание

Эта функция строит стены дву фаски, снизу и сверху, между поверхностями src и dst. Количество точек в исходных поверхностях должно быть одинаковым.  Различные комбинации знака в параметрах faceListWallDoubleBevelXY позволяют получить различные формы фасок.

Параметры

  • src - исходная поверхность
  • dst - целевая поверхность, полученная
  • radiusBot - радиус фаски снизу
  • radiusTop - радиус фаски сверху
  • angleStepBot - размер шага угла для аппроксимации кривой поверхности фаски снизу
  • angleStepTop - размер шага угла для аппроксимации кривой поверхности фаски сверху
  • height - высота между нижней и верхней фасками (внешний размер)

Пример применения

color1 = selectColor("#a00000")

c1 = faceCircleSide( 3, 56, matrix1 )
c2 = faceDuplicateShift( c1, 2 )

h1 = faceListWallDoubleBevelXY( c1, c2, 1, -1, -10, 10, 2 )

partModel = model( color1, color1, color1, h1, matrix1 )
