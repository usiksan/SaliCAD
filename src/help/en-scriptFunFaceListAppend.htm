faceListAppend

Content=>Script 3d=>Functions 3d=>faceListAppend

Prototype

FaceList faceListAppend( FaceList src, FaceList appended )

Description

This function returns a combined list of surfaces consisting of src and appended surfaces.

Parameters

  • src - source list of surfaces
  • appended - added list of surfaces

Example

color1 = selectColor("#a00000")

c1 = faceListCylinder( 2, 5 )
c2 = faceListDuplicate( c1, matrixTranslate( matrix1, 5, 0, 0 ) )
c3 = faceListAppend( c1, c2 )
partModel = model( color1, color1, color1, c3, matrix1 )
