Тип модель

Содержание=>Скрипт 3d=>Тип модель

Тип модель представляет собой комплексный объект образующий 3d модель.

Каждая модель состоит из списка тел, которые отображаются один или несколько раз в положении, заданном матрицей копирования. Каждая модель содержит минимум одну матрицу копирования. Например, изображение корпуса микросхемы будет содержать одну матрицу копирования, а изображение ножек - несколько, по количеству ножек у микросхемы.

Список тел состоит из одного или нескольких тел. Каждое тело представляет собой совокупность набора поверхностей изображаемых одним общим цветом. Таким образом, тело - это набор поверхностей, которые отображаются одним цветом.

С моделями доступна только операция присваивания. Также модели используются в вызовах функций. Модели можно создавать, можно добавлять тела к списку тел, можно добавлять матрицу копирования к списку матриц копирования.
