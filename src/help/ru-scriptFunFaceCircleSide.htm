faceCircleSide

Содержание=>Скрипт 3d=>Функции 3d=>faceCircleSide

Формат вызова

Face faceCircleSide( Float radius, Float sideCount, Matrix transfer )

Описание

Эта функция строит поверхность в виде равностороннего многоугольника вписанного в окружность. Количество сторон многоугольника определяется параметром sideCount. Полученный многоугольник можно преобразовать с помощью матрицы. Данная функция служит для построения аппроксимированных окружностей. Чем больше количество сторон многоугольника - тем выше качество аппроксимации. При малых значениях количества сторон с помощью этой функции можно строить другие правильные многоугольники: равносторонний треугольник, квадрат и т.п.

Параметры

  • radius - радиус описанной окружности вокруг многоугольника
  • sideCount - количество сторон вписанного многоугольника
  • transfer - заключительное преобразование многоугольника

Пример применения

#Пример окружности
color1 = selectColor("#800000")

c1 = faceCircleSide( 2, 36, matrix1 )

partModel = model( color1, color1, color1, [c1], matrix1 )

Данный пример сформирует следующее изображение:

#Пример шестиугольника
color1 = selectColor("#800000")

c1 = faceCircleSide( 2, 6, matrix1 )

partModel = model( color1, color1, color1, [c1], matrix1 )

Этот пример показывает формирование фигур при малых значениях количества сторон
