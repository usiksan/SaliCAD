Вставка корпуса компонента

Содержание=>Режимы=>Плата=>Вставка корпуса компонента

Панель инструментов:

Меню: Плата|Вставить корпус

Курсор:

Служит для вставки корпусов, не связанных с компонентами в плату. Примерами таких корпусов служат реперные знаки для систем автоматизированного производства. Корпуса в плате представляются вхождением корпуса.

1. В диалоге "Ввод корпусов" (см.Диалоговое окно "Ввод корпусов") выберите корпус для вставки.

2. Задайте атрибуты для корпуса.

3. Укажите точку вставки корпуса. При этом корпус вставляется в плату.

4. Повторяйте шаги 2 и 3 для ввода одинаковых корпусов.

5. Для прекращения ввода данного корпуса и возврата к выбору корпусов нажмите правую кнопку мыши.

6. Повторяйте шаги 1-5 для ввода различных корпусов в плату.

Атрибуты корпуса задаются с помощью панели свойств корпуса:

  • Указание точки осуществляется нажатием левой кнопки мыши или вручную.
  • Прекращение ввода осуществляется нажатием правой кнопки мыши.

"Разумный" режим отсутствует.
