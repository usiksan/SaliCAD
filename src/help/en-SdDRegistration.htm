Registration Dialog

Registration is required to distinguish between users and authors of components to access the remote library and to solve synchronization problems. See. remote library.

Each SaliCAD user must register with the Central library so that his or her name does not overlap with the names of other users of the System.

Each user can have five working machines, i.e. one and the same person can work on five different computers, including simultaneously.

When registering, the system generates an individual key for each user to access the Central library. An individual key is generated for each computer.

This dialog box allows you to register with the Central library and register additional computers for a specific user.

To register a user you need to fill in the fields:

  • SaliCAD server IP - IP address of the Central library server.
  • Name - the name under which the user will be registered in the Central library. This name will also be written to the author field for all user-created objects.
  • EMail - e-mail to be able to recover the key.

Leave the Machine key field blank. Upon successful registration, the key assigned by the Central library will appear here.

The Elements remain field shows the number of items available for download from the Central library.

To register, click the Registration button. The successful registration is indicated by the appearance of the key in the Machine key field.

To register an additional computer for an existing user, complete the following fields:

  • SaliCAD server IP - IP address of the Central library server.
  • Name - the name under which the user will be registered in the Central library. This name will also be written to the author field for all user-created objects.
  • EMail - e-mail to be able to recover the key.
  • Machine key - enter here the key of any registered computer of this user. Upon successful registration, this key will be replaced with a new one issued by the system.

To register a computer, click Add machine. Upon successful registration, a new key for this computer will appear in the key field.
