graphLine

Содержание=>Скрипт 3d=>Функции 3d=>graphLine

Формат вызова

Graph2d graphLine( FloatList [X0, Y0], FloatList [X1, Y1] )

Описание

Функция строит 2d объект в 3d пространстве. Функции данного класса предназначены для создания комплексных мастеров генерации компонентов, где в одном скрипте генерируются 3d модель и 2d изображение компонента. Для построения обычных 3d моделей использование данных функций не имеет смысла.

Все 2d объекты строятся в плоскости XY с небольшим смещением по осиZ.

Эта функция строит отрезок на основании своих параметров

Параметры

  • X0,Y0 - координаты начала отрезка
  • X1,Y1 - координаты конца отрезка

partFlat = graphLine( [0,0], [2,1] )
