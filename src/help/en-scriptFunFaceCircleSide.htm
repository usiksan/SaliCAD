faceCircleSide

Content=>Script 3d=>Functions 3d=>faceCircleSide

Prototype

Face faceCircleSide( Float radius, Float sideCount, Matrix transfer )

Description

This function constructs a surface in the form of an equilateral polygon inscribed in a circle. The number of sides of the polygon is determined by the sideCount parameter. The resulting polygon can be transformed using a matrix. This function is used to construct approximated circles. The greater the number of sides of the polygon, the higher the approximation quality. With small values of the number of sides, this function can be used to build other regular polygons: an equilateral triangle, a square, etc.

Параметры

  • radius - the radius of the circumscribed circle around the polygon
  • sideCount - the number of sides of the inscribed polygon
  • transfer - final polygon transformation

Example

#Circle example
color1 = selectColor("#800000")

c1 = faceCircleSide( 2, 36, matrix1 )

partModel = model( color1, color1, color1, [c1], matrix1 )
