matrixScale

Content=>Script 3d=>Functions 3d=>matrixScale

Prototype

Matrix matrixScale( Matrix src, Float scaleX, Float scaleY, Float scaleZ )

Description

This function builds builds the scaling matrix of the original matrix. Based on the scaleX, scaleY and scaleZ parameters, the original src matrix is scaled. If the scale parameter is equal to 1.0, then scaling along this axis is not performed, if it is less than 1, then a decrease is made, and if it is greater, an increase is made.

Parameters

  • src - the original matrix to which the scale is added
  • scaleX - scale on the x axis
  • scaleY - scale on the y axis
  • sclaeZ - scale on the z axis

Example

matrix2 = matrixScale( matrix1, 2.0, 1, 1 )

Builds a new matrix by adding a doubling of the X-axis to the original identity matrix matrix1.
