Редактор компонента

Содержание=>Режимы=>Редактор компонента

Редактор компонента не является графическим редактором, поэтому графические режимы в нем отсутствуют. Редактор компонента представляет собой диалоговую форму, с помощью которой можно:

  • задать и просмотреть схемные секции, из которых состоит компонент
  • задать и просмотреть соответствия номеров выводов корпуса именам выводов каждой секции
  • задать и просмотреть корпус компонента
  • задать и просмотреть параметры, установленные для компонента
  • задать и просмотреть таблицу параметров компонентов

Для работы со схемными изображениями секций компонента служит верхняя часть формы. Слева-направо расположены следующие элементы:

  • окно предварительного просмотра текущей выбранной секции
  • перечень секций компонента. При выборе секции в списке она показывается в окне предварительного просмотра, а в таблице правее показывается упаковочная информация
  • таблица упаковочной информации текущей выбранной секции. В этой таблице размещаются соответствия имен выводов секции номерам выводов корпуса. Каждая секция имеет собственную таблицу упаковочной информации, которая включается при выборе соответствующей секции
  • кнопки работы с секциями

Для работы с секциями служат следующие кнопки:

  • "Добавить секцию". При нажатии на эту кнопку включается диалоговое окно поиска и выбора секции. После выбора секции она вставляется в список секций компонента
  • "Дублировать". При нажатии на эту кнопку текущая выбранная секция дублируется. Эта кнопка очень удобна при создании компонентов имеющих несколько одинаковых секций
  • "Выбрать". При нажатии на эту кнопку включается диалоговое окно поиска и выбора секции. После выбора секции она заменяет текущую выбранную секцию компонента
  • "Удалить секцию". При нажатии на эту кнопку текущая выбранная секция удаляется
  • "Удалить все". При нажатии на эту кнопку удаляются все секции компонента

Все действия с секциями можно отменить.

Средняя часть формы служит для работы с корпусами. Слева-направо расположены следующие элементы:

  • окно предварительного просмотра текущего корпуса компонента
  • наименование корпуса компонента
  • кнопка выбора корпуса компонента. При нажатии на эту кнопку включается диалоговое окно поиска и выбора корпуса. После выбора корпуса он устанавливается "по умолчанию" для данного компонента

Правая нижняя части формы служит для работы с параметрами компонента. Для этого служат:

  • таблица параметров компонента. Эта таблица представляет собой таблицу вида "ключ-значение", где "ключ" служит названием параметра, а "значение" - значением параметра. Значения параметров можно изменить прямо в таблице параметров
  • для работы с самими параметрами служит группа кнопок
    • "Добавить параметр". При нажатии на эту кнопку предлагается ввести сначала имя параметра, а затем его значение. После этого параметр вместе со значением вставляется в таблицу параметров. Если в таблице уже имеется параметр с таким именем, то он заменяется вновь введенным
    • "По умолчанию". При нажатии на эту кнопку появляется диалоговое окно со списком параметров по умолчанию. В нем можно выбрать набор вставляемых параметров. Это диалоговое окно избавляет от ошибок связанных со вводом параметров вручную
    • "Копировать". При нажатии на эту кнопку включается диалоговое окно поиска и выбора компонента. После выбора компонента его параметры копируются в текущий компонент. Сам исходный компонент не изменяется
    • "Удалить". При нажатии на эту кнопку текущий параметр удаляется
    • "Таблица вариантов...". При нажатии на эту кнопку включается диалоговое окно просмотра и редактирования вариантов параметров компонента
