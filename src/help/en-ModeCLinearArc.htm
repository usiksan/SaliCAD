Arc enter mode

Contents=>Modes=>Draw modes=>Arc

Tool bar:

Menu: Draw|Arc

Cursor:

1. Setup arc attributes

2. Enter arc center point

3. Enter arc start point

4. Enter arc end point. The arc rotates from the start point to the end point counterclockwise.

  • You can specify a point by clicking the left mouse button or manually.
  • The termination of the input is done by pressing the right mouse button.
