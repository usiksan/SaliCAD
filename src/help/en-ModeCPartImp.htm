Insert part mode

Contents=>Modes=>Plate=>Insert part mode

Tool bar:

Menu: Plate|Insert part

Cursor:

Служит для вставки корпусов, не связанных с компонентами в плату. Примерами таких корпусов служат реперные знаки для систем автоматизированного производства. Корпуса в плате представляются вхождением корпуса.

1. Select part to insert in "Enter part" dialog (see "Enter part" dialog).

2. Setup part properties.

3. Specify the part location point. On this part is inserted into the Plate.

4. Repeat steps 2 and 3 for the same parts.

5. To stop entering this part and return to selecting parts, press the right mouse button.

6. Repeat steps 1-5 to insert into plate different parts.

Part properties set with part properties bar:

  • You can specify a point by clicking the left mouse button or manually.
  • The termination of the input is done by pressing the right mouse button.

"Smart" mode not present.
