faceListCylinder

Содержание=>Скрипт 3d=>Функции 3d=>faceListCylinder

Формат вызова

FaceList faceListCylinder( Float radius, Float height )

Описание

Эта функция строит цилиндр с основанием на плоскости XY. Окружности оснований цилиндра аппроксимируются равносторонними многоугольниками вписанными в окружность. Количество сторон многоугольника определяется автоматически исходя из радиуса, чтобы получить визуально гладкую поверхность.

Параметры

  • radius - радиус описанной окружности вокруг многоугольника основания цилиндра
  • height - высота цилиндра

Пример применения

color1 = selectColor("#a00000")

c1 = faceListCylinder( 2, 5 )

partModel = model( color1, color1, color1, c1, matrix1 )
