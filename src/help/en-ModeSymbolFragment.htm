Enter symbol fragment

Contents=>Modes=>Symbol specific modes=>Enter symbol fragment

Tool bar:

Menu: Symbol|Enter symbol fragment

Cursor:

It is used to insert library symbols when creating new symbols. Thus, library symbols can serve as the basis or components of a new symbol.

1. Find and select symbol in "Select symbol to insert" dialog (see "Select symbol to insert" dialog).

2. Specify the insertion point of the symbol fragment. In this case, a copy of the symbol is inserted into the current symbol.

3. Repeat step 2 to insert multiple copies of the fragment.

4. To stop entering this copy of the symbol and return to the symbol selection, right-click.

5. Repeat steps 1-4 to enter different character fragments into the current symbol.

  • You can specify a insertion point by clicking the left mouse button or manually.
  • The termination of the input is done by pressing the right mouse button.

"Smart" mode not present.
