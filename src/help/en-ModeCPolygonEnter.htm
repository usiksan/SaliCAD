Polygon enter mode

Contents=>Modes=>Plate modes=>Polygon enter mode

Tool bar:

Menu: Plate|Enter polygon

Cursor:

1. Set the polygon properties. Select the layer for the polygon and the polygon net name.

2. Specify the starting point of the polygon.

When you move the mouse, the catcher shows the possible starting point of the polygon. It is searched on the component pins and at the road ends. To take the starting point of the point proposed by the catcher, you must press the middle mouse button or space. In this case, the net name is set according to the starting point of the polygon.

3. (Optional) Change the polygon properties.

4. Specify the next vertex of the polygon outline.

5. Repeat steps 3-4 to enter the entire polygon. If the next vertex coincides with the initial one, the polygon input is automatically completed and the polygon is included in the project. This automatically generates Windows in the polygon. Windows are formed on the basis of roads and pads crossing the polygon. The gap for Windows in the polygon is set in the polygon properties bar.

The properties of the polygon are specified using the polygon property bar.

  • You can specify a point by clicking the left mouse button or manually.
  • The termination of the input is done by pressing the right mouse button.

"Smart" mode.

Smart mode offers automatic completion of the polygon. When you select a smart mode, the polygon closes and enters into the project.

Middle mouse button or "space" - confirmation of "smart" input.
