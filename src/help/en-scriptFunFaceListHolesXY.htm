faceListHolesXY

Content=>Script 3d=>Functions 3d=>faceListHolesXY

Prototype

FaceList faceListHolesXY( Face src, FaceList holes )

Description

This function converts the surface and individual holes into a set of surfaces, which is the original surface with holes built into it, for example, to form a pin contact pad. For the function to work correctly, it is necessary that the initial surface and the holes are located on the same XY plane. In addition, the vertices of the original surface and holes must have a clockwise vertex traversal order.

Parameters

  • src - initial surface
  • holes - list of hole surfaces

Example

color1 = selectColor("#a00000")

src = faceRect( 5, 5, matrix1 )
hole1 = faceRect( 1, 1, matrixTranslate( matrix1, -1, -1, 0 ) )
hole2 = faceRect( 1, 1, matrixTranslate( matrix1, 1, 1, 0 ) )
body = faceListHolesXY( src, [hole1,hole2] )

partModel = model( color1, color1, color1, body, matrix1 )
