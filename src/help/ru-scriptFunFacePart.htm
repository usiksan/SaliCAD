facePart

Содержание=>Скрипт 3d=>Функции 3d=>facePart

Формат вызова

Face facePart(Face src, FloatList vertexIndexes )

Описание

Эта функция строит многоугольник из вершин другого многоугольника. Исходный многоугольник задается параметром src. Параметр vertexIndexes содержит последовательный список индексов вершин исходного многоугольника, которые составят результирующий многоугольник.

Параметры

  • src - исходный многоугольник
  • vertexIndexes - список индексов исходного многоугольника, которые составляют результирующий многоугольник

Пример применения

color1 = selectColor("#800000")
color2 = selectColor("#008000")

c1 = faceFlatMatrix( [ 0,0, 0,2, 1,1, 1,-1, 0,-2], matrix1 )

c2 = facePart( c1, [ 1, 2, 3 ] )

partModel = model( color1, color1, color1, [c1], matrix1 )
partModel = model( color2, color2, color2, [c2], matrixTranslate( matrix1, 2.2, 0, 0) )
