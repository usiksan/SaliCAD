Enter bus nets dialog

In this dialog You define net list for fast enter bus. Net list can be entered or selected from previously entered list. You can also select one previously entered list and edit it.

Net list format is:

Net1[,...]

where Net1 represents net name. When You enter net list with this dialog You can not use commas in net names because it used as net delimiter. If some nets have same name but different index then You can use short description:

same_name_letters<start_index:stop_index>

For example:

D<0:3>,+5V,Gnd,A<0:2> will produce next net list result

D0
D1
D2
D3
+5V
Gnd
A0
A1
A2

The right part of the dialog contains a list of all named nets, i.e. those nets whose names were entered manually. Nets with automatic names are not represented in this list. When you select a net from this list, it is automatically placed in the bus list.
