faceRectSide

Content=>Script 3d=>Functions 3d=>faceRectSide

Prototype

Face faceRectSide( Float length, Float width, Float sideCount, Matrix transfer )

Description

This function constructs a surface in the form of a rectangle consisting of a polyline with a specified number of edges. The resulting rectangle can be transformed using a matrix. Despite the number of edges, the shape looks like a rectangle.

Parameters

  • length - rectangle length X
  • width - rectangle width Y
  • sideCount - the number of edges of the polygon must be a multiple of the 4th
  • transfer - final rectangle transformation

Example

color1 = selectColor("#800000")

r1 = faceRectSide( 2, 1, 40, matrix1 )

partModel = model( color1, color1, color1, [r1], matrix1 )
