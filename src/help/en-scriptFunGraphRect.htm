graphRect

Content=>Script 3d=>Functions 3d=>graphRect

Prototype

Graph2d graphRect( FloatList [cornerX0, cornerY0], FloatList [cornerX1, cornerY1] )

Description

The function builds a 2d object in 3d space. The functions of this class are designed to create complex component generation wizards, where a 3d model and a 2d image of a component are generated in one script. To build ordinary 3d models, using these functions does not make sense.

All 2d objects are constructed in the XY plane with a slight offset along the axis.

This function builds a rectangle based on its parameters

Parameters

  • cornerX0, cornerY0 - coordinates of the first corner of the rectangle
  • cornerX1, cornerY1 - coordinates of the opposite (diagonal) corner of the rectangle

Example

partFlat = graphRect( [0,0], [2,1] )
