Component editor

Contents=>Modes=>Component editor

The component editor is not a graphical editor, so there are no graphical modes in it. The component editor is a dialog form with which you can:

  • set and view the schematic sections that make up the component
  • set and view the correspondence of the pin numbers of the enclosure to the pin names of each section
  • set and view component part
  • set and view component parameters
  • set and view component parameters table

The upper topic of the form is used to work with schematic images of the component sections. From left to right are the following elements:

  • preview window for the currently selected section
  • the list of sections of the component. When you select a section in the list, it is displayed in the preview window, and the packing information is displayed in the table to the right
  • the packing information table of the currently selected section. In this table presented correspondence of the pin numbers of the enclosure to the pin names of selected section. Each section has its own packing information table, which is included when you select the appropriate section
  • buttons for working with sections

The following buttons are used to work with sections:

  • "Add section" Clicking this button enables the component section search and selection dialog box. After you select a section, it is inserted into the list of component sections
  • "Duplicate" Clicking this button duplicates the currently selected section. This button is very useful when creating components with several identical sections
  • "Select" Clicking this button enables the component section search and selection dialog box. After you select a section, it replaces the currently selected section of the component
  • "Delete section". Clicking this button deletes the currently selected section
  • "Delete all." Clicking this button deletes all sections of the component

All of the sections manipulations can be undo.

The middle topic of the form is used to work with parts. From left to right are the following elements:

  • preview window of the current component part
  • the name of the component part
  • button to select the component part. When you click this button, the component part search and select dialog box is enabled. After selecting the part, it is set to "default" for this component

The lower right topic of the form is used to work with the component parameters. For this purpose serve:

  • table of component parameters. This table is a table of the form "key-value", where "key" is the name of the parameter, and "value" is the value of the parameter. Parameter values can be changed directly in the parameter table
  • to work with the parameters themselves, use the group of buttons
    • "Add parameter" When you click this button, you are prompted to enter the parameter name first and then its value. The parameter is then inserted with the value into the parameter table. If the table already has a parameter with the same name, it is replaced with the newly entered one
    • "By default" When you click this button, a dialog box appears with a list of default settings. It allows you to select a set of parameters to be inserted. This dialog box eliminates errors associated with entering parameters manually
    • "Copy" When you click this button, the component search and select dialog box is enabled. After you select a component, its parameters are copied to the current component. The original component itself is not changed
    • "Delete" Clicking this button deletes the current setting
    • "Variant table...". When you click this button, the dialog box for viewing and editing for the component parameters table is enabled
