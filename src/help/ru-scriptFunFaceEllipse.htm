faceEllipse

Содержание=>Скрипт 3d=>Функции 3d=>faceEllipse

Формат вызова

Face faceEllipse( Float radiusX, Float radiusY, Float stepDegree, Matrix transfer )

Описание

Эта функция строит поверхность в виде равностороннего многоугольника вписанного в элипс. Количество сторон многоугольника определяется шагом угла. Полученный многоугольник можно преобразовать с помощью матрицы. Данная функция служит для построения аппроксимированных элипсов. Чем меньше шаг угла - тем выше качество аппроксимации.

Параметры

  • radiusX - радиус X описанноого элипса вокруг многоугольника
  • radiusY - радиус Y описанноого элипса вокруг многоугольника
  • stepDegree - шаг угла в градусах
  • transfer - заключительное преобразование многоугольника

Пример применения

#Пример окружности
color1 = selectColor("#800000")

c1 = faceEllipse( 3, 2, 10, matrix1 )

partModel = model( color1, color1, color1, [c1], matrix1 )
