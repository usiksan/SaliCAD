Insert fragment

Contents=>Modes=>Sheet specific modes=>Insert fragment

Tool bar:

Menu: Sheet|Insert fragment

This mode used to insert fragments into the schematic sheet. The fragments are the schematic sheets of other project.

1. Find and select fragment in "Select fragment to insert" dialog (see "Select fragment to insert" dialog).

2. Enter fragment location point. This inserts the fragment into the schematic sheet. All nets of the inserted fragment are combined with nets of the existing scheme. Nets having names are connected to nets having the same names. Nets with default names gain new default names.

3. Repeat step 2 to enter the same fragment.

4. To stop entering this fragment and return to selecting a new fragment, right-click.

5. Repeat steps 1-4 to enter different fragments into the schematic sheet.

  • You can specify a point by clicking the left mouse button or manually.
  • The termination of the input is done by pressing the right mouse button.

"Smart" mode not present.
