Layers Dialog

In this dialog box, you can see a list of available layers, the state of the layers, and their pairs (see layers).

In the Central part of the dialog there is a table with a list of available layers. Columns allocated to the name of the layer, the destination for the trace, status, visibility, color, and pairing.

Clicking an Trace cell allows you to cycle through the possible layer assignments for the trace.

Clicking on State cell allows you to cycle through the possible States of the layer.

Clicking on the Visibility cell allows you to turn on or off the visibility of the layer. The layer is visible in the Enabled and Editable state. The layer is invisible in the Off state. Visibility is a graphical representation of the state of a layer.

When you click on the Color cell, a dialog box for selecting a color for the layer appears. You can assign any color to the layers, including the same color. The color of layers is saved in Your profile on the computer. Another computer may have its own color palette.

When you click on the Pair name cell, the dialog box for selecting a pair layer for this one appears.

Tracing layers count in plate field specifies the number of trace layers for a particular Board. Therefore, this field is active only when the layers dialog is called from the graphics card editor.

When you click Close, the dialog box closes.

When you click the Create... button a dialog box for creating a new layer appears.

When you click Delete, the currently selected layer is deleted. Deletion is possible only if there is no information on this layer in any of the open projects.

Buttons to limit the displayed list of layers.


The number of layers can be quite large, but only a limited number of these layers can be used at any given time. These buttons allow you to reduce the amount of information displayed. They do not affect the addition or removal of layers.

  • Show actual. The layers that are relevant for the graphical editor are shown. Ie a list of layers to "default".
  • Show used. Layers on which some information is present are shown.
  • Show all. All currently known layers are shown.

When you click the Hide all button, the state of all layers changes to Off.

When you click the Switch all button state of all layers switches to the next state, as if you were clicking the State for all layers box.

Download and save the state of layers

Often, a specific set of layers is used for a particular type of operation. For example, to arrange the positional designations on the Board need images of the component, the Board and the positional designations themselves. Images of trace layers will only interfere, so it is logical to turn them off. To make it easier to work with such sets of enabled layers and use their loading and saving.

When you click Save... a dialog box opens to select the path to save the current set of enabled layers.

When you click Load... opens a dialog to select a file containing the desired set of enabled layers. When you select this file, it is loaded, all layers are made invisible except those visible, which are recorded in this particular file.
