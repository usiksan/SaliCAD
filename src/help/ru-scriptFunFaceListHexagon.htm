faceListHexagon

Содержание=>Скрипт 3d=>Функции 3d=>faceListHexagon

Формат вызова

FaceList faceListHexagon( Float lenght, Float topLenght, Float height, Float width )

Описание

Эта функция строит хексагональный корпус компонента.

Параметры

  • lenght - длина корпуса в средней части
  • topLenght - длина корпуса в верхней и в нижней частях (принимается одинаковой)
  • height - высота корпуса
  • width - ширина корпуса

Пример применения

color1 = selectColor("#a00000")

body = faceListHexagon( 4, 3.6, 1, 3 )

partModel = model( color1, color1, color1, body, matrix1 )
