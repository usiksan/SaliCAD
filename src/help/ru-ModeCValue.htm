Размещение значения

Содержание=>Режимы=>Режимы редактирования символа (Режимы редактирования корпуса)=>Размещение значения

Панель инструментов:

Меню: Символ|Размещение значения

Меню: Корпус|Размещение значения

Курсор:

Значение символа (корпуса) служит указателем места, где будет размещено содержимое  поля <value> для элемента в схеме (плате).

1. Установите атрибуты значения

2. Укажите точку расположения значения

"Разумный" режим отсутствует.

Для ввода атрибутов позиционного обозначения служит панель свойств текста.

  • Указание точки осуществляется нажатием левой кнопки мыши или вручную.
  • Прекращение ввода осуществляется нажатием правой кнопки мыши.
