vertexOffset

Содержание=>Скрипт 3d=>Функции 3d=>vertexOffset

Формат вызова

Vertex vertexOffset( Vertex src, Float  offx, Float offy, Float offz )

Описание

Эта функция строит точку в 3d пространстве путем смещения исходной точки src по трем осям на соответствующие значения offNN.

Параметры

  • src - исходная точка
  • offx - смещение по оси X
  • offy - смещение по оси Y
  • offz - смещение по оси Z

Пример применения

vertex1 = vertex( 0.2, 0.1, 0.4 )
vertex2 = vertexOffset( vertex1, 0, 0, 0.5 )
