Trace delete

Contents=>Modes=>Plate modes=>Trace delete

Tool bar:

Menu: Plate|Trace delete

Cursor:

This mode is used to remove traces (roads, vias, polygons) on the board, i.e. to change the topology of the trace.

1. Hover the mouse cursor over a road segment, via, or polygon. The selected object to delete will be highlighted.

2. Press the left mouse button while the highlighted element is removed.

3. Repeat steps 1-2 to remove other traces.

  • You can specify a point by clicking the left mouse button or manually.
  • The termination of the input is done by pressing the right mouse button.
This mode allows you to delete an entire group of traces at once.

1. Press and hold the Shift key.

2. Press and hold the left mouse button.

3. Move the mouse without releasing the left mouse button. In this case, a rectangular area is formed between the first point and the current one. All trace objects that fall inside this area are highlighted.

4. If you release the Shift key, the highlight is removed from all elements, if you press the Shift key again, the elements are highlighted again.

5. When you release the left mouse button, all highlighted elements are deleted.


"Smart" mode is not present.
