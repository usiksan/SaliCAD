stringPadCircleThrough

Содержание=>Скрипт 3d=>Функции 3d=>stringPadCircleThrough

Формат вызова

String stringPadCircleThrough( Float padDiameter, Float holeDiameter, Float mask )

Описание

Функция строит текстовое описание контактной площадки, заданной параметрами.

Параметры

  • padDiameter - диаметр контактной площадки в мм
  • holeDiameter - диаметр отверстия мм
  • mask - дополнительное расстояние маски мм. На это значение увеличивается окно маски по отношению к контактной площадке

Пример применения

  pin = stringPadCircleThrough( 2, 0.6, 0.1 )

Данный пример сформирует следующее описание контактной площадки "c2d0.6m0.1"
