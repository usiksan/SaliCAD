faceListWallDoubleBevelXY

Content=>Script 3d=>Functions 3d=>faceListWallDoubleBevelXY

Prototype

FaceList faceListWallDoubleBevelXY( Face src, Face dst, Float radiusBot, Float radiusTop, Float angleStepBot, Float angleStepTop, Float height )

Description

This function builds two chamfer walls, bottom and top, between the src and DST surfaces. The number of points in the source surfaces must be the same.  Different combinations of the sign in the faceListWallDoubleBevelXY parameters allow you to get different shapes of chamfers.

Parameters

  • src - source surface
  • dst - target surface
  • radiusBot - chamfer radius from below
  • radiusTop - chamfer radius from above
  • angleStepBot - the size of the angle step for approximating the curve of the chamfer surface from below
  • angleStepTop - the size of the angle step for approximating the curve of the chamfer surface from above
  • height - height between the lower and upper chamfers (external size)

Example

color1 = selectColor("#a00000")

c1 = faceCircleSide( 3, 56, matrix1 )
c2 = faceDuplicateShift( c1, 2 )

h1 = faceListWallDoubleBevelXY( c1, c2, 1, -1, -10, 10, 2 )

partModel = model( color1, color1, color1, h1, matrix1 )
