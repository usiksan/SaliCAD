Content=>Script 3d=>Functions 3d=>selectColor

Prototype

Color selectColor( String colorString )

Description

This function builds a color based on its text representation. If the cursor is on this function, pressing the F2 button causes the color selection dialog box to be displayed. After selecting a color through the dialog, it is replaced in the script text.

Parameters

  • colorString - the text representation of the color in the format "#rrggbb", where rr is the hexadecimal code of red, gg and bb are the codes of green and blue, respectively

Example

bodyColor1 = selectColor( "#202020" )
