faceRectRound

Content=>Script 3d=>Functions 3d=>faceRectRound

Prototype

Face faceRectRound( Float length, Float width, Float radius, Float stepDegree, Matrix transfer )

Description

This function builds a surface in the form of a rectangle with rounded corners. Rounding is performed by approximating a polyline. The resulting rectangle can be transformed using a matrix.

Parameters

  • length - rectangle length X
  • width - rectangle width Y
  • radius - radius of rounding corners
  • stepDegree - the step in degrees between adjacent vertices of the polyline rounding
  • transfer - final rectangle transformation

Example

color1 = selectColor("#800000")

r1 = faceRectRound( 2, 1, 0.2, 10, matrix1 )

partModel = model( color1, color1, color1, [r1], matrix1 )
