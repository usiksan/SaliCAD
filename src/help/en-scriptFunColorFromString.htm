Prototype

Color colorFromString( String colorString )

Description

This function builds a color based on its text representation. The selectColor function does the same job, but it also allows you to select a color in the dialog mode. Therefore, using the selectColor function is preferable.

Parameters

  • colorString - the text representation of the color in the format "#rrggbb", where rr is the hexadecimal code of red, gg and bb are the codes of green and blue, respectively

Example

bodyColor1 = colorFromString( "#202020" )
