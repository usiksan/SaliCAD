faceListExtrude

Содержание=>Скрипт 3d=>Функции 3d=>faceListExtrude

Формат вызова

FaceList faceListExtrude( Face src, Matrix transfer )

Описание

Эта функция делает вытяжку из исходной поверхности, формируя тело. Исходная поверхность дублируется с помощью матрицы и между получившимися поверхностями строятся стены. Получившийся набор поверхностей, включая исходную поверхность и дублированную, возвращаются в виде тела.

Параметры

  • src - исходная поверхность
  • transfer - матрица преобразования

Пример применения

color1 = selectColor("#a00000")

b1 = faceRect( 3, 2, matrix1 )
body = faceListExtrude( b1, matrixTranslate( matrix1, 1, 0, 1 ) )

partModel = model( color1, color1, color1, body, matrix1 )
