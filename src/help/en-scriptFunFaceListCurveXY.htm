faceListCurveXY

Content=>Script 3d=>Functions 3d=>faceListCurveXY

Prototype

FaceList faceListCurveXY( Face face, Float radius, Float angleSrc, Float angleDst, Float sideCount, Bool close )

Description

Builds walls between rotated surfaces, serves to form a bend, for example, output. Takes the original surface and rotates it to a given angle angle Dist. Between these surfaces forms intermediate rotated surfaces, their number is determined by sideCount. Builds walls between all the formed surfaces and returns their set.

Parameters

  • face - initial surface
  • radius - turning radius
  • angleSrc - the angle of rotation of the initial surface relative to the XY plane
  • angleDst - the angle to which the original surface should be rotated
  • sideCount - the number of intermediate surfaces between the initial and final, including the final

Example

color1 = selectColor("#a00000")

c1 = faceRect( 2, 3, matrix1 )

c2 = faceListCurveXZ( c1, 5, 0, 30, 10, true )

partModel = model( color1, color1, color1, c2, matrix1 )
