modelCopy

Content=>Script 3d=>Functions 3d=>modelCopy

Prototype

Model modelCopy( Model srcModel, Matrix translation )

Description

This function adds another copy of the existing model to the model. Using this function, it is convenient to create various arrays of component outputs.

Parameters

  • srcModel - source model
  • translation - the transformation matrix for the copy

Example

color1 = selectColor("#a00000")
color2 = selectColor("#00a000")

c1 = faceListCylinder( 2, 5 )
c2 = faceListDuplicate( c1, matrixTranslate( matrix1, 5, 0, 0 ) )

m1 = model( color1, color1, color1, c1, matrix1 )
m1 = modelAppend( m1, color2, color2, color2, c2 )

partModel = modelCopy( m1, matrixTranslate( matrix1, 0, 7, -3) )
