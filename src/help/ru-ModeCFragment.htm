Вставка фрагмента

Содержание=>Режимы=>Специальные режимы схемы=>Вставка фрагмента

Панель инструментов:

Меню: Лист схемы|Фрагмент

Служит для вставки фрагментов в лист схемы. Фрагменты являются листами схемы другого проекта.

1. В диалоге "Ввод фрагментов" (см.Диалоговое окно "Ввод фрагментов") выберите фрагмент.

2. Укажите точку вставки фрагмента. При этом фрагмент вставляется в лист схемы. Все цепи вставляемого фрагмента объединяются с цепями существующей схемы. Цепи имеющие имена подключаются к цепям имеющих такие-же имена. Цепи с именами по умолчанию получают новые имена по умолчанию.

3. Повторяйте шаг 2 для ввода одинаковых фрагментов.

4. Для прекращения ввода данного фрагмента и возврата к выбору другого фрагмента нажмите правую кнопку мыши.

5. Повторяйте шаги 1-4 для ввода различных фрагментов в лист схемы.

  • Указание точки осуществляется нажатием левой кнопки мыши или вручную.
  • Прекращение ввода осуществляется нажатием правой кнопки мыши.

"Разумный" режим отсутствует.
