faceListExtrudeShift

Content=>Script 3d=>Functions 3d=>faceListExtrudeShift

Prototype

FaceList faceListExtrudeShift( Face src, Float height )

Description

This function extrudes from the original surface in the direction perpendicular to this surface, forming a body. The original surface is duplicated by transferring in the direction perpendicular to the original surface by a given amount and walls are built between the resulting surfaces. The resulting set of surfaces, including the original surface and the duplicated one, are returned as a body.

Parameters

  • src - initial surface
  • height - hood height (positive numbers - hood down, negative numbers - up)

Example

color1 = selectColor("#a00000")

b1 = faceRect( 3, 2, matrixRotate( matrix1, 20, 0, 1, 0 ) )
body = faceListExtrudeShift( b1, -1 )

partModel = model( color1, color1, color1, body, matrix1 )
