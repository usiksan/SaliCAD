stringPinIndex

Содержание=>Скрипт 3d=>Функции 3d=>stringPinIndex

Формат вызова

String stringPinIndex( Float index )

Описание

Функция строит текстовое представление цифрового номера вывода. По сути преобразовывает число в строку.

Параметры

  • index - номер вывода

Пример применения

  pin = stringPinIndex( 5 )

Данный пример сформирует номер вывода "5"
