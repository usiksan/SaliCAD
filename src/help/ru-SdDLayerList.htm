Диалог Задание парного слоя


В этом диалоговом окне можно выбрать парный слой для заданного слоя, а также убрать парность для заданного слоя (см.слои).

В центральной части диалога находится список доступных слоев. Выберите слой из данного списка и нажмите Назначить. При этом выбранной слой становится парным для исходного слоя.

При нажатии кнопки Убрать парность для заданного слоя убирается.
