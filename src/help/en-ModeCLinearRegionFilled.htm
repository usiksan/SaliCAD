Filled region enter mode

Contents=>Modes=>Draw modes=>Filled region

Tool bar:

Menu: Draw|Filled region

Cursor:

1. Setup region attributes

2. Enter first point of region

3. (Optional) Change region attributes

4. Enter next vertex of region.

5. Repeat steps 2-4 to enter the entire region. If next vertex corresponds to first vertex then region closes and the region falls into the project.

  • You can specify a point by clicking the left mouse button or manually.
  • The termination of the input is done by pressing the right mouse button.

"Smart" mode.

Smart mode offers automatic completion of the region. When you select a smart mode, the region closes and the region falls into the project.

Middle mouse button or "space" - confirmation of "smart" input.
