modelAppend

Содержание=>Скрипт 3d=>Функции 3d=>modelAppend

Формат вызова

Model modelAppend( Model srcModel, Color ambient, Color diffuse, Color specular, FaceList faces )

Описание

Эта функция добавляет в модель набор поверхностей с другим цветом.

Параметры

  • srcModel - исходная модель
  • ambient - цвет модели
  • diffuse - цвет модели
  • specular - цвет модели
  • faces - набор поверхностей модели

Пример применения

color1 = selectColor("#a00000")
color2 = selectColor("#00a000")

c1 = faceListCylinder( 2, 5 )
c2 = faceListDuplicate( c1, matrixTranslate( matrix1, 5, 0, 0 ) )

m1 = model( color1, color1, color1, c1, matrix1 )
partModel = modelAppend( m1, color2, color2, color2, c2 )
