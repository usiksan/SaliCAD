vertex

Content=>Script 3d=>Functions 3d=>vertex

Prototype

Vertex vertex( Float x, Float y, Float z )

Description

This function builds a point in 3d space based on its coordinates.

Parameters

  • x - X-axis coordinate
  • y - Y-axis coordinate
  • z - Z-axis coordinate

Example

vertex1 = vertex( 0.2, 0.1, 0.4 )
