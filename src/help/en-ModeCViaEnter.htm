Via enter mode

Contents=>Modes=>Plate modes=>Via enter mode

Tool bar:

Menu: Plate|Via enter

Cursor:

This mode used for inserting additional vias.

1. Setup via properties. Select source layer for via.

2. Enter via location point. When mouse left button pressed via inserted into plate only if via placement accordings tracing rules.

3. Repeat steps 1-2 to enter all necessary vias.

Свойства переходных отверстий задаются с помощью панели свойств переходных отверстий.

  • You can specify a point by clicking the left mouse button or manually.
  • The termination of the input is done by pressing the right mouse button.

"Smart" mode is not present
