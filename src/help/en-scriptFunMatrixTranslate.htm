matrixTranslate

Content=>Script 3d=>Functions 3d=>matrixTranslate

Prototype

Matrix matrixTranslate( Matrix src, Float offX, Float offY, Float offZ )

Description

This function builds builds a displacement matrix based on the original matrix. Based on the parameters offX, offY and offZ, the function builds a displacement matrix.  The movement is determined by a vector with coordinates off.

Parameters

  • src - source matrix
  • offX - x-component of the displacement vector
  • offY - y-component of the displacement vector
  • offZ - z-component of the displacement vector

Example

color1 = selectColor("#800000")
color2 = selectColor("#008000")

c1 = faceCircle( 2, 10, matrix1 )

mat = matrixTranslate( matrix1, 1, 0, 1 )

c2 = faceDuplicate( c1, mat )

partModel = model( color1, color1, color1, [c1], matrix1 )
partModel = model( color2, color2, color2, [c2], matrix1 )
