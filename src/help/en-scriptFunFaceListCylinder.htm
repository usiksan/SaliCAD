faceListCylinder

Content=>Script 3d=>Functions 3d=>faceListCylinder

Prototype

FaceList faceListCylinder( Float radius, Float height )

Description

This function builds a cylinder with a base on the XY plane. The circles of the cylinder bases are approximated by equilateral polygons inscribed in a circle. The number of sides of the polygon is determined automatically based on the radius to get a visually smooth surface.

Parameters

  • radius - the radius of the circumscribed circle around the polygon of the base of the cylinder
  • height - cylinder height

Example

color1 = selectColor("#a00000")

c1 = faceListCylinder( 2, 5 )

partModel = model( color1, color1, color1, c1, matrix1 )
