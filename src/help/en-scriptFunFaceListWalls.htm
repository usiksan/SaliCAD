faceListWalls

Content=>Script 3d=>Functions 3d=>faceListWalls

Prototype

FaceList faceListWalls( FaceList floors, Bool close )

Description

This function builds walls between floors represented by a set of surfaces. The number of points in the surfaces must be the same. The close parameter determines whether to build a wall between the end and start points of each surface.

The layout of the floors is arbitrary, not just horizontal. Therefore, using this function, rotation bodies can be obtained, as shown in the example.

Parameters

  • floors - list of floor surfaces
  • close - when true, a wall is built between the last and first points, otherwise, this wall is not built and the figure turns out to be open

Example

color1 = selectColor("#a00000")

pin = faceListRotation( 0, 80, 20, matrix1, [ 0.5,0, 0.5,0, 0,1, -0.5,0 ] )

pin = faceListWalls( pin, false )

partModel = model( color1, color1, color1, pin, matrix1 )
