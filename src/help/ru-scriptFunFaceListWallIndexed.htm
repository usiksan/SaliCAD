faceListWallIndexed

Содержание=>Скрипт 3d=>Функции 3d=>faceListWallIndexed

Формат вызова

FaceList faceListWallIndexed( Face src, Face dst, FloatList indexes )

Описание

Эта функция строит стены согласно индексам между поверхностями src и dst. Количество точек в исходных поверхностях должно быть одинаковым. Фактически, поверхность dst обычно получается путем некоторого преобразования поверхности src. Индексы обозначают порядковый номер ребра в исходных поверхностях, начиная с нуля.

Параметры

  • src - исходная поверхность (пол)
  • dst - целевая поверхность (потолок)
  • indexes - набор номеров ребер, между которыми строятся стены

Пример применения

color1 = selectColor("#a00000")

c1 = faceCircleSide( 3, 6, matrix1 )
c2 = faceDuplicateShift( c1, 4 )

h1 = faceListWallIndexed( c1, c2, [0,1,3,4] )

partModel = model( color1, color1, color1, h1, matrix1 )
