Вставка имен цепей

Содержание=>Режимы=>Специальные режимы схемы=>Вставка имен цепей

Панель инструментов:

Меню: Лист схемы|Вставить цепь

Курсор:

Служит для простановки на схеме имен цепей. Изменение имен цепей с помощью этого режима невозможно. Вместо этого для этой цели нужно использовать режим выбора или режим вставки цепей.

1. Укажите цепь имя которой необходимо отобразить

2. Установите атрибуты имени цепи

3. Укажите местоположение имени цепи

4. Повторяйте шаги 1-3 пока не будут проставлены имена всех нужных цепей.

  • Указание точки осуществляется нажатием левой кнопки мыши или вручную.
  • Прекращение ввода осуществляется нажатием правой кнопки мыши.

Атрибуты имени цепи задаются с помощью панели свойств текста.

"Разумный режим" отсутствует.
