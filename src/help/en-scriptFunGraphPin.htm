graphPin

Content=>Script 3d=>Functions 3d=>graphPin

Prototype

Graph2d graphPin( FloatList [pinPosX, pinPosY], String padDescription, FloatList [pinNumberPosX,pinNumberPosY], String pinNumber, String pinNuberAttr, FloatList [pinNamePosX,pinNamePosY], String pinNameAttr )

Description

The function builds a 2d object in 3d space. The functions of this class are designed to create complex component generation wizards, where a 3d model and a 2d image of a component are generated in one script. To build ordinary 3d models, using these functions does not make sense.

All 2d objects are constructed in the XY plane with a slight offset along the axis.

This function builds the output of the component based on its parameters. In 3d space, only the contact area is displayed. The main construction takes place in 2d space.

Parameters

  • pinPosX, pinPosY - coordinates of the pin center
  • padDescription - pin pad description
  • pinNumberPosX,pinNumberPosY - pin number position
  • pinNumber - pin number
  • pinNuberAttr - pin number attributes, see text attributes
  • pinNamePosX,pinNamePosY - pin name position
  • pinNameAttr - pin name attributes, see text attributes

Text attributes

Text attributes are used to set the horizontal and vertical alignment of the text, as well as to set the angle of rotation of the text.

"LT90"

where

  • L - horizontal alignment, may be one of L-left, C-center, R-right
  • T - vertical alignment, may be one of T-top, M-middle, B-bottom
  • 90 - angle of text rotation in degree, may be one of 0, 90, 180 and 270

Example

partFlat = graphPin( [0,0], "c1.2d0.6m0.1", [0,0], "1", "CM0", [0,0], "CM0" )
