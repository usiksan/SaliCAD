stringPadRectThrough

Содержание=>Скрипт 3d=>Функции 3d=>stringPadRectThrough

Формат вызова

String stringPadRectThrough( Float widht, Float height, Float diameter, Float mask )

Описание

Функция строит текстовое описание контактной площадки, заданной параметрами.

Параметры

  • width - ширина контактной площадки в мм
  • height - высота контактной площадки в мм
  • diameter - диаметр отверстия
  • mask - дополнительное расстояние маски. На это значение увеличивается окно маски по отношению к контактной площадке

Пример применения

  pin = stringPadRectThrough( 2, 1, 0.6, 0.1 )

Данный пример сформирует следующее описание контактной площадки "r2x1d0.6m0.1"
