Тип список поверхностей

Содержание=>Скрипт 3d=>Тип список поверхностей

Список поверхностей представляет собой набор из одной или нескольких поверхностей.

Наборы поверхностей составляют основу моделей.

Набор поверхностей можно составить из отдельных поверхностей, а также других наборов при помощи квадратных скобок. Например:

fl1 = [ face0, face1 ]
fl2 = [ face2, fl1 ]

Набор поверхностей fl1 состоит из двух поверхностей: face0 и face1. Набор поверхностей fl2 состоит из поверхности face2 и  всех поверхностей набора fl1.

Важно! Для набора поверхностей первым элементом в квадратных скобках должна быть поверхность.

fl3 = [fl1, face2] #Ошибка!
