faceRectRound

Содержание=>Скрипт 3d=>Функции 3d=>faceRectRound

Формат вызова

Face faceRectRound( Float lenght, Float width, Float radius, Float stepDegree, Matrix transfer )

Описание

Эта функция строит поверхность в виде прямоугольника с закругленными углами. Скругление выполняется путем аппроксимацией ломаной линией. Полученный прямоугольник можно преобразовать с помощью матрицы.

Параметры

  • lenght - длина
  • width - ширина
  • radius - радиус скругления углов
  • stepDegree - шаг в градусах между соседними вершинами ломаной скругления
  • transfer - заключительное преобразование прямоугольника

Пример применения

color1 = selectColor("#800000")

r1 = faceRectRound( 2, 1, 0.2, 10, matrix1 )

partModel = model( color1, color1, color1, [r1], matrix1 )
