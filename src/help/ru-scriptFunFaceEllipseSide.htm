faceEllipseSide

Содержание=>Скрипт 3d=>Функции 3d=>faceEllipseSide

Формат вызова

Face faceEllipseSide( Float radiusX, Float radiusY, Float sideCount, Matrix transfer )

Описание

Эта функция строит поверхность в виде равностороннего многоугольника вписанного в элипс. Количество сторон многоугольника определяется параметром. Полученный многоугольник можно преобразовать с помощью матрицы. Данная функция служит для построения аппроксимированных элипсов. Чем больше количество сторон - тем выше качество аппроксимации.

Параметры

  • radiusX - радиус X описанноого элипса вокруг многоугольника
  • radiusY - радиус Y описанноого элипса вокруг многоугольника
  • sideCount - количество сторон
  • transfer - заключительное преобразование многоугольника

Пример применения

#Пример окружности
color1 = selectColor("#800000")

c1 = faceEllipseSide( 3, 2, 36, matrix1 )

partModel = model( color1, color1, color1, [c1], matrix1 )
