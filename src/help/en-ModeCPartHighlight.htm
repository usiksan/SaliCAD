Highlight component

Contents=>Modes=>Plate modes=>Highlight component

Tool bar:

Menu: Plate|Highlight component

Cursor:

It is used to highlight identical components on the board. When you click on a component, all the same components on the board are highlighted, and the rest are darkened. This allows you to visually determine the placement of all the same components on the board, for example, during manual installation.

When the mouse hovers over a component, the component that will be highlighted is shown in the status bar. If there are several components under the cursor, then you can switch them out by pressing the F6 key.

1. Specify the component to highlight. If necessary, perform a search by pressing the F6 key.

  • The component is specified by clicking the left mouse button on the component.
  • The backlight is removed from the component if it was highlighted.
