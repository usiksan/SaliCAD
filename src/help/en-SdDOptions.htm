Page Path


On this page, you can select or enter paths in the file system to different components of the system. To select a path, click the appropriate Select button... You can also edit the path directly.

Name Description
Current user home path User's base directory. By default, this path is used to write and read files.
Library path
 The library is a database of a special type. This path specifies the directory in which the library files are located.
Pattern path
 Templates are used in different places of the program and have different formats. This path points to the directory where the program templates are located.
Category hierarchy path The category hierarchy is an optional and optional way to group items in a library.


Page Editors


This page contains options for graphic editors.

Name Description
Show rat net
 Unfinished connections in the PCB editor are displayed with special "rubber bands" (rat net) - special lines. This parameter determines whether the rat net will be displayed. Control of this parameter is also duplicated in the graphics editor of boards in the toolbar.
Show remarks
 If you hover the mouse on some object, a tooltip appears on this object. This option determines whether a tooltip will appear or not.
Show message
 At the bottom of the program, the status bar shows the message bar. This parameter determines whether messages will be displayed or not.
Move cursor to center when zoom
 This option determines whether the mouse cursor to move to the center of the window when you change the scale, or will remain in the same position, which is due to the change of scale will also change.
Symbol pin size
 Unconnected pins for the symbol in the schematic editors are indicated by a cross. This parameter determines the size of this cross.
Part pin size Unconnected pins for the part in the part and pcb editors are displayed in circles. This parameter determines the size of the circle.
Wire dot size In the schematic editor to connect the three or more segments of the chain at one point celebrated a major point. This parameter determines the size of this point.
Wire dot width In the circuit editors, the connection of three or more chain segments at one point is marked by a large point. This parameter determines the thickness of the line to which this point is drawn. Together with the previous parameter allows you to organize a different appearance of the connection point of the circuits. For example, a filled circle, a ring, an empty circle.
Via default type When you move from one trace layer to another, the program automatically adds vias. The type of these vias can be set in the graphics editor. This parameter specifies the type of the vias by default.


 

Page Pads


This page displays a table of pin types and the corresponding pads that make up the default pads association table.

The Pad association name field specifies the name of the association object that is used by default. To select a different association, click Select..., the object selection dialog appears.

The Pad association table is located in the Central part of the page. Table editing is not provided in this mode.
