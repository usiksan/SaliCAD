stringPadCircleThrough

Content=>Script 3d=>Functions 3d=>stringPadCircleThrough

Prototype

String stringPadCircleThrough( Float padDiameter, Float holeDiameter, Float mask )

Description

The function builds a text description of the contact pad specified by the parameters.

Parameters

  • padDiameter - the diameter of the pin pad in mm
  • holeDiameter - pin pad hole diameter
  • mask - additional mask distance. The mask window increases by this value in relation to the pin pad

Example

  pin = stringPadCircleThrough( 2, 0.6, 0.1 )

This example will form the following description of the pin pad "c2d0.6m0.1"
