Draw modes

Contents=>Modes=>Draw modes

Drawing modes are designed to insert graphic primitives into various complex objects, which give the element an easy-to-understand look. The system has the following drawing modes:

  • Segment (line) draw mode
  • Rectangle draw mode
  • Filled rectangle draw mode
  • Region draw mode
  • Filled region draw mode
  • Circle draw mode
  • Filled circle draw mode
  • Arc draw mode
  • Text and field draw mode
