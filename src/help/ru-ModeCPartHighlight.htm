Подсвечивание компонентов

Содержание=>Режимы=>Плата=>Подсвечивание компонентов

Панель инструментов:

Меню: Плата|Подсвечивание компонентов

Курсор:

Служит для подсвечивания одинаковых компонентов на плате. При нажатии на какой-либо компонент все такие-же компоненты на плате подсвечиваются, а остальные затемняются. Это позволяет визуально определить размещение всех одинаковых компонентов на плате, например в процессе ручного монтажа.

При наведении мыши на компонент в строке подсказки показывается компонент, который будет подсвечен. Если под курсором находятся несколько компонентов, то перебрать их можно путем нажатия клавиши F6.

1. Укажите компонент, который нужно подсветить. При необходимости осуществите перебор путем нажатия клавиши F6.
  • Указание компонента осуществляется нажатием левой кнопки мыши на компоненте.
  • Снимается подсветка с компонента, если он был подсвечен.


"Разумный режим" отсутствует.
