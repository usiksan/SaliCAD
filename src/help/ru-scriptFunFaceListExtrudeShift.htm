faceListExtrudeShift

Содержание=>Скрипт 3d=>Функции 3d=>faceListExtrudeShift

Формат вызова

FaceList faceListExtrudeShift( Face src, Float height )

Описание

Эта функция делает вытяжку из исходной поверхности в направлении перпендикуляра к этой поверхности, формируя тело. Исходная поверхность дублируется путем переноса в направлении перпендикуляра к исходной поверхности на заданную величину и между получившимися поверхностями строятся стены. Получившийся набор поверхностей, включая исходную поверхность и дублированную, возвращаются в виде тела.

Параметры

  • src - исходная поверхность
  • height - высота вытяжки (положительные числа - вытяжка вниз, отрицательные - вверх)

Пример применения

color1 = selectColor("#a00000")

b1 = faceRect( 3, 2, matrixRotate( matrix1, 20, 0, 1, 0 ) )
body = faceListExtrudeShift( b1, -1 )

partModel = model( color1, color1, color1, body, matrix1 )
