faceListCylinderMatrix

Content=>Script 3d=>Functions 3d=>faceListCylinderMatrix

Prototype

FaceList faceListCylinderMatrix( Float radius, Float height, Matrix transfer )

Description

This function builds a cylinder with a base on the XY plane. This cylinder is then transformed using a matrix. The circles of the cylinder bases are approximated by equilateral polygons inscribed in a circle. The number of sides of the polygon is determined automatically based on the radius to get a visually smooth surface.

Parameters

  • radius - the radius of the circumscribed circle around the polygon of the base of the cylinder
  • height - cylinder height
  • transfer - final cylinder conversion

Example

color1 = selectColor("#a00000")

c1 = faceListCylinderMatrix( 2, 5, matrixRotate( matrix1, 45, 0, 1, 0 ) )

partModel = model( color1, color1, color1, c1, matrix1 )
