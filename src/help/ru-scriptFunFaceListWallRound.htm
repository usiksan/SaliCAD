faceListWallRound

Содержание=>Скрипт 3d=>Функции 3d=>faceListWallRound

Формат вызова

FaceList faceListWallRound( Face src, Face dst, Float scaleX, Float scaleY, Float radius, Float stepDegree )

Описание

Эта функция строит стены между поверхностями src и dst формируя скругление снизу. Количество точек в исходных поверхностях должно быть одинаковым. Поверхность dst обычно получается путем масштабирования поверхности src с теми-же коэффициентами, что и указываются в вызове этой функции. Параметр radius определяет радиус скругления, а stepDegree количество градусов на шаг аппроксимации кривой поверхности.

Параметры

  • src - исходная поверхность (пол)
  • dst - целевая поверхность (потолок)
  • scaleX - масштабирование целевой поверхности по отношению к исходной по оси X
  • scaleY - масштабирование целевой поверхности по отношению к исходной по оси Y
  • radius - радиус скругления
  • stepDegree - количество градусов на шаг аппроксимации кривой поверхности

Пример применения

color1 = selectColor("#a00000")

c1 = faceCircleSide( 3, 6, matrix1 )
c2 = faceDuplicate( c1, matrixTranslate( matrixScale( matrix1, 1.3, 1.3, 1.0), 0,0,2) )

h1 = faceListWallRound( c1, c2, 1.3, 1.3, 1, -10 )

partModel = model( color1, color1, color1, h1, matrix1 )
