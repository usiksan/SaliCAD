Enter and edit text mode

Contents=>Modes=>Draw modes=>Текст

Tool bar:

Menu: Draw|Text

Cursor:

1. Setup text attributes

2. Enter text reference point

3. Enter text

4. Press Enter key or mouse left button to enter complete

Note: if the text anchor point falls on an existing text, text editing is enabled. In this case, if the alphanumeric key is pressed first, the previous text is deleted, if the cursor movement key is pressed first, the previous text is saved. The Esc key returns the previous text, pressing it again stops editing.

  • You can specify a point by clicking the left mouse button or manually.
  • The termination of the input is done by pressing the right mouse button.
