graphPin

Содержание=>Скрипт 3d=>Функции 3d=>graphPin

Формат вызова

Graph2d graphPin( FloatList [pinPosX, pinPosY], String padDescription, FloatList [pinNumberPosX,pinNumberPosY], String pinNumber, String pinNuberAttr, FloatList [pinNamePosX,pinNamePosY], String pinNameAttr )

Описание

Функция строит 2d объект в 3d пространстве. Функции данного класса предназначены для создания комплексных мастеров генерации компонентов, где в одном скрипте генерируются 3d модель и 2d изображение компонента. Для построения обычных 3d моделей использование данных функций не имеет смысла.

Все 2d объекты строятся в плоскости XY с небольшим смещением по осиZ.

Эта функция строит вывод компонента на основании своих параметров. В 3d пространстве отображается только контактная площадка. Основное построение происходит в 2d пространстве.

Параметры

  • pinPosX, pinPosY - координаты центра вывода
  • padDescription - описание контактной площадки
  • pinNumberPosX,pinNumberPosY - позиция номера вывода
  • pinNumber - номер вывода
  • pinNuberAttr - атрибуты номера вывода (выравнивание и угол поворота)
  • pinNamePosX,pinNamePosY - позиция имени вывода
  • pinNameAttr - атрибуты имени вывода

Текстовые атрибуты

Текстовые атрибуты используются для горизонтального и вертикального выравнивания текста, а также задания угла поворота текста.

"LT90"

где

  • L - горизонтальное выравнивание, может быть одним из L-по левому краю, C-по центру, R-по правому краю
  • T - вертикальное выравнивание, может быть одним из T-по верху, M-посередине, B-по низу
  • 90 - угол поворота текста в градусах, может быть одним из 0, 90, 180 и 270

partFlat = graphPin( [0,0], "c1.2d0.6m0.1", [0,0], "1", "CM0", [0,0], "CM0" )
