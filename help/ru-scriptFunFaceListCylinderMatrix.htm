faceListCylinderMatrix

Содержание=>Скрипт 3d=>Функции 3d=>faceListCylinderMatrix

Формат вызова

FaceList faceListCylinderMatrix( Float radius, Float height, Matrix transfer )

Описание

Эта функция строит цилиндр с основанием на плоскости XY. Затем этот цилиндр преобразуется с помощью матрицы. Окружности оснований цилиндра аппроксимируются равносторонними многоугольниками вписанными в окружность. Количество сторон многоугольника определяется автоматически исходя из радиуса, чтобы получить визуально гладкую поверхность.

Параметры

  • radius - радиус описанной окружности вокруг многоугольника основания цилиндра
  • height - высота цилиндра
  • transfer - заключительное преобразование цилиндра

Пример применения

color1 = selectColor("#a00000")

c1 = faceListCylinderMatrix( 2, 5, matrixRotate( matrix1, 45, 0, 1, 0 ) )

partModel = model( color1, color1, color1, c1, matrix1 )
