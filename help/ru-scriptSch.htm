Скрипт для схем

Содержание=>Скрипт для схем

Скрипт для схем служит для проведения вычислений непосредственно в схеме. Значения, полученные в процессе вычисления могут использоваться для настройки компонентов, что, в свою очередь, позволяет автоматически подстроить схему под вычисленные значения.

r = ref( "Om", "" )
u = ref( "V", "" )
i = ref( "A", "" )
i = u / r

При этом таблица значений примет вид:

Имя Значение
r
u
i

Если заполнить поля r и u значениями, то значение поля i будет вычислено автоматически:

Имя Значение
r 1.2kOm
u 1.24V
i 1.03mA

Функция ref добавляет переменную в визуальную таблицу. Переменные без данной функции в визуальную таблицу не попадают, соответственно, увидеть их значение нельзя. Такие переменные можно использовать для промежуточных значений при вычислении.
