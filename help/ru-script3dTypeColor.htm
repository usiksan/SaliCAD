Тип цвет

Содержание=>Скрипт 3d=>Тип цвет

Тип цвет позволяет представлять различные цвета. Цвет представляет собой комбинацию базовых цветов RGB. Для создания цветов используются функции.

Для цветов не допускаются никакие операции, кроме присваивания. Также цвета находят применение в вызовах функций.
