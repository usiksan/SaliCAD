Insert bus

Contents=>Modes=>Sheet specific modes=>Insert bus

Tool bar:

Menu: Sheet|Insert bus

Cursor:

It is used to simplify the input net wires for a given pattern. The pattern consists of a net wire polyline and the net wire name. Thus, this mode is very convenient for bus input.

1. Enter bus net list in "Enter bus net list" dialog (see "Enter bus net list" dialog).

2. Enter the polyline of the first net. This polyline should preferably (but not necessarily) start at the component pin and can have an arbitrary number of segments. When you are finished entering the template, right-click.

3. Set net name properties

4. Specify the relative location of the net name (relative to the polyline).

5. Thus, the pattern for all bus nets is entered, if there was only one net in the bus, then the transition to step 1 is automatically carried out. If there is more than one net in the bus, proceed to step 6.

6. Specify the starting point of the next polyline net. In this case, the rest of the polyline for this net is formed automatically, and then the name of the chain is added.

Note: the name of the net to be formed is displayed in the chain properties panel.

  • You can specify a point by clicking the left mouse button or manually.
  • The termination of the input is done by pressing the right mouse button.

"Smart mode"

Smart mode after entering the bus nets pattern, the initial points of the remaining bus nets on the free pins are searched in the direction of the cursor movement. In this case, the search is carried out clearly only in the horizontal or vertical direction.
