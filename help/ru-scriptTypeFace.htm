Тип поверхность

Содержание=>Скрипт 3d=>Тип поверхность

Поверхность представляет собой плоскость в трехмерном прострастве, ограниченную многоугольником. Все вершины этого многоугольника лежат на одной плоскости. Поверхности должны быть представлены выпуклым многоугольником

Модели в трехмерном пространстве представлены набором поверхностей.

ечислены функции для работы с вершинами:

  • vertex - служит для задания вершины по трем координатам, например: v0 = vertex( 10, 5, 4 ) создает вершину с координатами x=10, y=5, z=4 и присваивает ее переменной v0
  • vertexOffset - создает вершину путем задания смещения относительно другой вершины, например: v1 = vertexOffset( v0, 1, -6, 2 ) создает вершину с координатами смещенными относительно вершины v0. Для v0 из предыдущего примера, v1 будет иметь координаты x=11, y=-1, z=6.
