facePart

Content=>Script 3d=>Functions 3d=>facePart

Prototype

Face facePart(Face src, FloatList vertexIndexes )

Description

This function builds a polygon from the vertices of another polygon. The source polygon is set by the src parameter. The vertex Indexes parameter contains a sequential list of indexes of the vertices of the source polygon that will make up the resulting polygon.

Parameters

  • src - source polygon
  • vertexIndexes - list of indexes of the original polygon that make up the resulting polygon

Example

color1 = selectColor("#800000")
color2 = selectColor("#008000")

c1 = faceFlatMatrix( [ 0,0, 0,2, 1,1, 1,-1, 0,-2], matrix1 )

c2 = facePart( c1, [ 1, 2, 3 ] )

partModel = model( color1, color1, color1, [c1], matrix1 )
partModel = model( color2, color2, color2, [c2], matrixTranslate( matrix1, 2.2, 0, 0) )
