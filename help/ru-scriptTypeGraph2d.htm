Тип графика 2d

Содержание=>Скрипт 3d=>Тип графика 2d

Тип графика 2d используется для добавления 2d графики в скрипт создания модели корпуса. Это позволяет делать скрипты, которые генерируют не только 3d модель корпуса компонента, но и 2d проекцию. 2d графика используется в скриптах, которые служат для автоматической генерации компонентов.

Для создания 2d графики используются функции.

Для 2d графики не допускаются никакие операции, кроме присваивания.
