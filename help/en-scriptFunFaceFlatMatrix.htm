faceFlatMatrix

Content=>Script 3d=>Functions 3d=>faceFlatMatrix

Prototype

Face faceFlatMatrix( FloatList pairs, Matrix matrix )

Description

This function builds a polygon on the plane according to the given relative two-dimensional coordinates in the XY plane. The vertex of the beginning of the polygon is determined by the first pair. The pairs parameter contains a list of pairs of relative coordinates of the vertices of the polygon. Each next vertex is defined as an offset relative to the previous one. The first pair defines the X Y coordinates of the first vertex. In order for a polygon to be used as a surface, it must be convex. If the polygon is non-convex, then it must be converted to a set of convex polygons using the function faceListSimplifyThe resulting polygon can be transformed using a matrix..

Parameters

  • pairs - a pair defining the first vertex and a list of pairs of relative coordinates. [ org0,org0,  offx1,offy1,  offx2,offy2 ... ]
  • matrix - transformation matrix of the resulting polygon

Example

color1 = selectColor("#800000")

c1 = faceFlatMatrix( [ 0,0, 0,2, 1,1, 1,-1, 0,-2], matrix1 )

partModel = model( color1, color1, color1, [c1], matrix1 )
