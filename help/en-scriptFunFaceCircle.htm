faceCircle

Content=>Script 3d=>Functions 3d=>faceCircle

Prototype

Face faceCircle( Float radius, Float stepDegree, Matrix transfer )

Description

This function constructs a surface in the form of an equilateral polygon inscribed in a circle. The number of sides of the polygon is determined by the angle pitch. The resulting polygon can be transformed using a matrix. This function is used to construct approximated circles. The smaller the angle step, the higher the approximation quality. With large values of the angle step, this function can be used to build other regular polygons: an equilateral triangle, a square, etc.

Parameters

  • radius - the radius of the circumscribed circle around the polygon
  • stepDegree - angle pitch in degrees
  • transfer - final polygon transformation

Example

#Circle example
color1 = selectColor("#800000")

c1 = faceCircle( 2, 10, matrix1 )

partModel = model( color1, color1, color1, [c1], matrix1 )
