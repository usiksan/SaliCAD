Insert component

Contents=>Modes=>Sheet specific modes=>Insert component

Tool bar:

Menu: Sheet|Insert component

Cursor:

This mode used to insert components into the schematic sheet. The components in the schematic sheet are represented by a symbol implement and are a single component section.

1. Find and select component section in "Select component to insert" dialog (see "Select component to insert" dialog).

2. Setup component section properties.

3. Enter component section location point. With this component section inserted into schematic sheet.

4. Repeat steps 2 and 3 to enter the same component sections.

5. To stop entering this section of the component and return to selecting a section of the component, right-click.

6. Repeat steps 1-5 to enter different sections of the components into the schematic sheet.

Component section properties are specified using component tool bar:

  • You can specify a point by clicking the left mouse button or manually.
  • The termination of the input is done by pressing the right mouse button.

"Smart" mode not present.
