faceFlat

Содержание=>Скрипт 3d=>Функции 3d=>faceFlat

Формат вызова

Face faceFlat( Vertex origin, Float orientation, FloatList pairs )

Описание

Эта функция строит многоугольник в плоскостях XY, YZ или XZ по заданным относительным двумерным координатам. Вершина начала многоугольника определяется параметром origin. Параметр orientation задает плоскость, в которой будет выполнено построение. Параметр pairs содержит список пар относительных координат вершин многоугольника. Каждая следующая вершина определяется как смещение относительно предыдущей. Первая пара определяет смещение относительно вершины origin. Для того, чтобы многоугольник мог использоваться в качестве поверхности он должен быть выпуклым. Если многоугольник невыпуклый, то его необходимо преобразовать к набору выпуклых многоугольников с помощью функции faceListSimplify.

Параметры

  • origin - начальная вершина многоугольника
  • orientation - код ориентации поверхности. 0 - плоскость XY, 1 - плоскость YZ, 2 - плоскость XZ
  • pairs - список пар относительных координат. [ offx0,offy0,  offx1,offy1,  offx2,offy2 ... ]

Пример применения

color1 = selectColor("#800000")

c1 = faceFlat( vertex(0,0,0), 0, [ 0,2, 1,1, 1,-1, 0,-2] )

partModel = model( color1, color1, color1, [c1], matrix1 )
