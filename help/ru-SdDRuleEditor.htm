Диалог Редактор правил трассировки

При проведении трассировки проводников для контроля параметров трассировки используется набор правил. Эти правила включают в себя:

  • ширина дорожки по умолчанию
  • зазор между дорожкой и дорожкой
  • зазор между дорожкой и площадкой
  • зазор между площадкой и площадкой

Правила могут быть назначены для каждой цепи индивидуально. Если для конкретной цепи правила не назначены, то используются правила для платы, которые назначены по умолчанию и могут быть изменены.

Данный диалог используется для редактирования правил цепей и платы. Полный список цепей представлен в списке слева. Для редактирования параметров цепи выберите цепь в этом списке. Допускается выбор нескольких цепей. В этом случае правила будут установлены для всех выбранных цепей.

Группа полей справа служит для изменения правил выбранных цепей (левая колонка) и платы целиком (правая колонка).

Для ускоренного задания зазоров, если они все одинаковые служат поля Все зазоры. После ввода этих зазоров нужно нажать кнопку Применить зазоры и все зазоры заполнятся заданным зазором.

Кнопка Все наследовать служит, чтобы для выбранных цепей установить что все правила не назначены. В этом случае будут использованы правила платы.
