Диалог Редактор правил трассировки


Номинал некоторых элементов (резисторов, конденсаторов, индуктивностей) соответствует некоторму "ряду". "Ряд" - это набор стандартных значений, которые повторяются для каждого порядка. Данный диалог позволяет выбрать одно значение из заданного ряда.

В левой части отражены группы значений. Каждая группа покрывает несколько порядков, в зависимости от принятых соглашений. Например, для резисторов - это по три порядка на группу. Для конденсаторов - по три и шесть порядков.

В правой части перечислены значения из выбранной группы.
