selectColor

Содержание=>Скрипт 3d=>Функции 3d=>colorFromString

Формат вызова

Color colorFromString( String colorString )

Описание

Эта функция строит цвет по его текстовому представлению. Функция selectColor выполняет ту-же работу, только еще позволяет выбрать цвет в диалоговом режиме. Поэтому использование функции selectColor предпочтительнее.

Параметры

  • colorString - текстовое представление цвета в формате "#rrggbb", где rr - шестнадцатиричный код красного цвета, gg и bb - коды зеленого и голубого цветов соответственно

Пример применения

bodyColor1 = colorFromString( "#202020" )
