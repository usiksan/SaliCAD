Move ident

Contents=>Modes=>Plate modes=>Move ident of components

Tool bar:

Menu: Plate|Move ident of components

Cursor:

Is used to transfer identifiers of the component in the plate. Each identifier has a default location. It is set relative to the component, so when the component is moved or rotated, the ID is moved and rotated with it. To change the position of the relative component identifier, use this mode.

1. Specify the component whose ID you want to move

2. Setup identifier properties

3. Set identifier location

4. Repeat steps 1-3 until all identifiers are placed

  • You can specify a point by clicking the left mouse button or manually.
  • The termination of the input is done by pressing the right mouse button.

Identifier properties are set using the text properties bar.

"Smart mode" not present.
