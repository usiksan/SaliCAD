Система PasCAD является системой автоматизированного проектирования схем 
радиоэлектронных устройств с возможностью сквозного проектирования печатных плат.

Этот файл содержит общие замечания по установочному комплекту, а также последние 
изменения, не вошедшие в печатную документацию.

Для установки системы на Ваш компьютер запустите программу автоматической установки 
Setup, расположенную в корневом каталоге компакт-диска. В дальнейшем следуйте указаниям 
программы установки.

Установочный компакт-диск содержит также программу демонстрации возможностей системы. 
Она запускается из программы установки. Для работы программы демонстрации 
необходим Sound Blaster или иное 
устройство воспроизведения файлов WAV (стандарт для MS Windows).
