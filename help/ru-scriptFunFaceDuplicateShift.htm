faceDuplicateShift

Содержание=>Скрипт 3d=>Функции 3d=>faceDuplicateShift

Формат вызова

Face faceDuplicateShift( Face src, Float shift)

Описание

Эта функция копирует исходную поверхность сдвинутую на заданную величину в направлении перпендикуляра к поверхности. Исходная поверхность может иметь любую ориентацию.

Параметры

  • src - исходная поверхность
  • shift - направление и величина смещения

Пример применения

color1 = selectColor("#800000")
color2 = selectColor("#008000")

c1 = faceCircle( 2, 10, matrix1 )

c2 = faceDuplicateShift( c1, 1 )

partModel = model( color1, color1, color1, [c1], matrix1 )
partModel = model( color2, color2, color2, [c2], matrix1 )
