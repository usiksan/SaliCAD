faceEquidistantXY

Content=>Script 3d=>Functions 3d=>faceEquidistantXY

Prototype

Face faceEquidistantXY( Face src, Float distance, Matrix matrix )

Description

This function constructs a polygon in the XY plane equidistant to the src polygon also in the XY plane. The new surface is transformed using a matrix. By building walls between the source and the resulting surfaces, you can get a chamfer.

Parameters

  • src - the original surface in the XY plane
  • distance - the distance between the faces of the source and the resulting polygon. Positive numbers define the construction inside the original polygon (decrease), and negative numbers - outside (increase)
  • matrix - transformation matrix of the resulting polygon (surface)

Example

color1 = selectColor("#800000")
color2 = selectColor("#008000")

c1 = faceCircle( 2, 10, matrix1 )

c2 = faceEquidistantXY( c1, -0.5, matrixTranslate( matrix1, 0, 0, 3 ) )

partModel = model( color1, color1, color1, [c1], matrix1 )
partModel = model( color2, color2, color2, [c2], matrix1 )
