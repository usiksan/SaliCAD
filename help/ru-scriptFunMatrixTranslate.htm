matrixTranslate

Содержание=>Скрипт 3d=>Функции 3d=>matrixTranslate

Формат вызова

Matrix matrixTranslate( Matrix src, Float offX, Float offY, Float offZ )

Описание

Эта функция строит строит матрицу перемещения на основе исходной матрицы. На основании параметров offX, offY и offZ функция строит матрицу перемещения.  Перемещение определяется вектором с координатами off.

Параметры

  • src - исходная матрица
  • offX - x-компонент вектора перемещения
  • offY - y-компонент вектора перемещения
  • offZ - z-компонент вектора перемещения

Пример применения

matrix2 = matrixTranslate( matrix1, 2, 0, 0 )

Выполняет сдвиг на 2 по оси X.
