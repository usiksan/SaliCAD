faceListPinCurveTwo

Содержание=>Скрипт 3d=>Функции 3d=>faceListPinCurveTwo

Формат вызова

FaceList faceListPinCurveTwo( Face src, Float firstLen, Float middleLen, Float lastLen, Float radius, Float angleFirst, Float angleSecond, Float sideCount )

Описание

Эта функция строит вывод по его основанию. На первом этапе основание вытягивается вверх на величину firstLen, затем загибается на угол angleFirst по радиусу radius, при этом для загиба используются sideCount поверхностей для аппроксимации кривой, затем происходит вытягивание на величину middleLen, затем - второй загиб на угол angleSecond с тем-же радиусом и количеством поверхностей, что и для первого загиба и, наконец, вытяжка на величину lastLen. Обычно эта функция используется для построения выводов, но может использоваться также и для построения других тел.

Параметры

  • src - исходная поверхность
  • firstLen - величина вытяжки вверх
  • middleLen - величина средней вытяжки
  • lastLen - величина финальной вытяжки
  • radius - радиус загиба
  • angleFirst - угол первого загиба
  • angleSecond - угол второго загиба
  • sideCount - количество сторон для аппроксимации кривой загиба

Пример применения

color1 = selectColor("#a00000")

c1 = faceRect( 0.2, 1, matrix1 )

pin = faceListPinCurveTwo( c1, 2, 3, 2, 1, 90, -90, 10 )

partModel = model( color1, color1, color1, pin, matrix1 )
