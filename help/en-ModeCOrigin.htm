Tool bar:

Menu: Symbol|Origin point

Cursor:

The origin point serves as the reference point for the library symbol. Usually it should be market on the left lower leg. By default, the origin point is assumed to be at 0,0 coordinates.

1. Enter new placement for origin point

"Smart" mode not present.

  • You can specify a point by clicking the left mouse button or manually.
  • The termination of the input is done by pressing the right mouse button.
