faceBevelXY

Содержание=>Скрипт 3d=>Функции 3d=>faceBevelXY

Prototype

Face faceBevelXY( Face src, Float radius )

Description

This function builds a surface in the XY plane from the original surface also in the XY plane. The new surface is shifted along the Z axis by the radius, in addition, it is scaled along the X and Y axes by the radius. By building walls between the source and the resulting surfaces, you can get a chamfer. There is also a function that generates a rounding between these two surfaces.

Parameters

  • src - the original surface in the XY plane
  • radius - radius of rounding or chamfer size

Example


color1 = selectColor("#800000")
color2 = selectColor("#008000")

c1 = faceCircle( 2, 10, matrix1 )

c2 = faceBevelXY( c1, -0.5 )

partModel = model( color1, color1, color1, [c1], matrix1 )
partModel = model( color2, color2, color2, [c2], matrix1 )
