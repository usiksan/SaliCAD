faceDuplicateScale

Содержание=>Скрипт 3d=>Функции 3d=>faceDuplicateScale

Формат вызова

Face faceDuplicateScale( Face src, Float scaleX, Float scaleY, Float radius)

Описание

Эта функция выполняет масштабирование размеров поверхности по осям X и Y. Затем результирующая поверхность смещается по оси Z на величину радиуса.  Исходная поверхность не обязательно должна быть в плоскости XY, но преобразования размеров ведутся по проекции именно на эту плоскость.

Параметры

  • src - исходная поверхность
  • scaleX - коэффициент масштабирования по оси X
  • scaleY - коэффициент масштабирования по оси Y
  • radius - величина смещения по оси Z

Пример применения

color1 = selectColor("#800000")
color2 = selectColor("#008000")

c1 = faceCircle( 2, 10, matrix1 )

c2 = faceDuplicateScale( c1, 0.5, 2.0, 1 )

partModel = model( color1, color1, color1, [c1], matrix1 )
partModel = model( color2, color2, color2, [c2], matrix1 )
