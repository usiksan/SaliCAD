Вставка шин

Содержание=>Режимы=>Специальные режимы схемы=>Вставка шин

Панель инструментов:

Меню: Лист схемы|Вставить шину

Курсор:

Служит для упрощенного ввода цепей по заданному шаблону. Шаблон состоит из ломаной цепи и имени цепи. Таким образом, этот режим очень удобен для ввода шин.

1. В диалоговом окне "Ввод цепей для шины" укажите список цепей шины (см.Диалоговое окно "Ввод цепей для шины").

2. Введите ломаную первой цепи. Эта ломаная желательно (но не обязательно) должна начинаться на выводе компонента и может иметь произвольное число сегментов. По завершении ввода шаблона нажмите правую кнопку мыши.

3. Установите атрибуты имени цепи

4. Укажите относительное местоположение имени цепи (относительно ломаной).

5. Таким образом шаблон для всех цепей шины оказывается введенным, если в шине была только одна цепь, то автоматически осуществляется переход к шагу 1. Если в шине более чем одна цепь, то переход к шагу 6.

6. Укажите начальную точку ломаной очередной цепи. При этом вся остальная часть ломаной для этой цепи формируется автоматически, а затем добавляется еще и имя цепи.

Замечание: Имя формируемой цепи отображается в панели свойств цепи.

  • Указание точки осуществляется нажатием левой кнопки мыши или вручную.
  • Прекращение ввода осуществляется нажатием правой кнопки мыши.

"Разумный режим"

Разумный режим после ввода шаблона цепей шины осуществляется поиск для начальных точек остальных цепей шины на свободных ножках в направлении движения курсора. При этом поиск ведется четко только в горизонтальном или вертикальном направлении.
