Справка по режимам

Содержание=>Справка по режимам

Система имеет развитую структуру режимов поддерживающую удобный стиль работы, кроме того некоторые режимы имеют специальные Разумные режимы, кардинально упрощающие ввод и редактирование на всех этапах.

Система поддерживает следующие классы режимов:

  • Режимы рисования
  • Режимы ввода объектов символа
  • Режимы ввода объектов корпуса
  • Режимы ввода объектов компонентов
  • Режимы ввода объектов схемы
  • Режимы ввода объектов платы
  • Режимы ввода объектов проекта
  • Режим редактирования
  • Режимы изменения вида

Для каждого режима указан метод вхождения в режим (это может быть меню или панель инструментов), курсор, который действует в этом режиме, разумный режим, который действует в этом режиме.
