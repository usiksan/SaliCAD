Размещение корпуса компонента

Содержание=>Режимы=>Плата=>Размещение корпуса компонента

Панель инструментов:

Меню: Плата|Разместить корпус

Курсоры:

  • когда курсор находится над компонентом
  • когда курсор находится вне какого-нибудь компонента
  • при переносе компонента
  • при выделении группы компонентов

Данный режим служит для размещения компонентов на плате, т.е. для расстановки. Расстановка осуществляется вручную с широким применением технологий автоматизации, что позволяет сохранить качество ручной расстановки и повысить скорость до автоматической расстановки.

1. Выберите компонент или группу компонентов для размещения. Такой выбор может осуществляться несколькими способами:

1.1 Наведите курсор мыши на компонент и нажмите левую кнопку. При этом компонент выделяется и переносится вместе с движением курсора мыши.

1.2 Чтобы выбрать группу компонентов нажмите левую кнопку мыши вне любого компонента и не отпуская ее охватите прямоугольником выделения требуемую группу. После отпускания левой кнопки мыши группа компонентов выделяется и переносится вместе с движением курсора мыши.

1.3 Введите идентификатор компонента в поле I панели свойств корпуса. При этом компонент выделяется и переносится вместе с движением курсора мыши.

1.4 Выберите компонент в списке компонентов в поле I. При этом компонент выделяется и переносится вместе с движением курсора мыши.

1.5 Выберите компонент по схеме. Эта уникальная возможность предоставляется только в системе SaliCAD. Включите режим выбора компонентов из схемы - кнопка G в панели свойств корпуса, выберите лист схемы, из которого нужно выбрать компонент - список H панели свойств корпуса и при нажатии кнопкой мыши вне любого компонента автоматически откроется выбранный лист схемы. Нажмите левой кнопкой мыши на компонент в схеме, который необходимо разместить. При этом лист схемы закрывается, компонент выделяется и переносится вместе с движением курсора.

2. Задайте атрибуты для корпуса.

3. Укажите точку вставки корпуса. При этом корпус вставляется в плату.

4. Повторяйте шаги 1-3 для расстановки всех корпусов.

Атрибуты корпуса задаются с помощью панели свойств корпуса:

  • Указание точки осуществляется нажатием левой кнопки мыши или вручную.
  • Прекращение ввода осуществляется нажатием правой кнопки мыши.

"Разумный" режим.

Разумный режим служит для помощи выбора следующего компонента для вставки. Когда активна функция последовательного выбора - кнопка F панели свойств корпуса, то разумный режим предлагает для следующего компонента компонент с очередным порядковым номером идентификатора, например, если был выбран компонент R35, то разумный режим предложит для следующего компонента R36.

Когда активна функция логического подбора компонента - кнопка E панели свойств корпуса, то разумный режим предлагает для вставки ближайший в схеме локальный неразмещенный компонент.

Средняя кнопка мыши или "пробел" - подтверждение "разумного" ввода.
