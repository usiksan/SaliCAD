Layer constructor Dialog

In this dialog You can construct  new layer (layers).

Layer constructing possible with builtin class and modifier and also by enter any user id.

The most preferred is to use the built-in class and modifier. Warning! The tracing layer can only be constructed using the built-in modifier.

To construct a layer, select the layer class and then the optional modifier. If You don't want to use a modifier, i.e. only class enough, then as a modifier, select "---". The layer ID constructed in this way is displayed in the Layer ID field.

The Layer name field displays the name of the layer corresponding to the current identifier. You can specify your own name of the layer in the national language. This name will be stored on your local computer and will be displayed instead of the identifier.

To create an arbitrary identifier instead of the built-in class and modifier, select the User ID class at the end of the class list.
