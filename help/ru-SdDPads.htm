Диалог Мастер площадок

Данное диалоговое окно содержит мастер создания и редактирования площадок. см.Площадки

В левой части окна - список параметров контактной площадки, а справа - области предварительного просмотра: в разрезе и вид сверху.

При изменении любого параметра внешний вид площадки автоматически перестраивается, показывая ее актуальный вид.

Наименование
 Описание
Круглая площадка или прямоугольная
 Если данный параметр отмечен, то будет формироваться круглая площадка. В противном случае - прямоугольная.
Штыревая площадка или планарная Штыревая площадка предназначена для штыревых выводов и переходных отверстий. Она характеризуется наличием отверстия и отсутствием отверстия в трафарете для нее. В планарной площадке нету отверстий, но имеется трафарет для нанесения паяльной пасты. Если данный параметр отмечен, то будет формироваться штыревая площадка. В противном случае - планарная.
Центр площадки X
 Площадка строится симметричной относительно центра. По умолчанию центр площадки совпадает с позицией вывода. Иногда, в зависимости от вывода площадку нужно сместить. Данный параметр задает значение такого смещения по оси X.
Центр площадки Y
 Площадка строится симметричной относительно центра. По умолчанию центр площадки совпадает с позицией вывода. Иногда, в зависимости от вывода площадку нужно сместить. Данный параметр задает значение такого смещения по оси Y.
Диаметр площадки
Для круглой площадки здесь задается ее диаметр.
Ширина площадки Для прямоугольной площадки здесь задается ширина площадки. Ось X.
Высота площадки
 Для прямоугольной площадки здесь задается высота площадки. Ось Y.
Зазор маски
 Область отсутствия паяльной маски имеет тот-же контур, что и площадка. Зазор определяет, насколько эта область больше (меньше), чем площадка. На практике эта область несколько больше, чем площадка.
Зазор трафарета
 Трафарет для паяльной пасты состоит из отверстий, через которые паста нанасится на площадки. Расположение и форма отверстий обычно совпадают с площадками. Данный параметр определяет, насколько эта область меньше (больше), чем площадка. На практике отверстие трафарета несколько меньше, чем площадка.
Рядов отверстий трафарета Если площадка слишком большая, то отверстие в трафарете разбивают на массив мелких отверстий. Данный параметр показывает количество рядов отверстий в таком массиве.
Колонок отверстий трафарета Если площадка слишком большая, то отверстие в трафарете разбивают на массив мелких отверстий. Данный параметр показывает количество колонок отверстий в таком массиве.
Ширина отверстия в массиве отверстий Если площадка слишком большая, то отверстие в трафарете разбивают на массив мелких отверстий. Данный параметр показывает ширину каждого мелкого отверстия в таком массиве.
Высота отверстия в массиве отверстий Если площадка слишком большая, то отверстие в трафарете разбивают на массив мелких отверстий. Данный параметр показывает высоту каждого мелкого отверстия в таком массиве.
Диаметр отверстия Диаметр отверстия для штыревого вывода или переходного отверстия.
