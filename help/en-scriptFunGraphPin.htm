graphPin

Contents=>Functions=>graphPin

Prototype

Graph graphPin( Vertex pinPosition, String pinPadDescription, Vertex pinNumberPosition, String pinNumber, String pinNumberAttributes, Vertex pinNamePosition, String pinNameAttributes)

Description

This function builds Graph pin from its params.

Parameters

  • pinPosition - pin position vertex
  • pinPadDescription - pin pad description
  • pinNumberPosition - pin number position vertex
  • pinNumber - pin number
  • pinNumberAttributes - pin number attributes, see text attributes
  • pinNamePosition - pin name position
  • pinNameAttributes - pin name attributes, see text attributes

Text attributes

Text attributes are used to set the horizontal and vertical alignment of the text, as well as to set the angle of rotation of the text.

"LT90"

where

  • L - horizontal alignment, may be one of L-left, C-center, R-right
  • T - vertical alignment, may be one of T-top, M-middle, B-bottom
  • 90 - angle of text rotation in degree, may be one of 0, 90, 180 and 270
