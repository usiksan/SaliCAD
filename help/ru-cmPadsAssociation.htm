Ассоциации контактных площадок

Содержание=>Режимы=>Плата=>Ассоциации контактных площадок

Панель инструментов:

Система SaliCAD поддерживает описательную систему контактных площадок. Это означает, что для каждого вывода корпуса можно задать описание контактной площадки. На основе описания контактной площадки система строит изображение контактной площадки. В некоторых случаях необходимо заменить контактную площадку формируемую по умолчанию на другую. Для этих целей служит ассоциация контактных площадок. Для каждого описания контактной площадки можно задать любое изображение контактной площадки в процессе проектирования платы. Данный режим открывает редактор этих ассоциаций.
