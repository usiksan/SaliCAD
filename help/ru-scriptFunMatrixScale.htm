matrixScale

Содержание=>Скрипт 3d=>Функции 3d=>matrixScale

Формат вызова

Matrix matrixScale( Matrix src, Float scaleX, Float scaleY, Float scaleZ )

Описание

Эта функция строит строит матрицу масштабирования исходной матрицы. На основании параметров scaleX, scaleY и scaleZ выполняется масштабирование исходной матрицы src. Если параметр масштаба равен 1.0, то масштабирование по данной оси не производится, если он меньше 1, то производится уменьшение, а если больше - увеличение.

Параметры

  • src - исходная матрица, к которой добавляется поворот
  • scaleX - масштаб по оси x
  • scaleY - масштаб по оси y
  • sclaeZ - масштаб по оси z

Пример применения

matrix2 = matrixScale( matrix1, 2.0, 1, 1 )

Строит новую матрицу путем добавления к исходной единичной матрице matrix1 увеличения в два раза по оси X.
