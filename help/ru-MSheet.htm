Режимы редактора схем

Содержание=>Режимы=>Режимы схем

Данный список содержит специфические режимы для схем:

  • Ввод фрагментов
  • Ввод компонентов
  • Ввод цепей
  • Ввод шин (несколько цепей в пакетном режиме)
  • Отключение цепей
  • Вставка имен цепей
  • Перенос идентификатора компонента
  • Перенос значения компонента
