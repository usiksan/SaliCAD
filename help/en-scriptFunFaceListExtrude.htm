faceListExtrude

Content=>Script 3d=>Functions 3d=>faceListExtrude

Prototype

FaceList faceListExtrude( Face src, Matrix transfer )

Description

This function makes an extrude from the original surface, forming a body. The original surface is duplicated using a matrix and walls are built between the resulting surfaces. The resulting set of surfaces, including the original surface and the duplicated one, are returned as a body.

Parameters

  • src - initial surface
  • transfer - transformation matrix

Example

color1 = selectColor("#a00000")

b1 = faceRect( 3, 2, matrix1 )
body = faceListExtrude( b1, matrixTranslate( matrix1, 1, 0, 1 ) )

partModel = model( color1, color1, color1, body, matrix1 )
