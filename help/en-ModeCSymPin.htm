Tool bar:

Menu: Symbol|Pin

Cursor:

The symbol pin specifies the point to which the nets are connected in the schematic diagram of the element.

1. Setup pin properties

2. Enter pin location point

3. Setup pin name properties

4. Enter pin name location point, when entering the pin name cursor is

5. Enter pin name and press Enter or mouse left button at end

6. Setup pin number properties

7. Enter pin number location point, when entering the pin number cursor is

For setup pin properties there is symbol pin properties bar:

  • You can specify a point by clicking the left mouse button or manually.
  • The termination of the input is done by pressing the right mouse button.

"Smart" mode.

When entering a pin location point, this point is suggested by the "smart" mode at the end of the nearest line segment.

When you enter the pin name and pin number location points, these points are suggested relative to the pin location point in the same way as the previous pin. This mode is particularly useful for serial pins where the relative position of the name and number is the same.

When you enter the name of the pin, the name of the previous pin is suggested with an index greater than 1, that is, if the previous name of the pin was, for example, A1, then the proposed name will be A2. This mode is convenient for sequential input of pins of different buses.

To enter pin name and pin number properties there is text properties bar.

Middle mouse button or "space" - confirmation of "smart" input.
