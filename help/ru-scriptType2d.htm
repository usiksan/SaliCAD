Тип 2d объект

Содержание=>Скрипт 3d=>Тип 2d объект

Тип 2d объект позволяет хранить различные 2d объекты: отрезки, прямоугольники, окружности, ножки микросхем. Для создания этих объектов используются функции.

Для 2d объектов не допускаются никакие операции, кроме присваивания. Также 2d объекты находят применение в вызовах функций.
