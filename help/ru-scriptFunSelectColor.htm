selectColor

Содержание=>Скрипт 3d=>Функции 3d=>selectColor

Формат вызова

Color selectColor( String colorString )

Описание

Эта функция строит цвет по его текстовому представлению. Если курсор находится на этой функции, то нажатие кнопки F2 вызывает показ диалогового окна выбора цвета. После выбора цвета через диалог, он заменяется в тексте скрипта.

Параметры

  • colorString - текстовое представление цвета в формате "#rrggbb", где rr - шестнадцатиричный код красного цвета, gg и bb - коды зеленого и голубого цветов соответственно

Пример применения

bodyColor1 = selectColor( "#202020" )
