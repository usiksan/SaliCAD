Показать площадки

Содержание=>Режимы=> Режимы редактирования корпуса=>Показать площадки

Панель инструментов:

Включает-отключает показ площадок вместе с выводами.

Система SaliCAD для каждого вывода позволяет задать описание контактной площадки, соответствующей этому выводу. На основе этого описания графические редакторы SaliCAD строят изображение контактных площадок. Когда показ контактной площадки включен, то ее изображение показывается вместе с выводом. Когда показ контактной площадки отключен, то контактные площадки становятся не видны, т.е. выводы показываются без контактных площадок.
