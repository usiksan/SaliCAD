faceDuplicateGrow

Содержание=>Скрипт 3d=>Функции 3d=>faceDuplicateGrow

Prototype

Face faceDuplicateGrow( Face src, Float growX, Float growY, Matrix mat)

Description

This function increases/decreases the size of the surface along the X and Y axes. Then the resulting surface is transformed using a matrix into another surface. The original surface does not have to be in the XY plane, but the size transformations are carried out by projection on this plane.

Parameters

  • src - initial surface
  • growX - a number with a positive sign is an increase in size along the X axis, with a negative sign is a decrease
  • growY - a number with a positive sign is an increase in size along the Y axis, with a negative sign is a decrease
  • mat - transformation matrix

Example

color1 = selectColor("#800000")
color2 = selectColor("#008000")

c1 = faceCircle( 2, 10, matrix1 )

mat = matrixTranslate( matrix1, 5, 0, 0 )

c2 = faceDuplicateGrow( c1, 0.5, -1, mat )

partModel = model( color1, color1, color1, [c1], matrix1 )
partModel = model( color2, color2, color2, [c2], matrix1 )
