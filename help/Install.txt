Установка системы.

Минимальные требования к системе:
IBM-совместимый компьютер с процессором не хуже 486,
установленная операционная система Microsoft Windows 9x,
свободного дискового пространства не менее 3МБ.

Установка системы осуществляется автоматически специальной программой Setup, которая 
находится на Вашем компакт-диске. Для установки выполните следующие шаги:
  вставьте в дисковод компакт-диск;
  нажмите мышью кнопку Пуск, расположенную в левой нижней части экрана;
  в появившемся меню выберите пункт Выполнить...;
  в появившемся диалоговом окне нажмите кнопку Обзор...;
  в появившемся диалоговом окне Обзор выберите компакт-диск, в каталоге Setup распологается 
программа Setup, откройте этот каталог, нажмите на программу Setup и нажмите на кнопку 
Открыть;
  в диалоговом окне Запуск программы нажмите кнопку ОК, при этом запускается программа 
автоматической установки системы;
  в дальнейшем следуйте указаниям программы установки.
