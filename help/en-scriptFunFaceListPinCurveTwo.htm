faceListPinCurveTwo

Content=>Script 3d=>Functions 3d=>faceListPinCurveTwo

Prototype

FaceList faceListPinCurveTwo( Face src, Float firstLen, Float middleLen, Float lastLen, Float radius, Float angleFirst, Float angleSecond, Float sideCount )

Description

This function builds the output based on its base. At the first stage, the base is pulled up by the value of firstLen, then it is bent by the angle angleFirst along the radius radius, while sideCount surfaces are used for bending to approximate the curve, then it is pulled by the value middleLen, then the second bend by the angle angleSecond with the same radius and number of surfaces as for the first bend and, finally, the hood by the value of lastLen. Usually this function is used to build conclusions, but it can also be used to build other bodies.

Parameters

  • src - initial surface
  • firstLen - the value of the hood up
  • middleLen - the value of the horizontal hood
  • lastLen - the value of the final hood
  • radius - bend radius
  • angleFirst - first bend angle
  • angleSecond - second bend angle
  • sideCount - number of sides to approximate the bend curve

Example

color1 = selectColor("#a00000")

c1 = faceRect( 0.2, 1, matrix1 )

pin = faceListPinCurveTwo( c1, 2, 3, 2, 1, 90, -90, 10 )

partModel = model( color1, color1, color1, pin, matrix1 )
