faceListRotation

Содержание=>Скрипт 3d=>Функции 3d=>faceListRotation

Формат вызова

FaceList faceListRotation( Float angleStart, Float angleStop, Float sideCount, Matrix translate, FloatList profilePairs )

Описание

Эта функция строит набор повернутых профилей на базе исходного профиля. Данная функция вместе с faceListWalls служит для построения поверхностей вращения. Исходный профиль строится на плоскости XZ. Затем формируется набор точек поворота путем последовательного поворота исходного профиля вокруг оси OZ на угол, равный шагу угла (разница начального и конечного углов деленная на количество сторон). Результирующий набор точек преобразуется с помощью матрицы.

Параметры

  • angleStart - начальный угол профиля
  • angleStop - конечный угол профиля
  • sideCount - количество сторон аппроксимации поворота
  • translate - матрица результирующего преобразования
  • profilePairs - набор пар значений, которые определяют вершины исходного профиля. Каждая пара представляет собой смещение вершины относительно предыдущей вершины

Пример применения

color1 = selectColor("#a00000")

pin = faceListRotation( 0, 80, 20, matrix1, [ 0.5,0, 0.5,0, 0,1, -0.5,0 ] )

pin = faceListWalls( pin, false )

partModel = model( color1, color1, color1, pin, matrix1 )
