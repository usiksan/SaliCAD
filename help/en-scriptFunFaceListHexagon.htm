faceListHexagon

Content=>Script 3d=>Functions 3d=>faceListHexagon

Prototype

FaceList faceListHexagon( Float lenght, Float topLenght, Float height, Float width )

Description

This function builds a hexagonal component body.

Parameters

  • lenght - the length of the body in the middle part
  • topLenght - the length of the body in the upper and lower parts (assumed to be the same)
  • height - body height
  • width - body width

Example

color1 = selectColor("#a00000")

body = faceListHexagon( 4, 3.6, 1, 3 )

partModel = model( color1, color1, color1, body, matrix1 )
