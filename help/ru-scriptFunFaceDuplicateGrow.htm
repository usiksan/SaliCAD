faceDuplicateGrow

Содержание=>Скрипт 3d=>Функции 3d=>faceDuplicateGrow

Формат вызова

Face faceDuplicateGrow( Face src, Float growX, Float growY, Matrix mat)

Описание

Эта функция выполняет увеличение/уменьшение размеров поверхности по осям X и Y. Затем результирующая поверхность преобразуется с помощью матрицы в другую поверхность. Исходная поверхность не обязательно должна быть в плоскости XY, но преобразования размеров ведутся по проекции именно на эту плоскость.

Параметры
  • src - исходная поверхность
  • growX - число с положительным знаком - увеличение размера по оси X, с отрицательным знаком - уменьшение
  • growY - число с положительным знаком - увеличение размера по оси Y, с отрицательным знаком - уменьшение
  • mat - матрица преобразования

Пример применения

color1 = selectColor("#800000")
color2 = selectColor("#008000")

c1 = faceCircle( 2, 10, matrix1 )

mat = matrixTranslate( matrix1, 5, 0, 0 )

c2 = faceDuplicateGrow( c1, 0.5, -1, mat )

partModel = model( color1, color1, color1, [c1], matrix1 )
partModel = model( color2, color2, color2, [c2], matrix1 )
