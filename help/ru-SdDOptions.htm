Страница Пути

На данной странице Вы можете выбрать или ввести пути в файловой системе к различным компонентам системы. Чтобы выбрать путь нажмите на соответствующую кнопку Выбрать... Можно также редактировать путь напрямую.

Наименование
 Описание
Текущий домашний путь Базовый каталог пользователя. По умолчанию по данному пути производится запись и чтение файлов.
Путь к библиотекам
 Библиотеки представляют собой базу данных специального типа. Данным путь указывает каталог, в котором расположены файлы библиотек.
Путь к шаблонам
 Шаблоны используются в различных местах программы и имеют различный формат. Данный путь указывает на каталог, в котором расположены шаблоны программы.
Путь к категориям Иерархия категорий - это дополнительный и не обязательный способ группировки элементов в библиотеке


Страница Редакторы

На данной странице сосредоточены параметры для графических редакторов.

Наименование
 Описание
Показывать "резинки"
 Незавершенные связи в редакторе печатных плат отображаются специальными "резинками" - специальными линиями. Данный параметр определяет будут ли отображаться резинки. Управление данным параметром также продублировано в графическом редакторе плат в панели инструментов.
Показывать подсказки
 Если задержать курсор на каком-то объекте, то появляется всплывающая подсказка по данному объекту. Данный параметр определяет будет появляться всплывающая подсказка или нет.
Показывать сообщения
 В нижней части программы в строке состояния показывается строка сообщений. Данным параметр определяет будут ли показываться сообщения или нет.
Перемещать курсор в центр при изменении масштаба
 Данный параметр определяет будет ли курсор мыши перемещаться в центр окна при изменении масштаба или будет оставаться на прежней позиции, которая из-за смены масштаба также изменится.
Размер вывода для символа
 В схемных редакторах неподключенный вывод обозначается крестиком. Данный параметр определяет размеры этого крестика.
Размер вывода корпуса В редакторах корпуса и платы ножки корпуса компонента (точнее точка подключения цепи) отображаются кружками. Данный параметр определяет размеры данного кружка.
Размер точки соединения цепей В схемных редакторах соединение трех и более сегментов цепи в одной точке отмечается крупной точкой. Данный параметр определяет размер этой точки.
Толщина линии точки соединения цепей В схемных редакторах соединение трех и более сегментов цепи в одной точке отмечается крупной точкой. Данный параметр определяет толщину линии, которой рисуется данная точка. Вместе с предыдущим параметром позволяет организовать различный внешний вид точки соединения цепей. Например, залитый кружок, кольцо, пустой кружок.
Тип переходного отверстия по умолчанию При переходе с одного трассировочного слоя на другой программа автоматически добавляет переходные отверстия. Тип этих отверстий можно задавать в графическом редакторе плат. Данный параметр задает тип переходного отверстия по умолчанию.


Страница Площадки

На данной странице отображается таблица типов выводов и соответствующие им контактные площадки, составляющие таблицу контактных площадок по умолчанию.

В поле Имя ассоциации площадок задается имя объекта ассоциации, используемого по умолчанию. Чтобы выбрать другую ассоциацию нажмите кнопку Выбрать..., при этом появляется диалог выбора объекта.

В центральной части страницы расположена таблица ассоциации. Редактирование таблицы в данном режиме не предусмотрено.
