faceListDuplicate

Содержание=>Скрипт 3d=>Функции 3d=>faceListDuplicate

Формат вызова

FaceList faceListDuplicate( FaceList src, Matrix transfer )

Описание

Эта функция делает копию исходного списка поверхностей, преобразовывая его с помощью матрицы.

Параметры

  • src - исходный список поверхностей
  • transfer - матрица преобразования

Пример применения

color1 = selectColor("#a00000")

c1 = faceListCylinder( 2, 5 )
c2 = faceListDuplicate( c1, matrixTranslate( matrix1, 5, 0, 0 ) )

partModel = model( color1, color1, color1, c1, matrix1 )
partModel = model( color1, color1, color1, c2, matrix1 )
