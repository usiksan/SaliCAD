faceListWallRound

Content=>Script 3d=>Functions 3d=>faceListWallRound

Prototype

FaceList faceListWallRound( Face src, Face dst, Float scaleX, Float scaleY, Float radius, Float stepDegree )

Description

This function builds walls between the src and dst surfaces forming a rounding from below. The number of points in the source surfaces must be the same. The dst surface is usually obtained by scaling the src surface with the same coefficients as specified in the call to this function. The radius parameter defines the radius of rounding, and stepDegree the number of degrees per step of approximation of the curve of the surface.

Parameters

  • src - source surface (floor)
  • dst - target surface (ceiling)
  • scaleX - scaling of the target surface relative to the source surface along the X axis
  • scaleY - scaling of the target surface relative to the source surface along the Y axis
  • radius - radius of rounding
  • stepDegree - the number of degrees per step of approximation of the surface curve

Example

color1 = selectColor("#a00000")

c1 = faceCircleSide( 3, 6, matrix1 )
c2 = faceDuplicate( c1, matrixTranslate( matrixScale( matrix1, 1.3, 1.3, 1.0), 0,0,2) )

h1 = faceListWallRound( c1, c2, 1.3, 1.3, 1, -10 )

partModel = model( color1, color1, color1, h1, matrix1 )
