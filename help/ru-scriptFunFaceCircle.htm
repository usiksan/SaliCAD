faceCircle

Содержание=>Скрипт 3d=>Функции 3d=>faceCircle

Формат вызова

Face faceCircle( Float radius, Float stepDegree, Matrix transfer )

Описание

Эта функция строит поверхность в виде равностороннего многоугольника вписанного в окружность. Количество сторон многоугольника определяется шагом угла. Полученный многоугольник можно преобразовать с помощью матрицы. Данная функция служит для построения аппроксимированных окружностей. Чем меньше шаг угла - тем выше качество аппроксимации. При больших значениях шага угла с помощью этой функции можно строить другие правильные многоугольники: равносторонний треугольник, квадрат и т.п.

Параметры

  • radius - радиус описанной окружности вокруг многоугольника
  • stepDegree - шаг угла в градусах
  • transfer - заключительное преобразование многоугольника

Пример применения

#Пример окружности
color1 = selectColor("#800000")

c1 = faceCircle( 2, 10, matrix1 )

partModel = model( color1, color1, color1, [c1], matrix1 )

Данный пример сформирует следующее изображение:

#Пример шестиугольника
color1 = selectColor("#800000")

c1 = faceCircle( 2, 60, matrix1 )

partModel = model( color1, color1, color1, [c1], matrix1 )

Этот пример показывает формирование фигур при больших значениях шага угла
