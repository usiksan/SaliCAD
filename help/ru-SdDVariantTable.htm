Диалог Таблица вариантов

Очень часто компоненты имеют одинаковый корпус и одинаковое схемное обозначение, но несколько различных параметров. Например, резисторы различаются номиналом и допуском. Чтобы уменьшить количество информации вводимой для таких компонентов используется таблица вариантов. Она содержит перечень различающихся параметров для таких компонентов и значения этих параметров. При выборе компонентов при вставке в схему компонент имеющий таблицу может показываться как единственный компонент или как множество компонентов различающихся значениями из данной таблицы.

Например, для компонента Резистор в таблице вариантов могут быть представлены поля value (значение) и tolerance (отклонение). Остальные параметры могут быть name (наименование). Предположим, что поле name имеет значение "smd-0201", а таблица вариантов имеет следующий вид:

value tolerance
1kOm 5%
2kOm 10%

Тогда для выбора будут доступны два компонента типа Резистор. Первый компонент

name - "smd-0201", value - "1kOm", tolerance - "5%"

и второй компонент

name - "smd-0201", value - "2kOm", tolerance - "10%"

Поле name не присутствует в таблице вариантов, поэтому для всех компонентов будет одинаковым. Поля, которые присутствуют в таблице вариантов будут формироваться из содержимого строк этой таблицы.

Данный диалог предназначен для создания и редактирования таких таблиц вариантов.

В левой части диалогового окна расположена таблица зарегистрированных полей. В правой части - расположена сама таблица вариантов.

Таблица полей

В данной таблице перечислены все доступные зарегистрированные поля для данного компонента. Изменить этот список можно с помощью редактора компонентов. Любое из этих полей можно добавить к таблице вариантов. Если поле добавлено к таблице вариантов, то значение этого поля может отличаться для различных вариантов данного компонента. Если же поле не добавлено к таблице вариантов, то для всех вариантов данного компонента будет использоваться значение заданное для данного поля по умолчанию. Визуальная таблица состоит из трех колонок:

  • наличие варианта
  • имя параметра
  • значение параметра

Колонка "наличие варианта" служит для добавления и исключения полей из таблицы вариантов. Чтобы добавить поле к таблице вариантов нужно поставить галочку для данного поля, а чтобы убрать поле из таблицы вариантов - убрать галочку.

Таблица вариантов

Данная таблица имеет переменное число колонок в зависимости от количества полей изменяемых в зависимости от варианта. Для добавления и удаления колонок служит таблица полей слева. Колонка "наличие варианта" отвечает за присутствие поля в таблице вариантов. Также можно добавить поле, которое отсутствует в перечне полей по умолчанию. Для этого служит кнопка Append field. Для удаления полей служит кнопка Delete field.

Чтобы добавить ряд к таблице нажмите кнопку Insert row. При этом новый ряд вставляется прямо под текущим выбранным рядом и все значения в новый ряд копируются из текущего. Остается их только отредактировать. Данный подход упрощает работу, поскольку очень часто изменяются не все поля, а только их часть.

Чтобы удалить ряд нажмите кнопку Delete row. При этом удаляется текущий выбранный ряд.

Чтобы удалить все ряды таблицы нажмите кнопку Delete all.
