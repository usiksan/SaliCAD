vertex

Содержание=>Скрипт 3d=>Функции 3d=>vertex

Формат вызова

Vertex vertex( Float x, Float y, Float z )

Описание

Эта функция строит точку в 3d пространстве по ее координатам.

Параметры

  • x - координата по оси X
  • y - координата по оси Y
  • z - координата по оси Z

Пример применения

vertex1 = vertex( 0.2, 0.1, 0.4 )
