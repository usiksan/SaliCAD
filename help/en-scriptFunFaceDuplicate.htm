faceDuplicate

Содержание=>Скрипт 3d=>Функции 3d=>faceDuplicate

Prototype

Face faceDuplicate( Face src, Matrix mat)

Description

This function performs the transformation of the original surface using a matrix into another surface.

Parameters

  • src - initial surface
  • mat - transformation matrix

Example

color1 = selectColor("#800000")
color2 = selectColor("#008000")

c1 = faceCircle( 2, 10, matrix1 )

mat = matrixTranslate( matrix1, 1, 0, 1 )

c2 = faceDuplicate( c1, mat )

partModel = model( color1, color1, color1, [c1], matrix1 )
partModel = model( color2, color2, color2, [c2], matrix1 )
