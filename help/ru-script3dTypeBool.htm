Тип логический

Содержание=>Скрипт 3d=>Тип логический

Тип логический позволяет принимать логические значения истина и ложно. Этот тип служит для представления результатов логических операций. Он также используется в качестве условий в условном операторе и операторе цикла.

Два ключевых слова true и false используются для представления логических значений истина и ложь соответственно.
