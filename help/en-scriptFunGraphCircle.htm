graphCircle

Content=>Script 3d=>Functions 3d=>graphCircle

Prototype

Graph2d graphCircle( FloatList [centerX, centerY, radius] )

Description

The function builds a 2d object in 3d space. The functions of this class are designed to create complex component generation wizards, where a 3d model and a 2d image of a component are generated in one script. To build ordinary 3d models, using these functions does not make sense.

All 2d objects are constructed in the XY plane with a slight offset along the axis.

This function builds a circle based on its parameters

Parameters

  • centerX - the center of the circle X
  • centerY - the center of the circle Y
  • radius - circle radius

Example

partFlat = graphCircle( [0,0,2] ))
