Диалог Мастер площадок

Данное диалоговое окно содержит мастер создания 3d моделей и 2d проекций корпусов. Использование данного мастера позволяет автоматически получить полностью готовый и оформленный корпус компонента, а также его 3d модель.

В левой части окна - список параметров конкретной модели компонента, а справа - области предварительного просмотра 3d модели компонента, включая контактные площадки, а также его 2d представление.

При изменении любого параметра внешний вид модели автоматически перестраивается, показывая ее актуальный вид.

Точка обзора 3d компонента в области предварительного просмотра изменяется с помощью мыши:

  • движение мыши по горизонтали при нажатой левой кнопке изменяет угол отображения компонента относительно оси Z, также этот угол изменяется при вращении колеса мыши
  • движение мыши по вертикали при нажатой левой кнопке изменяет угол наклона плоскости проекции к плоскости экран, этот угол изменяется также при вращении колеса мыши при нажатой клавише Ctrl
  • движение мыши при нажатой средней кнопке изменяет размещение точки начала координат относительно экрана
  • вращение колесамыши при нажатой клавише Shift масштабирует изображение корпуса компонента

Параметры в таблице параметров могут быть обычными числами, при этом они изменяются вручную; цветами, при этом их изменение происходит с помощью диалога изменения цвета; описанием площадок, при этом их изменение происходит с помощью диалога редактирования площадок.
