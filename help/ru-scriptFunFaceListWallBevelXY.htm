faceListWallBevelXY

Содержание=>Скрипт 3d=>Функции 3d=>faceListWallBevelXY

Формат вызова

FaceList faceListWallBevelXY( Face src, Face dst, Float radius, Float angleStep )

Описание

Эта функция строит стены фаски между поверхностями src и dst. Количество точек в исходных поверхностях должно быть одинаковым. Поверхность dst рекомендуется получать с помощью функции faceBevelXY. Различные комбинации знака в параметрах faceBevelXY и faceListWallBevelXY позволяют получить различные формы фаски.

Параметры

  • src - исходная поверхность
  • dst - целевая поверхность, полученная с помощью функции faceBevelXY
  • radius - радиус фаски
  • angleStep - размер шага угла для аппроксимации кривой поверхности фаски

Пример применения

color1 = selectColor("#a00000")

c1 = faceCircleSide( 3, 56, matrix1 )
c2 = faceBevelXY( c1, 1 )

h1 = faceListWallBevelXY( c1, c2, 1, 2 )

partModel = model( color1, color1, color1, h1, matrix1 )
