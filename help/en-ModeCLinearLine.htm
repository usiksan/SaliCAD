Line enter mode

Contents=>Modes=>Draw modes=>Линия

Tool bar:

Menu: Draw|Line

Cursor:

1. Setup line attributes

2. Enter start point of segment

3. (Optional) Change segment attributes

4. Enter end point of the segment. After you specify the end point of the segment, it is entered into the project. Automatically, the end point of the segment serves as the starting point of the next segment, thus allowing the formation of a polyline.

5. Repeat steps 2-4 to enter the entire polyline.

  • You can specify a point by clicking the left mouse button or manually.
  • The termination of the input is done by pressing the right mouse button.

"Smart" mode.

The relative input of the previous segment is performed. That is, you have just entered a segment and want the new segment to be parallel to the entered segment and have the same length. To do this, you specify the beginning of a new segment, and smart mode does the rest of the work.

Here A is the initial segment, B is the segment formed by the smart mode.


This "smart" mode is especially useful for sequential input of identical segments, for example, symbol pins.

If you move the mouse after specifying the first point, the smart mode looks for the end point of the segment in the direction of the mouse movement at the intersection with the already entered lines or rectangles.

Middle mouse button or "space" - confirmation of "smart" input.
