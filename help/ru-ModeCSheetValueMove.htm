Перенос значения

Содержание=>Режимы=>Специальные режимы схемы=>Перенос значения

Панель инструментов:

Меню: Лист схемы|Перенос значения

Курсор:

Служит для переноса значения компонента в схеме. Каждое значение имеет положение по умолчанию. Оно задается относительно компонента, поэтому когда компонент переносится или поворачивается значение  переносится и поворачивается вместе с ним. Чтобы изменить положение значения относительного компонента служит данный режим.

1. Укажите компонент, значение которого нужно перенести

2. Установите атрибуты значения

3. Укажите местоположение значения

4. Повторяйте шаги 1-3 пока не будут расставлены все значения

  • Указание точки осуществляется нажатием левой кнопки мыши или вручную.
  • Прекращение ввода осуществляется нажатием правой кнопки мыши.

Атрибуты значения задаются с помощью панели свойств текста.

"Разумный режим" отсутствует.
