Тип матрица

Содержание=>Скрипт 3d=>Тип матрица

Тип матрица представляет собой матрицу 4х4, которая используется для преобразований в трехмерном пространстве. Для создания матриц используются функции.

Для матриц не допускаются никакие операции, кроме присваивания. Также матрицы находят применение в вызовах функций.
