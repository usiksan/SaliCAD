Диалог Поиска и выбора объекта

Это диалоговое окно служит для поиска объекта в библиотеке и его ввода.

Раздел Категории содержит иерархический список категорий объектов. По умолчанию этот список поставляется вместе с системой SaliCAD и представлен на английском языке.

При выборе какой-либо категории в списке выбор компонентов фильтруется по этой категории.

Раздел Фильтр по полям содержит перечень всех полей текущих найденных объектов. Указывая значения в полях этого фильтра список найденных объектов можно сузить.

Поле Имя содержит набор слов для поиска объекта по имени. Слова в этом поле разделяются пробелами. Можно использовать для поиска части слов. В список найденных объектов попадают только те, в имени которых присутствуют ВСЕ слова из данного поля.

Флаг Раскрыть таблицу параметров. Если этот флаг не установлен, то в результат поиска попадают только сами компоненты. Если же этот флаг установлен, то для компонентов имеющих таблицу параметров будут сформированы компоненты для каждой строки из этой таблицы. Например, для резисторов если флаг не установлен, то в результате поиска будет только один резистор, а если установлен, то в результате поиска будет множество резисторов со всеми номиналами из таблицы параметров.

Раздел Результа поиска содержит список всех найденных компонентов, удовлетворяющих условиям поиска.

Раздел Секции содержит таблицу секций текущего выбранного компонента из таблицы Результаты поиска. Если ищется не компонент, то данный раздел будет пустым.

Раздел Символ содержит предварительный просмотр выбранной секции текущего компонента - для компонентов. Изображение схемы - для фрагмента. Изображение символа - для символа. Для корпуса данный раздел будет пустым.

Раздел Корпус содержит предварительный просмотр корпуса текущего компонента - для компонентов. Изображение корпуса - для корпуса. Для остальных объектов данный раздел будет пустым.

При нажатии кнопки ОК выбирается текущая секция текущего компонента - для компонента или сам объект - для остальных объектов, а сам диалог выбора закрывается.

При нажатии кнопки Отмена диалог выбора закрывается и объект считается невыбранным.

Поиск объектов ведется среди всех доступных объектов включая удаленный репозиторий. Если объект был найден в удаленном репозитории и еще не был загружен в локальную библиотеку, то его изображение будет не доступно. Чтобы загрузить объект из удаленной библиотеки следует нажать кнопку "Загрузить из репозитория".
