Тип текстовая строка

Содержание=>Скрипт 3d=>Тип текстовая строка

Тип текстовая строка содержит текстовую строку в виде последовательного набора знаков. В скриптах строки задаются ковычками:

"это строка"

Сами ковычки не входят в строку, а служат для определения ее границ.

Для строк не допустимы никакие операции, кроме присвания. Также строки находят применение в вызовах функций.
