Диалог Слои


В этом диалоговом окне можно посмотреть список доступных слоев, состояние слоев, а также их парность (см.слои).

В центральной части диалога находится таблица со списком доступных слоев. По колонкам распределены имя слоя, назначение для трассировки, состояние, видимость, цвет и парность.

Нажатие на ячейку Трасса позволяет циклически перебирать возможные назначения слоя для трассировки.

Нажатие на ячейку Состояние позволяет циклически перебирать возможные состояния слоя.

Нажатие на ячейку Видимость позволяет включать-выключать видимость слоя. Слой видимый в состоянии Включен и Редактируемый. Слой невидимый в состоянии Выключен. Видимость - это графическое представление состояния слоя.

При нажатии на ячейку Цвет появляется диалог выбора цвета для слоя. Вы можете назначать для слоев любые цвета, в том числе и одинаковые. Цвета слоев сохраняются в Вашем рабочем профиле на данном компьютере. На другом компьютере может быть собственная палитра цветов.

При нажатии на ячейку Парный слой появляется диалог выбора парного слоя для данного.

В поле Количество трассировочных слоев задается количество трассировочных слоев для конкретной платы. Поэтому данное поле активно только когда диалог слоев вызывается из графического редактора плат.

При нажатии кнопки Закрыть диалоговое окно закрывается.

При нажатии кнопки Создать... появляется диалог создания нового слоя.

При нажатии кнопки Удалить текущий выбранный слой удаляется. Удаление возможно только если на данном слое нету никакой информации ни в одном из открытых проектов.

Кнопки ограничения выводимого списка слоев.

Количество слоев может быть довольно значительным, но в каждый конкретный момент может использоваться только ограниченное множество этих слоев. Данные кнопки позволяют сократить количество выводимой информации. На добавление-удаление слоев они не влияют.

  • Актуальные. Показываются слои актуальные для графического редактора. Т.е. это список слоев "по умолчанию".
  • Используемые. Показываются слои, на которых присутствует какая-нибудь информация.
  • Все. Показываются все известные на данный момент слои.

При нажатии кнопки Скрыть все состояние всех слоев меняется на Выключено.

При нажатии кнопки Переключить все состояние всех слоев переключается на следующее, как если бы Вы нажимали на поле Состояние для всех слоев.
