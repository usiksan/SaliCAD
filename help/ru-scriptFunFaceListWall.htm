faceListWall

Содержание=>Скрипт 3d=>Функции 3d=>faceListWall

Формат вызова

FaceList faceListWall( Face src, Face dst, Bool close )

Описание

Эта функция строит стены между поверхностями src и dst. Количество точек в исходных поверхностях должно быть одинаковым. Фактически, поверхность dst обычно получается путем некоторого преобразования поверхности src. Параметр close определяет, нужно ли строить стену между конечной и начальной точками.

Параметры

  • src - исходная поверхность (пол)
  • dst - целевая поверхность (потолок)
  • close - когда true строится стена между последней и первой точками, иначе - эта стена не строится и фигура получается разомкнутой

Пример применения

color1 = selectColor("#a00000")

c1 = faceCircleSide( 3, 6, matrix1 )
c2 = faceDuplicateShift( c1, 4 )

h1 = faceListWall( c1, c2, true )

partModel = model( color1, color1, color1, h1, matrix1 )
