faceCurveXZ

Содержание=>Скрипт 3d=>Функции 3d=>faceCurveXZ

Формат вызова

Face faceCurveXZ( Face src, Float radius, Float angleSrc, Float angleDst)

Описание

Эта функция выполняет поворот в плоскости XZ исходной поверхности. Для исходной поверхности задается исходный угол. Для результирующей поверхности задается радиус поворота и результирующий угол. Данная функция используется, в основном, для построения изогнутых выводов.

Параметры
  • src - исходная поверхность
  • radius - радиус поворота от центра масс исходной поверхности
  • angleSrc - угол исходной поверхности
  • angleDst - угол результирующей поверхности

Пример применения

#Пример для поворота
color1 = selectColor("#800000")
color2 = selectColor("#008000")

c1 = faceCircle( 2, 10, matrix1 )

c2 = faceCurveXZ( c1, 4, 0, 35 )

partModel = model( color1, color1, color1, [c1], matrix1 )
partModel = model( color2, color2, color2, [c2], matrix1 )
