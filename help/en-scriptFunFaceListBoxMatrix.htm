faceListBoxMatrix

Content=>Script 3d=>Functions 3d=>faceListBoxMatrix

Prototype

FaceList faceListBoxMatrix( Float lenght, Float width, Float heigth, Matrix translate )

Description

This function builds a parallelepiped with a base on the XY plane, and then transforms it using a matrix.

Parameters

  • length - parallelepiped length (X-axis)
  • width - width of the parallelepiped (Y axis)
  • height - height of the parallelepiped (Z axis)
  • translate - transformation matrix

Example

color1 = selectColor("#a00000")

c1 = faceListBoxMatrix( 2, 3, -4, matrixRotate( matrix1, 45, 0, 1, 0 ) )

partModel = model( color1, color1, color1, c1, matrix1 )
