Вставка компонента

Содержание=>Режимы=>Специальные режимы схемы=>Вставка компонента

Панель инструментов:

Меню: Лист схемы|Компонент

Курсор:

Служит для вставки компонентов в лист схемы. Компоненты в листе схемы представляются вхождением символа и являются одной секцией компоненты.

1. В диалоге "Ввод компонентов" (см.Диалоговое окно "Ввод компонентов") выберите компонент и секцию внутри компонента.

2. Задайте атрибуты для компонента.

3. Укажите точку вставки компонента. При этом секция компонента вставляется в лист схемы.

4. Повторяйте шаги 2 и 3 для ввода одинаковых секций компонентов.

5. Для прекращения ввода данной секции компонента и возврата к выбору секции компонента нажмите правую кнопку мыши.

6. Повторяйте шаги 1-5 для ввода различных секций компонентов в лист схемы.

Атрибуты компонента задаются с помощью панели свойств компонента:

  • Указание точки осуществляется нажатием левой кнопки мыши или вручную.
  • Прекращение ввода осуществляется нажатием правой кнопки мыши.

"Разумный" режим отсутствует.
