faceListIndexed

Содержание=>Скрипт 3d=>Функции 3d=>faceListIndexed

Формат вызова

FaceList faceListIndexed( FaceList src, FloatList indexes )

Описание

Эта функция извлекает из списка поверхности, согласно индексам и из них формирует новый список.

Параметры

  • src - исходная поверхность
  • indexes - индексы поверхностей в исходном списке (начиная с 0)

Пример применения

color1 = selectColor("#a00000")
color2 = selectColor("#00a000")

c1 = faceCircleSide( 3, 6, matrix1 )
c2 = faceDuplicateShift( c1, 4 )
h1 = faceListWall( c1, c2, true )

b1 = faceListIndexed( h1, [0, 2, 4] )
b2 = faceListIndexed( h1, [1, 3, 5] )
partModel = model( color1, color1, color1, b1, matrix1 )
partModel = model( color2, color2, color2, b2, matrix1 )
