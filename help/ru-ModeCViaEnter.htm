Режим вставки переходных отверстий

Содержание=>Режимы=>Плата=>Вставка переходных отверстий

Панель инструментов:

Меню: Плата|Вставить переходное

Курсор:

Данный режим служит для расстановки дополнительных переходных отверстий.

1. Установите атрибуты переходного отверстия. Выберите исходный слой для переходного отверстия.

2. Укажите точку размещения переходного отверстия. При нажатии левой кнопки мыши переходное отверстие вставляется в плату, если правила трассировки позволяют разместить переходное отверстие в данной точке.

3. Повторяйте шаги 1-2 для ввода всех необходимых переходных отверстий.

Свойства переходных отверстий задаются с помощью панели свойств переходных отверстий.

  • Указание точки осуществляется нажатием левой кнопки мыши или вручную.
  • Прекращение ввода осуществляется нажатием правой кнопки мыши.

"Разумный" режим отсутствует.
