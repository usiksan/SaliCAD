Ножка символа

Содержание=>Руководства=> Режимы=>Режимы ввода объектов символа =>Ножка символа

Панель инструментов:

Меню: Объекты|Ножка символа

Курсор:

Ножка символа задает точку, к которой в схематическом изображении элемента подключаются цепи.

1. Установите атрибуты ножки

2. Укажите местоположение ножки

3. Установите атрибуты имени ножки

4. Укажите точку расположения имени ножки, при вводе имени ножки курсор имеет вид

5. Введите имя ножки, по завершении ввода нажмите Enter или левую кнопку мыши

6. Установите атрибуты номера ножки

7. Укажите точку расположения номера ножки, при вводе номера ножки курсор имеет вид

Для ввода атрибутов ножки символа служит панель ножек символа:

"Разумный" режим.

При вводе точки местоположения ножки, эта точка предлагается "разумным" режимом на конце ближайшего отрезка.

При вводе точек расположения имени и номера ножки, эти точки предлагаются относительно точки расположения вывода одинаково с предыдущим выводом. Этот режим особенно удобен для последовательного ввода выводов, у которых относительное положение имени и номера одинаково.

При вводе имени ножки предлагается имя предыдущей ножки с индексом большим на 1, то есть если предыдущее имя ножки было, например, А1, то предлагаемое имя будет А2. Этот режим удобен для последовательного ввода ножек различных шин.

Для ввода атрибутов имени ножки и номера ножки служит панель свойств текста.

  • Указание точки осуществляется нажатием левой кнопки мыши или вручную.
  • Прекращение ввода осуществляется нажатием правой кнопки мыши.
