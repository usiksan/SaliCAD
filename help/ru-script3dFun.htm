3d функции

Содержание=>Скрипт 3d=>3d функции

Здесь представлен полный перечень функций, которые можно использовать в скриптах построения 3d моделей.

Для взаимодействия с пользователем (получения от него необходимой информации для построения конкретной модели компонента) используются функции ввода. Эти функции добавляют один параметр в таблицу параметров скрипта с его значением по умолчанию. При изменении пользователем любого параметра весь скрипт пересчитывается и строится новая модель на основе этого введенного пользователем значения. К группе ввода относятся следующие функции:

  • inputFloat ввод чисел
  • inputBool ввод состояний вкл/откл
  • inputColor ввод цвета
  • inputPad ввод описания контактной площадки

Функции работы с цветами

  • colorBuild построить цвет по rgba составляющим
  • colorFromString построить цвет из текстового описания цвета

Функции работы с поверхностями

  • faceBuild построить поверхность по региону и цвету
  • faceTranslate создать копию поверхности с заданным смещением
  • faceTriangle создать поверхность-треугольник по трем точкам и цвету

Функции работы с 2d объектами

  • graphCircle создать окружность
  • graphLine создать отрезок
  • graphPin создать вывод
  • graphRect создать прямоугольник
  • graphRectFilled создать закрашенный прямоугольник

Функции работы с моделями

  • modelAddExtrude добавляет вытяжку модели с верхней стороны исходной модели в
    направление вектора нормали с величиной сдвига
  • modelBodyBeveled создает корпус микросхемы со срезами
  • modelBox создает параллелепипед
  • modelCopy создает копию модели с перемещением и поворотом
  • modelCurve создает модель изогнутого тела на основе контура и угла поворота
  • modelCylinder создает модель цилиндра
  • modelExtrude вытягивает модель и региона
  • modelHexagon создает модель хексагона
  • modelHole создает модель отверстия в параллелепипеде или цилиндре
  • modelPinTqfp создает модель вывода корпуса tqfp
  • modelTranslate создает копию модели путем перемещения
  • modelWall создает модель стен на основе региона
  • modelWallEven создает модель отдельных стен на базе региона, представляющего пары вершин отрезков основания стен

Функции для работы с регионами

  • regionCircle создает регион описывающий круг
  • regionRect создает регион описывающий прямоугольник
  • regionShift создает копию региона сдвинутого в перпендикулярном направлении плоскости региона
  • regionTranslate создает копию региона с перемещением

Функции создания описания контактных площадок

  • stringPadCircleThrou сквозная круглая площадка
  • stringPadRectThrou сквозная прямоугольная площадка

Функции формирования номера вывода

  • stringPinIndex номер вывода по индексу
  • stringPinMatrix номер матричного вывода по двум индексам

Функции работы с вершинами

  • vertexBuild строит вершину по трем координатам
  • vertexCenterOfRegion - находит вершину центра региона
  • vertexOffset - создает копию вершины смещенную на заданные координаты
  • vertexTranslate - создает копию вершины смещенную с помощью вектора
