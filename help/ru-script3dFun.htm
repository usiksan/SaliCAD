3d функции

Содержание=>Скрипт 3d=>3d функции

Здесь представлен полный перечень функций, которые можно использовать в скриптах построения 3d моделей.

Непосредственные значения задаются прямо в тексте скрипта. При адаптации скрипта под свою модель разработчик может изменять значения, например, размеры корпуса компонента, количество его ножек и т.п. Числовые значения вводятся несредственно. Однако, некоторые значения достаточно сложны для ручного ввода. Для упрощения их построения используются мастера, которые вызываются при нажатии кнопки F2 на клавиатуре когда курсор находится на одной функций выбора. Эти функции перечислены ниже:

  • selectColor диалоговый выбор цвета
  • selectPad мастер создания контактных площадок

Функции работы с цветами

  • color построить цвет по rgba составляющим
  • colorFromString построить цвет из текстового описания цвета

Некоторые компоненты корпусов должны быть представлены в виде строк, например, описание контактных площадок или номер вывода компонента. Для построения таких строк используются следующие функции:

  • stringPadRectThrough построить строку описания прямоугольной контактной площадки для штыревого вывода
  • stringPadCircleThrough построить строку описания круглой контактной площадки для штыревого вывода
  • stringPinIndex строит номер вывода по его индексу
  • stringPinMatrix строит номер вывода матричного корпуса, например, bga, по двум индексам

Для выполнения геометрических преобразований используются матрицы. В тех местах, где нужна матрица, а само преобразование не нужно можно использовать встроенную переменную единичной матрицы matrix1. Для упрощения выполнения типовых преобразований служат следующие функции:

  • matrix строит матрицу поворота и перемещения
  • matrixRotate строит новую матрицу путем добавления с существующей матрице операции поворота
  • matrixScale строит новую матрицу путем добавления с существующей матрице операции масштабирования
  • matrixTranslate строит новую матрицу путем добавления с существующей матрице операции смещения

Вершины описывают точки в 3d пространстве. Каждая вершина содержит тройку чисел, представляющих собой координаты вершины в пространстве. Для работы с вершинами служат следущие функции:

  • vertex - служит для задания вершины по трем координатам
  • vertexOffset - создает вершину путем задания смещения относительно другой вершины

Поверхность представляет собой плоскую область, ограниченную ломаной линией, образующей многоугольник. SaliCAD работает только с выпуклыми многоугольниками. Простейшим выпуклым многоугольником является треугольник. Многоугольники с количеством вершин более трех могут быть невыпуклыми. Для решения этой проблемы имеется функция разбиения такого многоугольника на треугольники.

  • faceBevelXY -генерит поверхность масштабированную на радиус в плоскости xy и сдвинутую на радиус по оси z. Используется для формирования фаски
  • faceCircle - генерит многоугольник с заданным шагом, вписанный в окружность. Служит для аппроксимации окружностей
  • faceCircleSide - генерит многоугольник с заданным количеством граней, вписанный в окружность. Служит для аппроксимации окружностей
  • faceCurveXZ - генерит поверхность повернутую относительно исходной на некоторый угол. Используется для формирования гнутых объектов, например выводов
  • faceDuplicate - создает копию исходной поверхности преобразованной с помощью матрицы
  • faceDuplicateGrow - генерит поверхность масштабированную в плоскости xy на заданные коэффициенты kx и ky. Результирующая поверхность преобразуется с помощью матрицы
  • faceDuplicateScale - генерит поверхность масштабированную в плоскости xy на заданные коэффициенты kx и ky. Результирующая поверхность сдвигается в направлении оси z
  • faceDuplicateShift - генерит копию поверхности сдвинутой в направлении перпендикуляра к поверхности
  • faceEllipse - генерит многоугольник с заданным шагом, вписанный в элипс. Служит для аппроксимации элипсов
  • faceEllipseSide - генерит многоугольник с заданным количеством граней, вписанный в элипс. Служит для аппроксимации элипс
  • faceEquidistantXY - генерит многоугльник в плоскости xy с гранями эквидистантными к граням исходного многоугольника. Результат преобразуется с помощью матрицы
  • faceFlat - создает многоугольник в заданной плоскости по парам относительных координат
  • faceFlatMatrix - создает многоугольник в плоскости xy по парам относительных координат, а затем преобразуется с помощью матрицы
  • facePart - создает многоугольник из заданного многоугольника по списку вершин
  • faceRect - создает прямоугольник
  • faceRectRound - создает прямоугольник с закругленными углами
  • faceRectSide - создает многоугольник с заданным количеством граней вписанный в прямоугольник

Несколько поверхностей составляют модель. Следующие функции служат для работы с набором поверхностей.

  • faceListAppend - создает набор, объединенный из двух наборов
  • faceListBodyBeveled
  • faceListBox - создает параллелепипед
  • faceListBoxMatrix - создает параллелепипед и преобразует его с помощью матрицы
  • faceListCurveXZ
  • faceListCylinder - создает цилиндр с основание в плоскости xy
  • faceListCylinderMatrix - создает цилиндр и преобразует его с помощью матрицы
  • faceListDuplicate - создает копию набора поверхностей преобразованную с помощью матрицы
  • faceListExtrude - из поверхности основания создает вытяжку с помощью матрицы
  • faceListExtrudeShift - из поверхности основания создает вытяжку в направлении перпендикуляра к основанию
  • faceListHexagon - создает хексагональный корпус
  • faceListHolesXY - преобразует поверхность в плоскости xy в набор поверхностей при добавлении окон (отверстий)
  • faceListIndexed - создает набор поверхностей по исходному набору и индексам
  • faceListPinCurveOne - создает вывод на базе поверхности основания с одним изгибом, например, выводы горизонтальных штыревых резисторов
  • faceListPinCurveTwo - создает вывод на базе поверхности основания с двумя изгибами, например, вывод микросхемы tqfp
  • faceListRotation - создает набор поверхностей поворота. Набор строится на основе исходной поверхности путем последовательного поворота на заданный угол с заданным шагом
  • faceListSimplify - создает набор поверхностей в плоскости xy из невыпуклой поверхности путем разбиения ее на треугольники
  • faceListUnion - создает набор, объединенный из двух наборов
  • faceListWall - создает стены между двумя поверхностями (пол и потолок)
  • faceListWallBevelXY - создает стены между двумя поверхностями в плоскости xy с формированием фаски на полу
  • faceListWallDoubleBevelXY- создает стены между двумя поверхностями в плоскости xy с формированием двух фасок: на полу и на потолке
  • faceListWallIndexed - создает стены между двумя поверхностями (пол и потолок). Создаются только стены, соответствующие индексу
  • faceListWallList - создает стены между наборами полов и потолков
  • faceListWallRound - создает закругленные стены между двумя поверхностями (пол и потолок)
  • faceListWalls - создает стены между "этажами", которые представлены списком поверхностей
  • faceListPinTqfp - создает вывод tqfp

Набор поверхностей вместе с цветами и матрицами копирования составляют модель. Для работы с моделями служат следующие функции:

  • model - создает модель на базе списка поверхностей, цветов этих поверхностей и матрицы копирования
  • modelAppend - добавляет к модели новый список поверхностей с цветами этих поверхностей
  • modelCopy - добавляет матрицу копирования к модели

Функции работы с 2d объектами

  • graphCircle создать окружность
  • graphLine создать отрезок
  • graphPin создать вывод
  • graphRect создать прямоугольник
  • graphRectFilled создать закрашенный прямоугольник
