PasCAD Соглашения по параметрам

Содержание=>Руководства=> Соглашения=>Соглашения по параметрам

Параметры в системе имеют текстовый вид и предназначены для задания параметров, которые могут быть восприняты другими программами с целью проведения расчетов, уточнения схем, плат, конструктива и т.п. Для этих программ система не делает никаких ограничений. Количество и формат параметров могут быть произвольными. В настоящем приложении дан формат лишь тех параметров, которые используются самой системой для внутренних целей.

Параметры компонента

Система воспринимает четыре класса параметров компонентов:

  • параметр, описывающий цепи питания
  • параметр, задающий цепь по умолчанию
  • параметр, описывающий вид записи элемента в перечне элементов
  • параметры, используемые для построения структурных схем

Параметр ЦЕПИ

Параметр ЦЕПИ предназначен для процедуры автоматического создания текстового списка цепей. Текстовый список цепей предназначен для указания тех, соединений, которые невозможно отобразить на принципиальной схеме в графическом виде. В большинстве случаев это цепи питания, которые необходимо подключить к определенным выводам микросхем.

Общий вид параметра ЦЕПИимеет следующий вид:

ЦЕПИ=<Список цепей>

Где Список цепей имеет вид:

<Имя цепи>(<Номер вывода>[;< Номер вывода>...])[;...]

Например:

ЦЕПИ=GND(7);+5V(14)

цепь с именем GND подключится к выводу с номером 7, цепь с именем +5V подключится к выводу с номером 14.

ЦЕПИ=GND(1;20);+5V(40)

цепь с именем GND подключится к выводам с номерами 1 и 20, цепь с именем +5V подключится к выводу с номером 40.

ЦЕПИ=+15V(А32)

цепь +15V подключится к выводу с номером А32.

Как видно из общего вида и примеров, цепи в списке цепей разделяются точкой с запятой, список номеров ножек, к которым необходимо подключить данную цепь находится в скобках сразу за именем цепи. Список номеров ножек разделяется точкой с запятой.

При использовании этого параметра накладываются дополнительные ограничения на указываемые в данном параметре цепи и номера ножек. Все они не должны содержать символов «(», «)» и «;». На остальные цепи и ножки проекта (т.е. на те, которые не указаны в данном параметре) приведенные ограничения не распространяются.

Параметр Цепь

Параметр Цепь предназначен для автоматического назначения имени цепи при подключении к выводу. Эта возможность предусмотрена для специальных компонентов, которые обозначают элемент Земля, цепь +5В и т.п. Автоматическое назначение осуществляется только для первого вывода компонента (того, который был введен первым). Поэтому наилучшим вариантом будет, чтобы у компонента был единственный вывод.

Общий вид параметра Цепь имеет следующий вид:

Цепь=<Имя цепи по умолчанию>

где <Имя цепи по умолчанию> это имя, которое назначается цепи при подключении к первому выводу данного компонента.

Например:

Цепь=GND

В данном примере, при подключении цепи к первому выводу этого компонента цепи будет присвоено имя GND.

Параметр Перечень

Параметр Перечень предназначен для автоматического составления перечня элементов по принципиальной схеме. Этот параметр задает текстовую строку помещаемую в перечень элементов для данного элемента, а также для отображения в окне справки если курсор мыши находится над компонентом без движения более чем 3 секунды.

Общий вид параметра перечень имеет следующий вид:

Перечень=<Строка>

Где Строка это текстовая строка, построенная по определенным правилам, которая и помещается в перечень элементов.

Ниже приведен пример для транзистора КТ3102А

Перечень=КТ3102А аАО.336.122 ТУ

Строка может состоять из любой последовательности символов, однако, если в последовательности символов встречается символ «<», то находящиеся за ним символы трактуются как имя параметра, значение которого необходимо поместить вместо имени этого параметра, т.е. например рассмотрите запись строки Перечень для резистора

Значение=1кОм

Отклонение=+-10%

Мощность=0.125

Перечень=С2-33Н-<Мощность>-<Значение> < Отклонение>-А-В-В ОЖО.467.173ТУ

набор этих параметров при формировании перечня дает следующий результат:

С2-33Н-0.125-1кОм ±10%-А-В-В ОЖО.467.173ТУ

Из приведенного примера видно, что фраза <Мощность> была заменена на значение параметра Мощность, <Значение> -- на значение параметра Значение, а <Отклонение> -- на значение параметра Отклонение.

Такая возможность упрощает построение смежных программ выбора компонентов (Выбор резисторов, Выбор конденсаторов), а также других сервисных программ (например программа расстановки значений на принципиальной схеме).

Параметр Загрузить

Этот параметр используется в символах для автоматической загрузки проектов при двойном нажатии на символ левой кнопкой мыши. Параметр используется при создании структурных схем. Параметр имеет следующий формат:

Загрузить=<путь и имя проекта>

после знака равенства должен следовать полный путь к проекту, который требуется загрузить. Например:

Загрузить=Проект1\плата процессора.pis

при использовании указанной последовательности загружается проект плата процессора.pis из подкаталога Проект1.

Параметр Открыть

Этот параметр испльзуется в символах и открывает для редактирования заданный объект при двойном нажатии левой кнопкой мыши на данный символ. Объект ищется во всех проектах. Для редактирования открывается первый попавшийся (если имеется несколько объектов в разных проектах с одинаковыми именами). Параметр используется при создании структурных схем для ускорения доступа к объектам, изображенным на структурной схеме. Параметр имеет следующий формат:

Открыть=<имя объекта

после знака равенства следует полное имя объекта. Следует отметить, что система различает большие и маленькие буквы, поэтому имя объекта должно точно сопадать с именем того объекта, который Вы хотите открыть для редактирования.

Например:

Открыть=ЦПУ лист 1

открывает для редактирования объект с именем ЦПУ лист 1

Параметр Запустить

Этот параметр используется в символах и осуществляет запуск другой программы при двойном нажатии левой кнопки мыши на этот символ. Значение этого параметра по сути дела представляет собой командную строку, которая без каких либо изменений передается в операционную систему для исполнения. Параметр имеет следующий формат:

Запустить=<командная строка >

Например:

Запустить=c:\MSOffice\Winword\Winword.exe c:\Мои документы\тексты\Описание проекта 1.doc

Выполняет запуск программы Word с автоматической загрузкой документа Описание проекта 1.
