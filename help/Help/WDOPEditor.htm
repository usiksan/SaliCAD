Графические редакторы

Содержание=>Руководства=> Диалоги=>Параметры среды=>Графические редакторы

Это диалоговое окно служит для задания поведения графических редакторов.

Включить отображение неразведенных связей в плате. Этот флажок включает/отключает отображение сети в плате. Имеется аналогичная команда меню.
Включить отображение подсказок над компонентами Если этот пункт помечен, то при нахождении курсора мыши над компонентом неподвижно некоторое время, появляется всплывающая подсказка с названием этого компонента. Аналогично подсказка действует и над цепью..
Включить показ поз.обозначений в строке состояния Если этот пункт помечен, то при нахождении курсора мыши над компонентом или цепью название этого компонента или цепи отображаются в строке состояния.
Размер перекрестья для ножки символа Ножки в символе, а также неподключенные ножки компонентов в схеме отображаются крестиком. Этот параметр задает размер этого крестика.
Размер окружности для ножки корпуса Ножка корпуса, а также ножки без контактных площадок компонентов в плате отображаются кружком. Этот параметр задает размер этого кружка.
Размер точки соединения цепей Вместе со следующим параметром задает вид точки соединения цепей. Чтобы получить сплошную точку нужно толщину линии задавать одинаковой с размером.
Толщина линии точки соединения цепей
Размер окружности для переходных отверстий Переходные отверстия в плате без подключенных контактных площадок отображаются кружком. Этот параметр задает размер этого кружка.
Шаг изменения толщины линий Толщина линий может задаваться явно, а может изменяться при нажатии кнопочек в панели свойств. Этот параметр задает шаг изменения толщины линий при нажатии на такие кнопочки.
Шаг изменения высоты текста Высота текста может задаваться явно, а может изменяться при нажатии кнопочек в панели свойств. Этот параметр задает шаг изменения высоты текста при нажатии на такие кнопочки.
