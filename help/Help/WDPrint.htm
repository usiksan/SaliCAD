Печать

Содержание=>Руководства=> Диалоги=>Печать

Это диалоговое окно служит для определения параметров печати. Здесь можно выбрать:

  • В графе "Принтер для печати" показывается принтер, на который будет производиться печать. Чтобы сменить принтер следует нажать кнопку Свойства. Кроме того нажав на эту кнопку можно выбрать формат бумаги, а также установить другие параметры конкретного принтера, которые предусмотрены соответствующим драйвером.
  • Способ печати задает тип масштабирования. Растянуть изображение на весь лист приводит к автоматическому вычислению масштаба таким образом, чтобы изображение заполнило весь лист. При этом происходит сохранение пропорций. Вывести с точным масштабом позволяет задать точный масштаб результирующего изображения. Если масштаб позволяет разместить изображение на одном листе, то печатается один лист, в противном случае изображение разбивается на несколько листов таким образом, чтобы обеспечить требуемый масштаб на выбранном формате бумаги.
  • Количество копий задает количество печатаемых копий.
  • Печатать. Весь рисунок - это полное изображение объекта независимо от вида и масштаба окна. Выделенный фрагмент допустим только если есть выделение. Окно позволяет конкретно указать или выбрать область для печати.
  • Толщина линий нулевой толщины позволяет задать толщину для печати для линий, которые нарисованы с толщиной 0.
  • Толщина линий текста. То же самое, что и для линий, но это действует только для векторного шрифта.
