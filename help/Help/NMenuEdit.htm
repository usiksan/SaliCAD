Редактирование

Содержание=>Руководства=> Меню=>Редактирование

Позволяет производить общие операции редактирования над выделенными объектами. Смысловой оттенок операции несколько меняется в зависимости от типа объектов, к которым они применяются. Группа меню редактирование состоит из следующих команд:

  • Копировать
  • Вырезать
  • Удалить
  • Дублировать
  • Маскирование объектов
  • Вставить
  • Переименовать
  • Поворот
  • Выделить все
  • Убрать выделение
  • Перенос по поз.обозначению
  • Перенос идентификатора
  • Отмена
  • Переименовать
