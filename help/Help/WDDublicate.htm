Дублировать

Содержание=>Руководства=> Диалоги=>Дублировать

Этот диалог позволяет задать параметры дублирования выделенных элементов. Выделенные элементы размножаются в виде прямоугольного массива Рядов и Колонок. Расстояние между рядами равно "Расстояние по Y", а расстояние между колонками равно "Расстояние по X".

  • Рядов: задает количество рядов в массиве
  • Колонок: задает количество колонок в массиве
  • Расстояние по X: задает расстояние в мм между колонками
  • Расстояние по Y: задает расстояние в мм между рядами
