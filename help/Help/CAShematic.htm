2.1 Как подключить цепи к выводам питания микросхем

  1. Проведите ренумерацию
  2. В схеме (обыно, по ГОСТу это первый лист схемы) вставьте объект список цепей: меню Объект|Список цепей. Далее укажите место для списка (обычно, по ГОСТу, список должен быть над штампом) появляется пустой список. Нажмите АвтоЦепь, затем нажмите Автотекст. (Кстати имена цепей в таблице можно менять, если Вы хотите чтобы выводы питания были подключены к другим цепям, нежели те которые назначаются автоматически). Для завершения нажмите Применить.

2.2 При нескольких перетаскиваниях компонента с подключенной цепью цепь сильно изламывается.

Компонент с подключенной цепью двигается великолепно (естественно немного не так, как в P-CAD). Чтобы сегмент цепи не ломался, его нужно захватывать за середину или целиком. (Пользуйтесь клавишей Shift для выделения нескольких объектов)
