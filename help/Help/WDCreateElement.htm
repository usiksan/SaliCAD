Создать элемент

Содержание=>Руководства=> Диалоги=>Создать элемент

Это диалоговое окно служит для создания объекта проекта.

Имя В этой графе Вы указываете имя вновь создаваемого объекта.
Класс В предлагаемом списке следует выбрать класс создаваемого объекта.
Создать на базе При нажатии на эту кнопку осуществляется создание нового элемента с использованием прототипа. Прототип выбирается из списка имеющихся элементов данного класса. Следует отметить, что создание схем и плат с использованием прототипов не возможно.
С помощью мастера При нажатии на эту кнопку осуществляется создание элемента с использованием соответствующего мастера. Мастер вызывается только если он устанавливался. Поэтому если при нажатии на эту кнопку ничего не происходит, то скорее всего соответствующий мастер не установлен.

Для текстовых документов дополнительно выводится диалог, в котором задается тип текстового документа.
