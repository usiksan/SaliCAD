Рисование текста 1 из 2

Укажите точку привязки (расположения) текста. Если точка привязки текста попадет на уже существующий текст, то включится режим редактирования этого текста.

Координаты точки можно ввести вручную, как показано здесь.

Чтобы отменить режим, нажмите правую кнопку мыши.

Свойства вводимого текста (высоту, слой, шрифт, наклон и т.д.) можно изменить как показано здесь.

Для редактируемого текста свойства можно менять только в процессе редактирования этого текста.
