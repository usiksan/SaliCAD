Введите шаблон для цепей шин. Шаблон представляет собой список цепей, разделенных запятыми. Кроме того группа цепей может обозначаться в форме:

ИмяЦепи<начало:конец>, например:

+5V,GND,D<0:7>,+15V

или выберите шаблон из списка. В списке представлены введенные ранее шаблоны.

Чтобы отменить режим, нажмите кнопку Отмена.
