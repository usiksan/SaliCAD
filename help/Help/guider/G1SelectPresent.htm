Выделено.

Вы выделили один или несколько объектов. Такие объекты рисуются черным цветом. С этими объектами можно проводить различные операции:

  • чтобы удалить объекты нажмите клавишу Delete;
  • чтобы быстро скопировать объекты, наведите на выделенные объекты мышь, нажмите клавишу Ctrl, нажмите левую кнопку мыши и, не отпуская ее, перемещайте. При этом образуется копия выделенных объектов. Отпускайте левую кнопку мыши в том месте, куда необходимо вставить копию;
  • чтобы скопировать объекты в карман, выберите пункт меню Редактировать|Копировать;
  • чтобы вырезать объекты в карман, выберите пункт меню Редактировать|Вырезать;
  • чтобы перенести объекты, наведите на выделенные объекты мышь, нажмите левую кнопку мыши и, не отпуская ее, перемещайте. При этом выделенные объекты перемещаются. Отпускайте левую кнопку мыши в том месте, куда необходимо переместить объекты;
  • чтобы изменить свойства объектов, воспользуйтесь панелью свойств. Измените свойства и нажмите клавишу Enter, чтобы изменения вступили в силу.

Чтобы убрать выделение нажмите левую кнопку мыши вне любого объекта.

Нажатие правой кнопки мыши покажет меню с командами, которые можно выполнять над этими объектами.
