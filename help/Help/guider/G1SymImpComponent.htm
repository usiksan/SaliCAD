Вставка символа 2 из 2

Укажите место для расположения символа. Вы можете вставить сразу несколько символов. Символ вставляется при каждом нажатии левой кнопки мыши. Чтобы перейти к другому символу нажмите правую кнопку мыши.

Чтобы отменить режим, нажмите правую кнопку мыши, а затем в диалоговом окне кнопку Отмена.

С помощью панели свойств компонент можно вращать и делать зеркальным.
