Положение подписи. Сейчас следует указать точку, в которой будет распологаться подпись для диаметра. Для этого нажмите левую кнопку мыши.

Координаты точки можно ввести вручную, как показано здесь.

Чтобы отменить режим, нажмите правую кнопку мыши.

Свойства подписи (высоту, слой, шрифт, наклон и т.д.) можно изменить как показано здесь.
