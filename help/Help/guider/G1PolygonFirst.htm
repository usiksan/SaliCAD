Ввод полигона 1 из 3

Полигон представляет собой залитую область на печатной плате. Ввод полигона заключается в задании границ полигона с помощью замкнутой ломаной линии. Сейчас следует указать начало ломаной.

Координаты точки можно ввести вручную, как показано здесь.

Чтобы отменить режим, нажмите правую кнопку мыши.

Свойства линии (толщину, слой, тип) можно изменить как показано здесь.
