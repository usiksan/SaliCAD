Изменение формы объектов

Можно изменить форму следующих объектов:

  • Отрезка, для этого следует тащить отрезок за один из кончиков.

  • Прямоугольник, для этого следует тащить прямоугольник за один из углов.

  • Сегмента цепи, для этого следует тащить сегмент за кончик или тащить связанный сегмент или компонент.

  • Региона или полигона, для этого следует тащить регион или полигон за одну из вершин.

Нажатие правой кнопки мыши вернет предыдущий режим.
