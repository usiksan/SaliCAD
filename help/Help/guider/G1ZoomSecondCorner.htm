Масштабирование окном 2 из 2

Этот режим позволяет растянуть заданную прямоугольную область на все окно. Для этого нужно указать эту область. Область задается через два противоположных угла. Сейчас нужно указать второй угол области.

Координаты точки можно ввести вручную, как показано здесь.

Чтобы отменить режим, нажмите правую кнопку мыши.
