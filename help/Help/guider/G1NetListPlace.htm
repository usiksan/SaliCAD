Список цепей. Список цепей это специальный текстовый список цепей, которые по каким либо причинам не отображены графически. Чаще всего это цепи питания цифровых микросхем. Сейчас следует указать точку расположения этого списка. Обычно текстовый список цепей распологается над штампом на первом листе принципиальной схемы. ВНИМАНИЕ! Для правильной работы текстового списка необходимо, чтобы компоненты были пронумерованы!!! Поэтому, если компоненты не пронумерованы, пронумеруйте их (меню объекты|ренумерация).

Координаты точки можно ввести вручную, как показано здесь.

Чтобы отменить режим, нажмите правую кнопку мыши.

Свойства текстового списка (высоту, слой, шрифт, наклон и т.д.) можно изменить как показано здесь.
