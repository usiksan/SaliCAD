Таблица контактных площадок

Содержание=>Руководства=> Диалоги=>Таблица контактных площадок

Это диалоговое окно служит для организации таблицы соответствия контактных площадок типам выводов.

Если отмечена фраза «Разрешить использование контактных площадок», то контактные площадки подключаются к выводам и отображаются в графическом редакторе печатной платы.

В таблице представлено соответствие контактных площадок типам выводов. Разрешается редактировать любую колонку таблицы. Отметим, что в качестве типов выводов могут выступать только числа. В качестве контактных площадок могут выступать любые объекты проекта типа Корпус.

При нажатии на кнопку «Накопить используемые» текущая таблица очищается и происходит накопление типов контактных площадок используемых в данной печатной плате.

При нажатии на кнопку «Взять из другого проекта» появляется диалоговое окно открытия файла, а Вам предлагается указать имя проекта, из которого следует взять образы контактных площадок. После выбора проекта, Вам предлагается список печатных плат этого проекта, чтобы Вы могли выбрать плату, из которой следует взять таблицу соответствия контактных площадок типам выводов. После выбора платы осуществляется копирование записей из таблицы соответствия платы загруженного проекта в текущий проект. При этом копируются только те типы выводов, которые присутствуют в текущей таблице. В текущий проект копируются также отсутствующие в текущем проекте образы контактных площадок.

При нажатии на кнопку «Выбрать из списка» Вам предлагается список корпусов из всех загруженных проектов, из которого Вы должны выбрать корпус, который Вы хотите использовать в качестве образа контактной площадки.

При нажатии на кнопку Вставить осуществляется вставка новой строки в таблицу соответствий.

При нажатии на кнопку Удалить осуществляется удаление текущей строки из таблицы соответствий.
