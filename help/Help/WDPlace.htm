Следующий компонент

Содержание=>Руководства=> Диалоги=>Следующий компонент

Это диалоговое окно служит для выбора следующего компонента или для задания режима его выбора. С помощью этого окна задается перечень элементов группы и шаблон группы для групповой расстановки по образцу.

Три зависимые кнопки в левом верхнем углу задают способ выбора следующего компонента при проведении ручной расстановки. Имеются три варианта:

  • "Разумный" режим. Здесь компонент выбирается по специальному алгоритму.
  • Следующий по номеру. Здесь, например, если был компонент С10, то следующим будет расставляться С11.
  • Через диалог. В этом случае, после расстановки каждого компонента, для выбора следующего появляется это диалоговое окно.

Идентификатор (позиционное обозначение) компонента можно явно ввести в поле Компонент или просто выбрать из списка под ним. В любом случае, нажатие на кнопку Размещать приводит к началу процесса размещения выбранного компонента, а кнопки Закрыть - к закрыванию диалогового окна без каких либо действий.

Кнопка "Выбрать из схемы" совместно со списком позволяет выбирать компонент прямо из схемы. Т.е. для выбора компонента Вы просто указываете его прямо в схеме. Для активизации этого режима нужно выбрать (пометить) схему, из которой будет производиться выбор и нажать на эту кнопку.

В строке Исходный система отображает список компонентов, которые можно считать шаблоном. В строке Похожий, система предлагает список компонентов, которые можно расставить похожим образом. Вы можете ввести свои списки. Списки состоят из идентификаторов (позиционных обозначений) компонентов, разделенных запятыми.

При нажатии кнопки Расставить, похожие фрагменты размещаются по аналогии с Исходным.
