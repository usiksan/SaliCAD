PasCAD Соглашения по контактным площадкам

Содержание=>Руководства=> Соглашения=>Соглашения по контактным площадкам

В комплект поставки системы входит специальный файл базы данных «Контактные площадки.pis», в котором сгруппированы контактные площадки для односторонних и двухсторонних плат. Поставляемые в комплекте системы библиотеки разработаны с учетом следующих соглашений в перспективе применения контактных площадок из файла базы данных «Контактные площадки.pis».

Указанные соглашения ни в коем случае не являются обязательными. Они приводится лишь для того, чтобы Вы при составлении своих соглашений могли ориентироваться по типам выводов поставляемых библиотек.
