Удаление слоя

Содержание=>Руководства=> Диалоги=>Удаление слоя

Это диалоговое окно служит для удаления слоев.

  • Для удаления одного слоя выберите в списке слой для удаления и нажмите кнопку Удалить. Система проверяет наличие информации на этом слое и, если информации нет, то происходит удаление слоя.
  • При нажатии на кнопку "Удалить неиспользуемые" система проверяет каждый слой на предмет наличия информации и, в случае отсутствия, производит удаление соответствующего слоя. В конце сообщается о количестве удаленных слоев.

Для завершения работы диалога нажмите кнопку "Закрыть".
