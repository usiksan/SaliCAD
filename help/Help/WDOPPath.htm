Это диалоговое окно служит для указания каталогов расположения различных составных частей системы.

Рабочий каталог К этому каталогу система автоматически обращается при загрузке или сохранении файлов.
Каталог для библиотек Файлы из этого каталога выводятся в список при вставке компонентов из библиотек.
Каталог для шаблонов К этому каталогу система обращается при формировании текстовых документов.
