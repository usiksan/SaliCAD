Режим цепей

Содержание=>Руководства=> Диалоги=>Режим цепей

Это диалоговое окно служит для задания режима объединения цепей при копировании, вставке из кармана, вставки фрагмента в лист схемы.

Предположим, что в листе схемы есть цепи XN1, +5В и ЗЕМЛЯ. Пусть в вставляемом фрагменте тоже присутствуют цепи с такими именами. Тогда возможны следующие варианты:

  • В случае когда всем цепям оставляются старые имена, все цепи вставляемого фрагмента объединяются по именам с цепями листа схемы.
  • Когда новые имена присваиваются только цепям по умолчанию, цепи +5В и ЗЕМЛЯ вставляемого фрагмента и листа схемы объединяются, а цепи XN1 фрагмента назначается имя цепи отличное от всех других цепей текущего проекта.
  • Когда всем цепям присваиваются новые имена, то всем цепям вставляемого фрагмента назначаются имена цепей отличные от всех других цепей текущего проекта.

При каждой операции копирования, вставке из кармана, вставки фрагмента в лист схемы будет выдаваться этот запрос если не отмечена фраза «Не выводить больше этот запрос».

Нужный режим объединения задается пометкой соответствующего пункта.
