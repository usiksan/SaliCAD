Компонент

Содержание=>Руководства=> Диалоги=>Компонент

Это диалоговое окно служит для создания и редактирования компонентов.

В строке редактирования «Схемное обозначение» следует указать имя схемного обозначения компонента. Это схемное обозначение можно выбрать из списка, для этого следует нажать кнопку "..."

В строке редактирования «Изображение корпуса » следует указать имя изображения корпуса компонента. Это изображение корпуса можно выбрать из списка, для этого следует нажать кнопку "..."

В области редактирования в центре диалогового окна следует вводить параметры компонента.
