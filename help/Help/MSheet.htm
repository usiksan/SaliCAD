Режимы ввода объектов схемы

Содержание=>Руководства=> Режимы=>Режимы ввода объектов схемы

В системе имеются следующие режимы ввода объектов схемы:

  • Режим ввода компонентов
  • Режим ввода цепей
  • Режим отключения цепей
  • Режим ввода имен цепей
  • Режим ввода шин
  • Режим ввода списка цепей
  • Режим задания области платы
  • Режим ручного назначения позиционного обозначения
  • Режим автоматического назначения позиционных обозначений
  • Режим объединения секций в один корпус (или последовательную цепочку корпусов)
