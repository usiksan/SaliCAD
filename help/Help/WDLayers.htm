Слои

Содержание=>Руководства=> Диалоги=>Слои

В этом диалоговом окне осуществляется ввод, редактирование и удаление слоев системы. Отметим, что слои являются глобальными для системы. То есть для всех загруженных проектов используется одна и та же таблица слоев.

При нажатии мышью на наименование слоя включается диалоговое окно «Слой».

При нажатии мышью на состояние слоя это состояние изменяется на следующее. Всего в системе имеется четыре состояния слоев, переключение которых осуществляется циклически. (см. «Слои»).

При нажатии на цвет слоя включается диалог редактирования цветов.

При нажатии кнопки Добавить включается диалоговое окно «Слой», в котором Вам предлагается задать имя нового слоя. Состояние слоя устанавливается РЕД, а цвет устанавливается черным. И состояние и цвет слоя можно тут же изменить.

При нажатии кнопки Удалить выводится диалог удаления слоев.

При нажатии на кнопку Выключить все, состояние всех слоев становится ОТКЛ, т.е. отключенное.

При нажатии на кнопку Переключить все, состояние всех слоев переключается на следующиее.

При нажатии кнопки Парность выводится диалог определения парности слоев.
