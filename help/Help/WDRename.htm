Переименование

Содержание=>Руководства=> Диалоги=>Переименование

Это диалоговое окно служит для задания другого имени объекта проекта.

В строке редактирования отображается старое имя объекта. Введите новое имя объекта в этой строке. Помните, что вновь вводимое имя должно быть уникально во всем проекте. При попытке задать объекту уже существующее имя, выдается соответствующее сообщение. Для того чтобы изменение вступило в силу нужно нажать кнопку ОК. Чтобы отменить переименование следует нажать кнопку Отмена.
