Выбор объектов (действий)

Содержание=>Руководства=> Диалоги=>Выбор объектов (действий)

В этом диалоговом окне осуществляется выбор объектов для вставки в графические редакторы, а также выбор инструментов, с помощью которых нужно обработать данный объект. В качестве примера выбора объектов для вставки служит приведенный выше рисунок. Навигация осуществляется с помощью мыши. При наведении мыши на какой либо пункт и ее неподвижности, включается отображение свойств выбранного таким образом объекта. При нажатии левой кнопки мыши объект незамедлительно выбирается для вставки.

В общем, диалог позволяет следующее:

  • выбрать компонент или фрагмент для вставки в схему или в плату
  • выбрать инструмент для проведения действия с объектом

При выборе компонентов или фрагментов становится доступным предварительный просмотр этих объектов. Причем просмотр доступен, в зависимости от объекта, в нескольких видах. Для компонентов это параметры, схема и корпус.

На приведенном выше примере показана компонент КР1533ИР10 и вид его символа.

Выбор компонента

Для выбора компонентов или фрагментов диалог предлагает иерархическую навигационную систему подобно файловой структуре диска или подобно навигатору по сети Интернет. В этом смысле, навигатор имеет начальный уровень (кнопка В начало), на котором стандартно расположены четыре опции.

Библиотеки Выбор осуществляется из файлов библиотек. При нажатии на эту кнопку выдается список файлов, расположенных в каталоге библиотек. Этот каталог задается в соответствующем поле настроек. Поэтому, если при нажатии на этот пункт ничего не происходит, следует проверить корректность установки пути к библиотекам.
Открытые проекты Выдает список открытых в настоящий момент проектов (в том числе и текущий проект), позволяя производить выбор из них.
По категориям Специальные программы-мастера, которыми комплектуется система позволяют осуществлять выбор компонентов по соответствующим категориям, с использованием различных таблиц параметров, таблиц быстрого выбора и т.п. Конкретный набор мастеров зависит от поставки.
Файл Позволяет указать конкретный файл на диске, из которого нужно осуществить вставку компонента или фрагмента.

Выбор инструмента

По аналогии с выбором компонентов, выбор инструментов имеет иерархическую структуру. Конкретный состав пунктов навигатора зависит от объекта и комплекта поставки программ-мастеров.
