Маскирование

Содержание=>Руководства=> Диалоги=>Маскирование

Это диалоговое окно служит для маскирования (запрета редактирования) отдельных групп объектов. Если соответствующая группа отмечена, то редактирование объектов этой группы разрешено. В противном случае, редактирование объектов этой группы запрещено, то есть хотя объекты этой группы могут оставаться видимыми, но редактирование объектов этой группы становится невозможным.

Не рекомендуется оставлять на длительное время объекты запрещенными для редактирования.
