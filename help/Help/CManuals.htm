Руководства

Содержание=>Руководства

Для Вас подготовлены следующие руководства:

Быстрый старт Руководство для быстрого освоения навыков работы с системой PasCAD, а также для начинающих пользователей.
Для начинающих Детальное упрощенное руководство для начинающих пользователей. Представлены только основные операции.
Объектная иерархия Представлено разъяснение внутренней структуры системы PasCAD и ее компонентов.
По диалоговым окнам Представлено разъяснение назначения и использования диалоговых окон.
По режимам Приведено полное описание режимов работы системы.
По командам меню Приведено подробное описание команд меню, их назначение и использование.
По операциям Представлено описание выполнения наиболее типовых операций с помощью системы PasCAD.
Соглашения Представлены различные стандартные предположения (соглашения), используемые системой PasCAD. Включают в себя список стандартных слоев, использование в параметрах стандартных ключевых слов и т.п.
Руководство пользователя в электронном виде Представлено полное руководство пользователя в формате PDF (Adobe Acrobat). Подготовлено для распечатки на принтере.
