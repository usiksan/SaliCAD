Ножка корпуса

Содержание=>Руководства=> Режимы=>Режимы ввода объектов корпуса =>Ножка корпуса

Панель инструментов:

Меню: Объекты|Ножка корпуса

Курсор:

Ножка символа задает точку, к которой в плате подключаются цепи.

1. Установите атрибуты ножки

2. Укажите местоположение ножки

3. Установите атрибуты номера ножки

4. Укажите точку расположения номера ножки, при вводе номера ножки курсор имеет вид

5. Введите номер ножки, по завершении ввода нажмите Enter или левую кнопку мыши

6. Установите атрибуты имени ножки

7. Укажите точку расположения имени ножки, при вводе имени ножки курсор имеет вид

Для ввода атрибутов ножки служит панель свойств ножки корпуса:

"Разумный" режим.

При вводе точки местоположения ножки, эта точка предлагается "разумным" режимом на конце ближайшего отрезка.

При вводе точек расположения номера и имени ножки, эти точки предлагаются относительно точки расположения вывода одинаково с предыдущим выводом. Этот режим особенно удобен для последовательного ввода ножек, у которых относительное положение номера и имени одинаково.

При вводе номера ножки предлагается номер предыдущей ножки с индексом большим на 1, то есть если предыдущий номер ножки был, например, А1, то предлагаемый номер будет А2. Этот режим удобен для последовательного ввода ножек корпусов, поскольку корпуса, как правило, имеют последовательную нумерацию ножек.

Для ввода атрибутов номера и имени ножки служит панель свойств текста.

  • Указание точки осуществляется нажатием левой кнопки мыши или вручную.
  • Прекращение ввода осуществляется нажатием правой кнопки мыши.
