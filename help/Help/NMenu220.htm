Замена объекта

Содержание=>Руководства=> Меню=>В проекте=>Замена объекта

Панель инструментов: нет

С помощью этого пункта можно заменить объект в данном проекте на объект того же типа из другого проекта. Таким образом можно поменять вид элемента в схеме или плате просто заменив его в проекте.

При выборе этого пункта Вам предлагается выбрать объект для замены.
