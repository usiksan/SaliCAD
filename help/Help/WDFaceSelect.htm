Выбор шрифта

Содержание=>Руководства=> Диалоги=>Выбор шрифта

Это диалоговое окно служит для выбора шрифта для сопоставления. Из списка выберите определенный шрифт и нажмите кнопку ОК. Нажав на кнопку "Шрифт по умолчанию", можно сопоставить векторный шрифт, который будет одинаково отображаться на всех машинах, поскольку состоит из отрезков прямых линий. Кроме того, этот шрифт можно изменить воспользовавшись входящим в комплект компилятором.

Внимание! Некоторые виды шрифтов могут иногда некорректно отображаться системой Windows. В этом случае Вам нужно будет выбрать другой шрифт.
