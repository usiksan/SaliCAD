Ренумерация

Содержание=>Руководства=> Меню=>Объекты=>Ренумерация

Панель инструментов: нет

Выполняет ренумерацию компонентов в схеме. Расставляет позиционные обозначения в соответствии с действующими нормативами ГОСТ. Нумерация ведется по платам, а внутри плат по листам. Изменить порядок листов можно с помощью команды меню Нумеровать листы схем
