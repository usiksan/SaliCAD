Сетка

Содержание=>Руководства=> Диалоги=>Сетка

В этом диалоговом окне задаются параметры и характеристики сетки графического редактора.

Группа параметров Шаг сетки

По оси X задает размер ячейки сетки по оси X
По оси Y задает размер ячейки сетки по оси Y

В центральном списке представлены несколько предыдущих размеров сетки. Вы можете быстро выбрать сетку из этого списка.

Отображать сетку - если этот пункт отмечен, то на в окне графического редактора отображается сетка в виде точек цветом Сетки в узлах точек пересечения линий сетки.

Курсор по сетке - если этот пункт отмечен, то перемещения курсора мыши выравнивается по узлам сетки несмотря на то, что курсор продолжает двигаться плавно.
