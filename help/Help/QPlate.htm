Подготовка печатных плат для автоматической расстановки и трассировки в системе Specctra

Подготовка печатных плат для автоматической (ручной) расстановки элементов и автоматической (ручной) трассировки печатных соединений заключается в следующем:

  • Задание габаритов печатной платы
  • Задание границ трассировки
  • Подключение контактных площадок

Рассмотрим эти этапы подробнее.

Задание габаритов печатной платы

Габариты печатной платы это замкнутая область, которая ограничивает печатную плату. Эта область задается путем рисования прямоугольника  в слое "Контур платы". Прямоугольник, который изображает контур платы должен быть единственным в этом слое. 

Задание границ трассировки

Границы трассировки представляют собой замкнутую область внутри области печатной платы, которая определяет границы, в которых можно проводить печатные проводники.  Эта область задается путем рисования прямоугольника или замкнутой ломаной линии в слое "Границы трассировки". Если для рисования границ используется прямоугольник, то он должен быть единственным в этом слое. Если для рисования границ используется линии, то необходимо помнить, что эти линии не должны пересекаться, должны быть нарисованы за один прием (т.е. без отрыва; если Вы при рисовании допустили ошибку, придется стереть всю нарисованную часть и прорисовать все границы заново) и заканчиваться в той же точке, что и начинались. Другими словами границы трассировки все время должны быть замкнутыми, независимо от того, какой инструмент для этого используется. Если используется прямоугольник, то замкнутость обеспечивается автоматически, а для линий за этим нужно следить самостоятельно. Еще очень важный момент. Контур платы должен располагаться внутри контура платы или, на крайний случай, совпадать с ним.

Подключение контактных площадок

Подключение контактных площадок является очень важным моментом в технологии проектирования печатных плат. В системе PasCAD выводы компонентов не привязаны строго к определенным контактным площадкам, вместо этого они имеют индексы для входа в таблицу контактных площадок. Такой подход дает возможность использовать различные наборы контактных площадок для различных технологических процессов. Например, контактные площадки для вывода с типом 1 для однослойной печатной платы может иметь размер 2 на 2, а для двухслойной - 1.5 на 1.5. Вообще Вы можете создать столько таблиц соответствий контактных площадок типам выводов, сколько нужно. Скажем больше, каждая плата в системе PasCAD имеет свою собственную таблицу соответствий, которая действует только для этой платы.

Очевидно, что создавать такую таблицу каждый раз заново - неудобно. Для решения этой проблемы система PasCAD предлагает удобный инструмент копирования таблиц из одного проекта в другой, сохраняя возможность ручного редактирования этой таблицы соответствий. Так в составе стандартной поставки имеется файл "Набор контактных площадок.pis", в котором содержатся стандартные контактные площадки, а также две платы с таблицами соответствий соответственно для одностороннего и двухстороннего технологического цикла.

Рассмотрим использование всех этих возможностей PasCAD на конкретном примере.

  • Откройте Вашу плату дважды щелкнув на ее названии левой кнопкой мыши.
  • Выберите пункт меню Объекты|Площадки..., при этом появляется диалоговое окно редактирования таблицы соответствия контактных площадок типам выводов
  • В центре диалогового окна располагается таблица соответствия, в левой колонке которой отображаются типы выводов, представляющие собой простые числа, а в правой колонке показываются соответствующие этим типам контактные площадки. Для редактирования записей этой таблицы служат кнопки диалогового окна:
    • Для вставки новой строки в таблицу служит кнопка "Вставить". При нажатии этой кнопки внизу таблицы добавляется новая строка, значение типа которой устанавливается в "-1", а поле контактной площадки остается пустым. Для нормального функционирования необходимо обязательно отредактировать эти поля, чтобы они содержали реальные значения.
    • Для удаления строки служит кнопка "Удалить". При нажатии этой кнопки.
    • Для выбора контактной площадки из списка для данного типа можно нажать кнопку "Выбрать". При этом появляется список всех доступных корпусов из всех открытых проектов. Нужно выбрать из списка корпусов контактную площадку и нажать кнопку ОК.
    • Кнопка "Накопить используемые" служит для сканирования текущей платы и занесения в таблицу соответствий всех типов выводов, имеющихся в компонентах данной платы.
    • Кнопка "Взять из другого проекта" служит для задания соответствий типов выводов контактным площадкам аналогично какой-либо другой плате.
  •  Нажмите кнопку "Накопить используемые", при этом происходит автоматическое накопление типов выводов используемых в данной плате и размещение их в таблице соответствий.
  • Нажмите кнопку "Вставить". При этом в конец таблицы соответствий добавляется запись с типом вывода установленным в -1.
  • Нажмите мышью на ячейку с типом -1, при этом включается редактирование данной ячейки. Введите 0 для данного типа (это стандартный тип для переходного отверстия).
  • Нажмите мышью на ячейку с именем контактной площадки для типа 0.
  • Нажмите кнопку "Взять из другого проекта". При этом появляется диалоговое окно загрузки файла. Перемещаясь по дереву каталогов найдите файл "Набор контактных площадок.pis". Он расположен в каталоге "Библиотеки" корневого каталога PasCAD. Затем нажмите кнопку Открыть.
  • Появляется список печатных плат имеющихся в данном проекте. В данном случае имена плат определяют технологию, для которой заданы таблицы соответствий контактных площадок. В стандартном файле "Набор контактных площадок.pis" имеются платы "Односторонняя плата" и "Двусторонняя плата". Выберите нужный вариант и нажмите кнопку ОК.
  • При этом в текущую таблицу соответствий копируются соответствия, заданные в таблице соответствий выбранной Вами платы. Отметим, что копируются только те соответствия, которые имеются в обеих таблицах. Соответствия, которые не заданы, должны задаваться вручную.
  • Отметьте ячейку с именем контактной площадки, где оно не задано.
  • Нажмите кнопку "Выбрать" диалогового окна. При этом появляется список всех доступных корпусов из всех открытых проектов. Выберите корпус, который Вы хотите видеть в качестве контактной площадки для данного вывода и нажмите кнопку ОК.
  • Для закрытия диалогового окна работы с контактными площадками нажмите кнопку "Закрыть".

После закрытия диалогового окна работы с контактными площадками вместо стилизированных выводов корпусов высвечиваются контактные площадки, подключенные к данным выводам.

На этом подготовка информации для трассировки в системе Specctra считается законченной.

Передача информации в систему Specctra.

Интерфейс с системой автоматической трассировки печатных плат Specctra есть практически во всех крупных САПР печатных плат. Это объясняется высокой скоростью работы трассировщика Specctra, а также высоким качеством получаемых при этом плат.

В качестве входной информации, Specctra воспринимает специальный текстовый файл с собственным форматом данных. Этот формат документирован в фирменном руководстве, поставляемом вместе с системой. Обычно, этот файл имеет расширение DSN. Подобный файл можно создать вручную. Однако все САПР, использующие в своем цикле проектирования систему  Specctra имеют специальные утилиты, позволяющие перекодировать информацию о печатной плате в файл DSN для  Specctra и обратно. Подобную утилиту имеет и PasCAD.

Однако, в отличие от многих традиционных САПР, в которых утилиты оформлены в виде отдельных программ, в PasCAD утилиты называются мастерами и спроектированы таким образом, что они вызываются прямо из среды проектирования. Тем не менее это отдельный инструмент и он должен быть установлен в Вашей системе. Поэтому, если Вы не видите соответствующего значка в панели инструментов, значит соответствующий мастер не установлен. Программа установки официальных комплектов автоматически устанавливает этот мастер.

Для экспорта в систему  Specctra подходит только плата. Откройте для редактирования плату, которую Вы хотите трассировать. Перед запуском мастера следует убедиться, что эта печатная плата подготовлена для автоматической трассировки в системе  Specctra. Для запуска мастера нажмите закладку "Инструменты", при этом появляется список доступных инструментов. Среди этих инструментов следует выбрать такой значок:

 

При этом появляется диалоговое окно, в котором Вы должны указать место, куда записывать файл DSN. После того, как Вы нажмете кнопку ОК мастер экспорта начинает анализ Вашей платы. В случае обнаружения каких-либо проблем выдается соответствующее сообщение, если никаких недостатков в Вашей плате не обнаружено (т.е. она правильно подготовлена для трассировки в системе Specctra), то формируется файл DSN в с указанным Вами именем.

На этом этапе можно приступить к работе в системе Specctra. Для этого запустите ее главный исполняемый файл. При запуске, если система Specctra установлена правильно и у Вас есть соответствующая лицензия на ее использование, появляется диалоговое окно, в котором нужно указать файл проекта для загрузки. Отметим, что систему Specctra нельзя запустить без файла проекта. Имя файла проекта можно набрать с клавиатуры вручную или выбрать, для этого нужно нажать кнопку Browse... При этом появляется диалог поиска-выбора файла, в котором нужно выбрать файл проекта DSN, только что сформированный мастером экспорта системы PasCAD. Файл проекта - это единственный файл, который нужен для трассировки плат, подготовленных в PasCAD. Остальные входные файлы Вы можете указывать самостоятельно по Вашему желанию. Естественно, что эти файлы должны быть созданы заранее либо вручную, либо с помощью иных инструментов, например, самой системы Specctra.

После того, как в поле Design указано имя Вашего проекта, можно нажать кнопку Start SPECCTRA для загрузки проекта в систему Specctra. При загрузке файла проекта Specctra проводит контроль его синтаксиса, к сожалению, в большинстве случаев, если есть ошибка в синтаксисе или структуре файла проекта, то работа системы Specctra аварийно завершается без каких-либо сообщений. Это можно считать самым большим недостатком этой системы. К счастью, мастер экспорта, поставляемый в комплекте с PasCAD проверяет большинство типичных ошибок и формирует правильный файл проекта, который обязательно загружается в Specctra. Свидетельством правильной загрузки файла проекта служит появление основного окна Specctra с видом загруженного файла.

Теперь можно приступать к автоматической/ручной расстановке элементов и/или автоматической/ручной трассировке печатных соединений.

Ручная расстановка элементов в системе Specctra

Расстановка элементов состоит в размещении элементов на плоскости печатной платы с учетом электрических особенностей схемы и последующей трассировки печатных соединений. Эта книга не является руководством по конструированию радиоэлектронной аппаратуры, вместо этого она разъясняет способы проведения тех или иных операций с помощью описываемых инструментов.

Таким образом, ручная расстановка заключается в ручном размещении компонентов на плоскости печатной платы.

Для перехода в режим размещения нажмите на такую кнопку:

При этому становятся доступными следующие кнопки панели инструментов:

Указанные кнопки имеют следующий смысл:

Кнопка  
Общие кнопки обоих режимов
Перерисовать изображение. После выполнения серии графических операций, изображение на экране обычно загрязняется остатками изображений. Чтобы очистить изображение от мусора его обычно обновляют путем перерисовки.
Отобразить весь рисунок. Как и любая другая графическая система, Specctra позволяет в процессе работы различным образом изменять масштаб изображения. После изменения масштаба обычно бывает видна только часть полного рисунка, чтобы увидеть весь рисунок целиком и используется эта кнопка. При ее нажатии изображение масштабируется таким образом, что оно целиком помещается на экране.
Открыть список слоев. Как и в других графических системах САПР, в системе Specctra применяется система слоев. Одни слои создаются автоматически (слой изображения корпуса, слой позиционного обозначения, слой сетки и т.д.), другие определяются файлом проекта. К последним относятся слои трассировки. В окне слоев со слоями можно производить различные действия: включать/выключать, задавать цвет, определять активность (для слоев трассировки). Все вносимые изменения моментально вступают в силу. Окно списка слоев может постоянно оставаться на экране. Его не обязательно убирать для продолжения дальнейшей работы. Просто когда это окно больше не нужно или оно мешает работе его можно закрыть нажав кнопку Close.
Открывает окно Status Report. Как с ним работать х его знает. Когда разберетесь - пишите нам, мы дополним это руководство Вашими рекомендациями.
Режим измерения. Нажимаете левую кнопку мыши и не отпуская ее перемещаете мышь, при этом в строке состояния в графе Message отображаются расстояния по X и по Y от первой точки до текущей. Если просто нажать и отпустить кнопку мыши, то в окно сообщений выдается информация об объекте под курсором. Если под курсором нет никакого объекта, то фиксируется просто точка.
   
Включает режим выбора объектов. Этот режим позволяет пометить объект и снять с него пометку. На моей машине помеченные компоненты отображаются желтым цветом. Нажмите на объект левой кнопкой мыши. При первом нажатии объект помечается, а при повторном нажатии левой кнопки над этим объектом пометка снимается. Если нажать кнопку мыши и, удерживая ее, перемещать мышь, то выделение осуществляется окном. Алгоритм тот же что и для левой кнопки. При первом попадании объекта в охваченную область, он помечается, а при повторном - пометка снимается.
Блокировка компонентов. Этот режим позволяет заблокировать компоненты от перемещения. При первом нажатии компонент блокируется, а при повторном нажатии левой кнопки над этим компонентом блокировка снимается. Если нажать кнопку мыши и, удерживая ее, перемещать мышь, то блокировка осуществляется окном. Алгоритм тот же что и для левой кнопки. При первом попадании компонента в охваченную область, он блокируется, а при повторном - блокировка снимается.
 Кнопки режима размещения компонентов.
Режим переноса компонентов. При нажатии левой кнопки мыши над компонентом, этот компонент выделяется (если он не заблокирован) и начинает перемещаться синхронно с курсором мыши. Перемещая мышь укажите новое местоположение компонента (оно должно быть внутри контура трассировки и внутри контура размещения, а также компонент не должен "налезать" на другие компоненты и на области запрета расстановки), для фиксации компонента на новом месте нажмите левую кнопку мыши. Захватывать компоненты можно и окном, для этого нажмите левую кнопку мыши и, не отпуская ее, перемещайте мышь, при этом образуется прямоугольная область. При отпускании левой кнопки мыши выделяются все компоненты, попадающие в эту область, выделяются и начинают перемещаться синхронно с курсором мыши.
Режим сдвига компонента. При нажатии левой кнопки над компонентом, этот компонент выделяется и начинает работать режим сдвига компонента, т.е. этот компонент сдвигает или раздвигает другие компоненты таким образом, чтобы он разместился на нужном месте. При повторном нажатии левой кнопки мыши компонент фиксируется на новом месте. Для группы компонентов этот режим не работает. При попытке все-таки охватить группу компонентов, происходит выделение этой группы, но сдвиг осуществляется только того компонента, на который попал курсор при отпускании левой кнопки мыши.
Поворот компонента. При нажатии левой кнопки мыши над компонентом, этот компонент выделяется и включается режим его поворота. При перемещении мыши вокруг компонента осуществляется поворот компонента с дискретностью 90 градусов. При нажатии в этот момент правой кнопки мыши появляется меню, в котором можно изменить дискретность поворота компонента на одно из следующих 180, 90, 45 и 1. Поворачивать можно сразу группу компонентов. Для этого нажимаете левую кнопку мыши и, не отпуская ее, перемещаете мышь. При отпускании левой кнопки мыши все компоненты попадающие в окно охвата выделяются и для них включается режим поворота. Поворот каждого компонента осуществляется вокруг его центра.
Перенос на другую сторону платы. Компоненты можно располагать на двух сторонах печатной платы. Для переноса компонентов с одной стороны платы на другую и служит этот режим. При нажатии левой кнопки мыши над компонентом, этот компонент перемещается на противоположную сторону платы. Т.е. если компонент был на верхней стороне, то он перемещается на нижнюю и наоборот. Компоненты на разных сторонах отображаются разными цветами. Эти цвета задаются в окне слоев. Перемещать можно сразу группу компонентов. Для этого нажимаете левую кнопку мыши и, не отпуская ее, перемещаете мышь. При отпускании левой кнопки мыши все компоненты попадающие в окно охвата перемещаются на противоположную сторону. Отметим, что все компоненты группы перемещаются не на одну сторону, а на противоположную той, на которой они находились до переноса, т.е. если один компонент был на верхней стороне, а другой на нижней, то после переноса первый компонент окажется на нижней стороне, а второй на верхней.
Обмен компонентов. Этот режим позволяет обменять одинаковые компоненты. При нажатии левой кнопки мыши над компонентом, этот компонент выделяется, при повторном нажатии левой кнопки над другим таким же компонентом происходит обмен двух компонентов местами. Отметим, что обмен работает только для одинаковых компонентов.
  Выравнивание компонентов. Этот режим осуществляет выравнивание одного компонента относительно другого. При нажатии левой кнопки мыши над компонентом он выделяется и подготавливается для выравнивания. При нажатии левой кнопки мыши над другим компонентом, осуществляется выравнивание по одной из сторон (в зависимости от взаиморасположения компонентов), причем первый компонент выравнивается относительно второго. Можно осуществлять выравнивание сразу группы компонентов. Сначала нужно выделить группу компонентов, которую необходимо выровнять, а затем указать компонент, относительно которого осуществлять выравнивание.

Особенности ручного размещения можно свести к серии простых рекомендаций, которые, разумеется, можно игнорировать, но, в любом случае, полезно прочитать. Если Вы обладаете опытом работы с системой Specctra и у Вас есть собственные рекомендации, посылайте их нам и мы включим их в это руководство.

  • Перед началом размещения лучше всего задать сетку. Для этого на поле рисунка нажмите правую кнопку мыши, при этом появляется меню. В этом меню выберите пункт Setup... Появляется диалоговое окно, в котором можно установить общие параметры расстановки. В поле "PCB placement Grid" установить шаг сетки расстановки. В поле "PCB placement Spacing" можно установить зазор между компонентами по умолчанию.
  • Отметим, что при попытке разместить компонент с нарушением правил размещения (например, один компонент налезает на другой) или вне области платы компонент не размещается.

Самое интересное - автоматическое размещение остается за рамками этой книги, поскольку у автора нет опыта работы с этим расстановщиком. Если Вы обладаете нужной информацией, пишите!

Трассировка печатных соединений в системе Specctra

На мой взгляд система Specctra предоставляет самые удобные инструменты для трассировки печатных плат. Здесь, как говорится "в одном флаконе", и графический редактор с автоматическим контролем правил трассировки в процессе редактирования и автоматический трассировщик, который позволяет развести достаточно сложную плату без участия человека. Кроме того, замечательная возможность чередовать автоматическую и ручную трассировку позволяет развести практически любую плату.

Перед началом какой либо трассировки необходимо задать правила трассировки. В нашу задачу не входит описание всех возможных параметров, которые влияют на результирующую топологию, кратко остановимся лишь на некоторых наиболее употребимых.

Во-первых следует сказать об иерархии правил. В системе Specctra свои конкретные правила можно задать практически для каждого объекта платы. Какие конкретно будут применяться правила зависит от иерархии. Ниже приведена эта иерархия в порядке убывания приоритета:

Приоритет Уровень   Доступные правила
0 высокий region для региона зазоры, ширина дорожки
1 padstack для контактных площадок зазоры
2 class_class layer между классами на определенном слое зазоры, взаимное влияние, шум
3 class_class между классами зазоры, взаимное влияние, шум
4 fromto layer для связи на определенном слое зазоры, задержки прохождения сигнала, взаимное влияние, шум
5 fromto для связи зазоры, задержки прохождения сигнала, экранировка, взаимное влияние, шум
6 group layer для группы на определенном слое зазоры, задержки прохождения сигнала, взаимное влияние, шум
7 group для группы зазоры, ограничения для цепей, задержки прохождения сигнала, взаимное влияние, экранировка, шум
8 net layer для цепи на определенном слое зазоры, ограничения для цепей, задержки прохождения сигнала, взаимное влияние, шум
9 net для цепи зазоры, задержки прохождения сигнала, взаимное влияние, экранировка, шум
10 group_set layer для набора групп на определенном слое зазоры, ширина цепей
11 group_set для набора групп зазоры, ширина цепей, задержки прохождения сигнала
12 class layer для класса цепей на определенном слое зазоры, ограничения для цепей, задержки прохождения сигнала, взаимное влияние, шум
13 class для класса цепей зазоры, ограничения для цепей, задержки прохождения сигнала, экранировка, взаимное влияние, шум
14 layer для слоя зазоры, ограничения для цепей, фактор времени, взаимное влияние, шум, фактор шума для слоя, стоимость прокладки для слоя
15 низкий pcb для платы зазоры, ограничения для цепей, задержки прохождения сигнала, взаимное влияние, шум

Эту систему приоритетов нужно понимать следующим образом: при выборе правил и ограничений для каждого конкретного случая, который возникает в процессе трассировки берутся правила имеющие более высокий приоритет. Например, предположим, что для платы установлена ширина проводника равная 16, для цепи +5V - 40, для слоя bottom - 20, тогда все цепи на всех слоях отличных от bottom будут разводиться с толщиной 16, на слое bottom все цепи будут разводиться с толщиной 20, а цепь +5V на всех слоях, включая bottom будет разводиться с толщиной 40. Так происходит потому, что приоритет для net более высокий чем для layer и pcb, а для layer более высоки чем для pcb.

Всю эту систему правил можно (и нужно) задать в меню Rules в соответствующих диалоговых окнах.

Итак, прежде чем приступить к трассировке следует задать правила этой трассировки. После этого можно приступать к автоматической трассировке.

Перед трассировкой рекомендуется указать сетку для проводников и переходных отверстий. Это можно сделать выбрав пункт меню Autoroute, а в нем подпункт Setup... При этом появляется диалоговое окно Routing Setup, в этом окне при нажатии на кнопки "Setup Wire Grid..." и "Setup Via Grid..." можно задать сетки для проводников и переходных отверстий соответственно. В поле "PCB Clearance" можно задать зазор от краев границ трассировки, которые обозначают, в данном случае, контур платы. В поле "PCB Wire Width" задается ширина проводников по умолчанию.

 После выполнения установок параметров трассировки можно приступать к, собственно трассировке. Автоматическая трассировка подразделяется в системе Specctra на три этапа:

  • Предварительная трассировка.
  • Многопроходная автоматическая трассировка
  • Постобработка.

Предварительная трассировка содержит набор специализированных трассировочных алгоритмов, которые выполняют следующие операции:

     
fanout   Делает маленькие отводы от планарных контактных площадок до переходных отверстий. Эта операция позволит трассировщику более эффективно произвести разводку большой платы. Эту операцию имеет смысл проводить для плат с четырьмя и более слоями, для двухслойных печатных плат она практически бесполезна.
Seed Vias   Разрывает одно короткое соединение на два и добавляет переходное отверстие. Перед выполнением данной команды Вы должны определить хотя бы одно переходное отверстие проходящее через все сигнальные слои.
Wirebonds    
Bus Routing Прокладка шин Производит разводку регулярных цепей (обычно это различные шины в цифровых схемах). Регулярными считаются цепи попадающие на одинаковые ножки у нескольких микросхем. Такое взаимное расположение, а также регулярность позволяют применить специальные алгоритмы для прокладки таких цепей.

Многопроходная автоматическая трассировка. Она запускается с помощью команды Route. При этом появляется диалоговое окно, в котором задается количество проходов трассировщика, а также вид трассировки.

Постобработка заключается в проведении технологической обработки готовой печатной платы. Она включает в себя следующие возможности:

Critic   Удаляет лишние изгибы без выполнения перекраивания топологии и дополнительной трассировки.
Filter Routing   Удаляет конфликты трассировки. Если после завершения многих проходов трассировки и подчистки остаются несколько конфликтов, эта опция позволяет удалить конфликты и оставить цепи неразведенными.
Testpoints   Назначает тестовые точки сигнальным цепям. Тестовые точки повышают тестируемость печатной платы.
[Un]miter Corners   Сгладить/восстановить углы. С помощью этой опции изгибы дорожек под 90 градусов модифицируются как два по 45 или полукруглые. (т.е. снимаются фаски) А также можно сделать наоборот.

 

Ручная доводка топологии в системе Specctra

Ручная доводка служит для исправления трассировки, выполненной автоматически, а также для трассировки неразведенных цепей. Естественно, ее можно применять и для редактирования (внесения изменений) уже готовых плат, а также для плат, где автоматическая трассировка нежелательна по каким либо причинам.

В режиме трассировки на экране видна специализированная панель инструментов.

Передача информации из системы Specctra в систему PasCAD

Сама система Specctra не содержит инструментов для оформления готовой печатной платы и подготовки информации для технологического оборудования. Поэтому после того как все операции по расстановке элементов и трассировке печатных соединений выполнены, необходимо передать эту информацию назад в систему PasCAD для дальнейшего оформления конструкторской документации.

Чтобы передать информацию из Specctra в PasCAD нужно сохранить Ваш в Specctra сессию. Для этого выберите пункт меню File, а в нем подпункт Write, при этом появляется дополнительное меню, в котором нужно выбрать пункт Session. Появляется диалоговое окно, в котором нужно указать имя файла, после чего следует нажать кнопку OK.

После выполнения этих операций можно переходить к PasCAD. Импорт из Specctra возможен только в плату. Поэтому откройте Ваш проект, а в нем откройте разводимую плату. Отметим, что импорт нужно осуществлять именно в ту плату, из которой был осуществлен экспорт. Нажмите на закладку Инструменты, в появившемся наборе инструментов нажмите на такую кнопку:

При этом появляется диалоговое окно с требованием указать входной файл сессии. Здесь нужно указать тот файл, который Вы сохранили из Specctra, после чего нужно нажать кнопку Открыть. В результате осуществляется трансляция файла сессии, а в плату вносятся изменения в соответствии с записанной в сессию информацией.

Ручная расстановка элементов в системе PasCAD

Расстановка элементов в системе Specctra является, по большей части, ручной. Для небольшого количества элементов можно применить встроенную автоматическую расстановку. Однако, следует отметить, что с возрастанием количества элементов, участвующих в автоматической расстановке, время занимаемое расстановкой возрастает в геометрической прогрессии. Поэтому даже на современных мощных компьютерах это может быть совершенно неприемлемо. В каждом конкретном случае (имеется ввиду конкретный компьютер) следует приемлемое количество элементов определять экспериментально.

Ручная расстановка заключается в перетаскивании элементов на поверхность платы и их ориентация. Перетаскивание может осуществляться как отдельных элементов, так и группы элементов. Выделение  и перенос осуществляется как обычно (см. основные операции). Кроме того, компоненты можно переносить и по позиционному обозначению. Для включения этого режима выберите пункт меню Редактирование, а в нем подпункт Перенос по поз. обозначению. При этом появляется диалоговое окно, в котором нужно ввести позиционное обозначение и нажать кнопку ОК. При этом компонент выделяется и прикрепляется к курсору мыши. Чтобы разместить компонент, нужно нажать левую кнопку мыши в месте его размещения. Переносимый компонент можно вращать с помощью кнопок панели инструментов, а также ставить зеркально и переносить на другую сторону платы. Все эти операции выполняются соответствующими кнопками на панели инструментов.

Автоматическая расстановка реализует очень неэффективный по времени выполнения алгоритм перебора позиций, поэтому запускать его нужно только для очень ограниченного набора элементов. Отметим, что важным фактором является также правильный выбор сетки. Очень мелкая сетка ведет к геометрическому возрастанию времени выполнения операции расстановки, а крупная сетка приводит к неэффективному размещению. Критерием размещения является наименьшая длина связей между компонентами. Как правило, рекомендуется следующая схема работы с авторазмещением: выделить и вынести за область платы компоненты, подлежащие размещению (часто это компоненты, относящиеся к одной группе или фрагменту), затем выбирается необходимая сетка размещения. Она равна текущей сетке графического редактора. После этого над компонентами нажимается правая кнопка мыши и появившемся меню выбирают пункт Расставить. Программа просчитывает наилучший способ размещения и расставляет компоненты. Сформированный таким образом фрагмент можно переместить на печатную плату.

Несмотря на наличие во многих САПР средств автоматического размещения, современные разработчики используют ручной способ расстановки элементов, как наиболее качественный и быстрый.

 

Ручная трассировка печатных соединений в системе PasCAD

К сожалению, система PasCAD версии 1.х не предоставляет таких развитых средств редактирования печатных плат, как например, Specctra. Ее возможности ограничиваются системой редактирования печатных соединений, похожей на P-CAD 4.5. К особенностям этой технологии следует отнести отсутствие контроля правил трассировки в процессе ручной прокладки трасс, отсутствие полуавтоматических режимов завершения текущей трассы, отсутствие режима указания наиболее перспективного пути прокладки трасс. Все эти обстоятельства делают ручное редактирование печатных соединений в системе PasCAD лишь самым крайним средством. Во всех случаях, когда это возможно, рекомендуется использовать для редактирования печатных соединений систему Specctra.

Итак, для ручной трассировки печатных соединений в системе PasCAD предусмотрены два режима: режим ввода цепей и режим редактирования цепей. При входе в один из этих режимов, переключение между ними осуществляется по нажатию правой кнопки мыши. В этом их существенное отличие от всех остальных режимов, поскольку во всех режимах при нажатии правой кнопки мыши осуществляется переход к режиму выделения.

Режим ввода цепей служит для прокладки печатных проводников. При выборе этого режима система ожидает указания начала цепи. Цепь должна начинаться на выводе элемента или на уже имеющейся цепи. Начать цепь вне этих двух объектов не возможно. После того как цепь начата, можно с помощью левой кнопки мыши указать ломаную линию этой цепи. Переключение между слоями, изменение ширины цепи и т.п. осуществляется как обычно с помощью панели инструментов. Единственная особенность этого режима заключается в том, что при смене слоя автоматически вставляется переходной отверстие. Для прекращения ввода цепи следует нажать правую кнопку мыши.

Режим редактирования цепей служит для оперативного изменения (редактирования) цепей в печатной плате. Этот режим служит для переноса, излома или удаления цепей (дорожек) печатной платы. Находясь в этом режиме и нажав левую кнопку мыши можно выделить один или несколько сегментов цепей, выделение происходит как обычно. Если удерживая левую кнопку мыши перетаскивать выделенные сегменты, то они будут перемещаться. Нажатие клавиши Delete приводит к удалению выделенных сегментов.
