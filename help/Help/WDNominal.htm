Подбор номинала

Содержание=>Руководства=> Диалоги=>Подбор номинала

В этом диалоговом производится подбор номиналов компонентов. Подбор номиналов представляет собой многофункциональную систему выбора, оборудованную развитой системой фильтрации и отображения.

В центральной части диалога расположена таблица выбора. В первой строке таблицы представлен заголовок, в котором перечислены параметры доступные для данного типа компонентов.

Сразу же под заголовком размещается строка с текущими параметрами компонента. При нажатии левой кнопки в этой строке появляется диалог с выбором исключительно данного параметра. Список параметров формируется из всей таблицы.

Параметр будучи выбранным становится фильтрующим элементом. Содержимое таблицы формируется с учетом этого фильтра.

Последовательность выбора параметров произвольная. Таким образом, можно получить произвольную последовательность фильтрации.

Нажатие в заголовке правой кнопки, очищает данный параметр. Одновременно фильтрация по этому параметру отключается. Нажатие кнопки Очистить очищает все параметры и показывает полную таблицу.

Если нажать левую кнопку мыши на произвольной строке таблице, устанавливаются все параметры в соответствии с этой строкой, давая таким образом значительный выигрыш по скорости выбора компонентов.
