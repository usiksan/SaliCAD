Ручная нумерация

Содержание=>Руководства=> Диалоги=>Ручная нумерация

Это диалоговое окно служит для ручного назначения позиционных обозначений секциям принципиальной схемы.

В строке редактирования «Номер позиционного обозначения» введите порядковый номер позиционного обозначения. Например, для позиционного обозначения DD1.1 порядковым номером является 1.

В строке редактирования «Логический номер секции» введите логический номер секции. Например, для позиционного обозначения DD32.3 логическим номером секции является 3, а для позиционного обозначения R1 логическим номером секции является 0.

Для принятия сформированного позиционного обозначения нажмите ОК. Для отмены формирования позиционного обозначения нажмите Отмена.
