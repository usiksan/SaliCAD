PasCAD Соглашения

Содержание=>Руководства=>Соглашения

В системе PasCAD используются следующие соглашения

По контактным площадкам Приведены соглашения по контактным площадкам. Соответствия типов КП виду.
По использованию слоев Приведены соглашения по использованию (наименованию) слоев. Описано специальное использование имен слоев системой PasCAD и другими системами.
По параметрам Приведены соглашения по параметрам, используемым системой PasCAD.
