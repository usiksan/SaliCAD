Объекты

Содержание=>Руководства=> Меню=>Объекты

Группа меню Объекты состоит из команд, которые вызывают режимы, диалоги и т.п. для взаимодействия с объектами системы. Имеются следующие пункты в зависимости от текущего редактируемого объекта:

  • Режим цепей
  • Угловой размер
  • Шина
  • Диаметральный размер
  • Линейный размер
  • Компонент
  • Заполнение корпуса
  • Вставка идентификатора
  • Имя цепи
  • Текстовый список цепей
  • Создание объекта
  • Задание точки привязки
  • Цоколевка
  • Параметры
  • Расстановка
  • Область платы
  • Полигон
  • Ножка корпуса
  • Размещение компонентов
  • Дорожка
  • Ренумерация
  • Отключение вывода
  • Цепь
  • Контактные площадки
  • Переходное отверстие
  • Размещение
