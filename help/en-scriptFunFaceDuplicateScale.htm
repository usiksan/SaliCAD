faceDuplicateScale

Содержание=>Скрипт 3d=>Функции 3d=>faceDuplicateScale

Prototype

Face faceDuplicateScale( Face src, Float scaleX, Float scaleY, Float radius)

Description

This function scales the surface dimensions along the X and Y axes. Then the resulting surface is shifted along the Z axis by the amount of the radius.  The original surface does not have to be in the XY plane, but the size transformations are carried out by projection on this plane.

Parameters

  • src - initial surface
  • scaleX - axis X scaling factor
  • scaleY - axis Y scaling factor
  • radius - the value of the offset along the Z axis

Example

color1 = selectColor("#800000")
color2 = selectColor("#008000")

c1 = faceCircle( 2, 10, matrix1 )

c2 = faceDuplicateScale( c1, 0.5, 2.0, 1 )

partModel = model( color1, color1, color1, [c1], matrix1 )
partModel = model( color2, color2, color2, [c2], matrix1 )
