ref

Contents=>Schematic script functions=>ref

Prototype

Float ref( String dim, String row )

Description

This function adds a variable to the visual table.

Parameters

  • dim - dimension for value of this variable. For example "Om", "A", "V", "Hz" and so on
  • row - one of standard row for adjusting result value. Row must be one of the values "", "E3", "E6", "E12", "E24", "E48", "E96", "E192". If row equals to "" (i.e. empty string) than adjusting not performed

Example

r = ref( "Om", "E24" )
