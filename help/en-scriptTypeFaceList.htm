Face list type

Content=>Script 3d=>Face list type

A list of surfaces is a set of one or more surfaces.

Sets of surfaces form the basis of models.

A set of surfaces can be made up of individual surfaces, as well as other sets using square brackets. For example:

fl1 = [ face0, face1 ]
fl2 = [ face2, fl1 ]

The fl1 surface set consists of two surfaces: face0 and face1. The fl2 surface set consists of the face2 surface and all the surfaces of the fl1 set.

Important! For a set of surfaces, the first element in square brackets should be a surface.

fl3 = [fl1, face2] #Ошибка!
