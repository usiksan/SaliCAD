vertexOffset

Content=>Script 3d=>Functions 3d=>vertexOffset

Prototype

Vertex vertexOffset( Vertex src, Float  offx, Float offy, Float offz )

Description

This function builds a point in 3d space by shifting the original src point along three axes by the corresponding offNN values.

Parameters

  • src - source vertex
  • offx - X-axis offset
  • offy - Y-axis offset
  • offz - Z-axis offset

Example

vertex1 = vertex( 0.2, 0.1, 0.4 )
vertex2 = vertexOffset( vertex1, 0, 0, 0.5 )
