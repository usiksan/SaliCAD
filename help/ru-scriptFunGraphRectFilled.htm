graphRectFilled

Содержание=>Скрипт 3d=>Функции 3d=>graphRectFilled

Формат вызова

Graph2d graphRectFilled( FloatList [cornerX0, cornerY0], FloatList [cornerX1, cornerY1] )

Описание

Функция строит 2d объект в 3d пространстве. Функции данного класса предназначены для создания комплексных мастеров генерации компонентов, где в одном скрипте генерируются 3d модель и 2d изображение компонента. Для построения обычных 3d моделей использование данных функций не имеет смысла.

Все 2d объекты строятся в плоскости XY с небольшим смещением по осиZ.

Эта функция строит залитый прямоугольник на основании своих параметров

Параметры

  • cornerX0,cornerY0 - координаты первого угла прямоугольника
  • cornerX1, cornerY1 - координаты противоположного (по диагонали) угла прямоугольника

Пример применения

partFlat = graphRectFilled( [0,0], [2,1] )
