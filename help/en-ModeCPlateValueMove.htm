Move value

Contents=>Modes=>Plate modes=>Move value of components

Tool bar:

Menu: Plate|Move value of components

Cursor:

Is used to transfer values of the component in the Plate. Each value has a default location. It is set relative to the component, so when the component is moved or rotated, the value is moved and rotated with it. To change the position of the relative component value, use this mode.

1. Specify the component whose value you want to move

2. Setup value properties

3. Set value location

4. Repeat steps 1-3 until all values are placed

  • You can specify a point by clicking the left mouse button or manually.
  • The termination of the input is done by pressing the right mouse button.

Value properties are set using the text properties bar.

"Smart mode" not present.
