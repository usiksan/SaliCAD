Model type

Content=>Script 3d=>Model type

The model type is a complex object forming a 3d model.

Each model consists of a list of bodies that are displayed one or more times in the position specified by the copy matrix. Each model contains at least one copy matrix. For example, the image of the chip body will contain one copy matrix, and the image of the legs will contain several, according to the number of legs of the chip.

The list of bodies consists of one or more bodies. Each body is a collection of a set of surfaces represented by one common color. Thus, a body is a set of surfaces that are displayed in one color.

Only assignment operation is available with models. Models are also used in function calls. Models can be created, bodies can be added to the list of bodies, and a copy matrix can be added to the list of copy matrices.
