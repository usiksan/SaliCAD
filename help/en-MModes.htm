SaliCAD modes help

Contents=>Modes help

The system has a advanced mode structure that supports a convenient style of operation, in addition, some modes have special Smart modes that radically simplify input and editing at all stages.

The system supports the following classes of modes:

  • View change modes
  • Draw modes
  • Complex select and edit mode
  • Symbol specific modes
  • Part specific modes
  • Schematic sheet specific modes
  • PCB specific modes

For each mode, the method of entering the mode is specified (it can be a menu or a toolbar), the cursor that operates in this mode, a reasonable mode that operates in this mode.
