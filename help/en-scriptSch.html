Schematic script

Contents=>Schematic script

The schema script is used to perform calculations directly in the schema. The values obtained during the calculation can be used to configure components, which, in turn, allows you to automatically adjust the scheme to the calculated values.

r = ref( "Om", "" )
u = ref( "V", "" )
i = ref( "A", "" )
i = u / r

In this case, the table of values will take the form:

Name Value
r
u
i

If you fill in the r and u fields with values, the value of the i field will be calculated automatically:

Name Value
r 1.2kOm
u 1.24V
i 1.03mA

The ref function adds a variable to the visual table. Variables without this function do not get into the visual table, respectively, it is impossible to see their value. Such variables can be used for intermediate values in the calculation.
