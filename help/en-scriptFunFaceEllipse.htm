faceEllipse

Content=>Script 3d=>Functions 3d=>faceEllipse

Prototype

Face faceEllipse( Float radiusX, Float radiusY, Float stepDegree, Matrix transfer )

Description

This function constructs a surface in the form of an equilateral polygon inscribed in an ellipse. The number of sides of the polygon is determined by the angle pitch. The resulting polygon can be transformed using a matrix. This function is used to construct approximated ellipses. The smaller the angle step, the higher the approximation quality.

Parameters

  • radiusX - the radius X of the described ellipse around the polygon
  • radiusY - the radius Y of the described ellipse around the polygon
  • stepDegree - angle pitch in degrees
  • transfer - final polygon transformation

Example

color1 = selectColor("#800000")

c1 = faceEllipse( 3, 2, 10, matrix1 )

partModel = model( color1, color1, color1, [c1], matrix1 )
