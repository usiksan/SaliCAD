Диалог Печать

С помощью данного диалогового окна можно установить параметры печати и распечатать содержимое текущего редактора.

В поле Принтер представлено название принтера, на который будет осуществлена печать. Нажатие кнопки Опции открывает диалоговое окно, в котором можно настроить параметры текущего принтера или выбрать другой принтер из списка доступных. При установке соответствующего драйвера возможна печать в pdf файл.

Группа параметров Управление печатью позволяет задать точный масштаб вывода рисунка на печать или растянуть изображение на весь лист. По умолчанию изображение растягивается на весь лист.

Группа Ориентация определяет вертикальное или горизонтальное расположение рисунка. По умолчанию выбирается оринетация для масксимального заполнения листа.

Группа Область печати позволяет выбрать прямоугольную область рисунка, отправляемую на печать. По умолчанию распечатывается весь рисунок. Если на рисунке выделена какая-то часть, то можно распечатать только выделенный фрагмент. Также можно указать координаты окна, содержимое которого будет распечатано. При нажатии кнопки Выбрать диалоговое окно печати скрывается и включается режим, с помощью которого это окно можно указать прямо на рисунке в диалоговом режиме.

Группа остальных параметров содержит:

  • количество копий - здесь указано количество копий, которое будет распечатано
  • толщина нулевой линии - на рисунках схем и плат некоторые линии имеют толщину равную нулю. При распечатке толщина таких линий принимается равной размеру одной точки принтера. Для принтеров, имеющих высокое разрешение такие линии могут быть слишком тонкими. В данном поле можно указать толщину для таких линий в пикселях (точках) принтера
  • цветная печать - по умолчанию рисунок печатается черно-белым. Т.е. любые элементы печатаются черным, а фон остается белым, даже если принтер позволяет цветную печать. Установив этот флаг можно получить цветную распечатку на цветном принтере
