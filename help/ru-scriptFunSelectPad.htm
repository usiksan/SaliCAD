selectPad

Содержание=>Скрипт 3d=>Функции 3d=>selectPad

Формат вызова

String selectPad( String padString )

Описание

Эта функция служит для диалогового построения контактной площадки. Если курсор находится на этой функции, то нажатие кнопки F2 вызывает показ диалогового окна мастера контактной площадки. После конструирования площадки через диалог, ее текстовое описание заменяется в тексте скрипта.

Параметры

  • padString - текстовое представление контактной площадки

Пример применения

pin1Pad = selectPad( "c1d0.4m0.1" )
