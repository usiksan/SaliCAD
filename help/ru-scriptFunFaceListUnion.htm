faceListUnion

Содержание=>Скрипт 3d=>Функции 3d=>faceListUnion

Формат вызова

FaceList faceListUnion( FaceList src, FaceList appended )

Описание

Эта функция возвращает объединенный список поверхностей, состоящий из поверхностей src и appended.

Параметры

  • src - исходный список поверхностей
  • appended - добавляемый список поверхностей

Пример применения

color1 = selectColor("#a00000")

c1 = faceListCylinder( 2, 5 )
c2 = faceListDuplicate( c1, matrixTranslate( matrix1, 5, 0, 0 ) )
c3 = faceListUnion( c1, c2 )
partModel = model( color1, color1, color1, c3, matrix1 )
