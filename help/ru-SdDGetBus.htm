Диалог Ввод цепей шины

В этом диалоговом окне задается список цепей для ускоренного формирования шины. Список цепей можно вводить в строке редактирования или выбрать из предлагаемого списка. В этом списке в порядке использования отображаются ранее вводимые списки. Можно также выбрать список цепей из предлагаемого списка и затем отредактировать.

Формат списка имеет следующий вид:

Цепь1[,...]

где Цепь1 представляет собой имя цепи (при вводе имени цепи через это диалоговое окно наличие запятой в имени цепи не допускается, поскольку запятая служит для разделения имен цепей в списке). Если вводимые цепи имеют единое буквенное обозначение, но различаются индексом, то для таких цепей можно применять укороченное обозначение:

общие_буквы_имени<начальный_индекс:конечный_индекс>

Например:

D<0:3>,+5V,Земля,A<0:2> сформируют в результате следующие цепи

D0
D1
D2
D3
+5V
Земля
A0
A1
A2

В правой части диалога представлен список всех именованных цепей, т.е. тех цепей, имена которых вводились вручную. Цепи с автоматическими именами в этом списке не представлены. При выборе цепи из данного списка она автоматически помещается в список шины.
