faceBevelXY

Содержание=>Скрипт 3d=>Функции 3d=>faceBevelXY

Формат вызова

Face faceBevelXY( Face src, Float radius )

Описание

Эта функция строит поверхность в плоскости XY по исходной поверхности также в плоскости XY. Новая поверхность смещается по оси Z на величину радиуса, кроме того она масштабируется по осям X и Y на величину радиуса. Построив стены между исходной и результирующей поверхностями можно получить фаску. Такжу есть функция, которая генерирует скругление между этими двумя поверхностями.

Параметры

  • src - исходная поверхность в плоскости XY
  • radius - радиус скругления или размер фаски

Пример применения

#Пример для фаски
color1 = selectColor("#800000")
color2 = selectColor("#008000")

c1 = faceCircle( 2, 10, matrix1 )

c2 = faceBevelXY( c1, -0.5 )

partModel = model( color1, color1, color1, [c1], matrix1 )
partModel = model( color2, color2, color2, [c2], matrix1 )

Данный пример сформирует следующее изображение:


#Построение фаски
color1 = selectColor("#800000")

c1 = faceCircle( 2, 10, matrix1 )

c2 = faceBevelXY( c1, -0.5 )

partModel = model( color1, color1, color1, faceListWall( c1, c2, true ), matrix1 )

Этот пример показывает формирование стен в виде фаски между двумя поверхностями
