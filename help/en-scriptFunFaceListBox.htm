faceListBox

Content=>Script 3d=>Functions 3d=>faceListBox

Prototype

FaceList faceListBox( Float length, Float width, Float height )

Description

This function builds a parallelepiped with a base on the XY plane.

Parameters

  • length - parallelepiped length (X-axis)
  • width - width of the parallelepiped (Y axis)
  • height - height of the parallelepiped (Z axis)

Example

color1 = selectColor("#a00000")

c1 = faceListBox( 2, 3, -4 )

partModel = model( color1, color1, color1, c1, matrix1 )
