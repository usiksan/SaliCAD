stringPadRectThrough

Content=>Script 3d=>Functions 3d=>stringPadRectThrough

Prototype

String stringPadRectThrough( Float widht, Float height, Float diameter, Float mask )

Description

The function builds a text description of the contact pad specified by the parameters.

Parameters

  • width - the width of the pin pad in mm
  • height - the height of the contact pad in mm
  • diameter - pin pad hole diameter
  • mask - additional mask distance. The mask window increases by this value in relation to the pin pad

Example

  pin = stringPadRectThrough( 2, 1, 0.6, 0.1 )

This example will form the following description of the pin pad "r2x1d0.6m0.1"
