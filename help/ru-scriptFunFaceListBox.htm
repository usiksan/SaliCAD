faceListBox

Содержание=>Скрипт 3d=>Функции 3d=>faceListBox

Формат вызова

FaceList faceListBox( Float lenght, Float width, Float height )

Описание

Эта функция строит параллелепипед с основанием на плоскости XY.

Параметры

  • lenght - длина параллелепипеда (ось X)
  • width - ширина параллелепипеда (ось Y)
  • height - высота параллелепипеда (ось Z)

Пример применения

color1 = selectColor("#a00000")

c1 = faceListBox( 2, 3, -4 )

partModel = model( color1, color1, color1, c1, matrix1 )
