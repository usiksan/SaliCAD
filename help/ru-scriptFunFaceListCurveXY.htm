faceListCurveXY

Содержание=>Скрипт 3d=>Функции 3d=>faceListCurveXY

Формат вызова

FaceList faceListCurveXY( Face face, Float radius, Float angleSrc, Float angleDst, Float sideCount, Bool close )

Описание

Cтроит стены между повернутыми поверхностями, служит для формирования загиба, например вывода. Берет исходную поверхность и поворачивает ее на заданный угол angleDst. Между этими поверхностями формирует промежуточные повернутые поверхности, их количество определяется sideCount. Между всеми сформированными поверхностями строит стены и возвращает их набор.

Параметры

  • face - исходная поверхность
  • radius - радиус поворота
  • angleSrc - угол поворота исходной поверхности относительно плоскости XY
  • angleDst - угол, на который нужно повернуть исходную поверхность
  • sideCount - количество промежуточных поверхностей между начальной и конечной, включая конечную

Пример применения

color1 = selectColor("#a00000")

c1 = faceRect( 2, 3, matrix1 )

c2 = faceListCurveXZ( c1, 5, 0, 30, 10, true )

partModel = model( color1, color1, color1, c2, matrix1 )
