Parameter editor dialog

This dialog box provides a list of the options for the wrapper. You can add, remove, and modify parameters.

In the Central part of the dialog a table of parameters is presented. In the left column - the name of the parameter, and in the right - the value of the parameter. To change the value of the parameter, edit the corresponding cell.

Add param - when you click this button, you are prompted to enter the parameter name, after which the parameter is added to the list.

Add defaults - clicking this button displays a dialog with a list of default parameters. Select the desired options and when you click OK they will be added to the list.

Copy param - copy parameters from another object. Clicking this button displays the object search dialog. When you select an object and click OK, the parameters from it are copied to the current object, replacing all the original parameters.

Delete param - clicking this button deletes the currently selected parameter from the list.

Select value - allows you to select a value from a series. Series, minimum and maximum values are set by parameters. These options are listed by default.
