Диалог Регистрация

Регистрация требуется для различения пользователей и авторов компонентов для доступа к удаленной библиотке и решения проблем синхронизации. См. удаленная библиотека.

Каждый пользователь системы SaliCAD должен зарегистрироваться в центральной библиотеке, чтобы его имя не пересекалось с именами других пользователей системы.

Каждый пользователь может иметь пять рабочих машин, т.е. один и тот-же человек может работать на пяти различных компьютерах, в том числе одновременно.

При регистрации каждому пользователю система формирует индивидуальный ключ для доступа к центральной библиотеке. На каждый компьютер формируется индивидуальный ключ.

С помощью данного диалогового окна можно зарегистрироваться в центральной библиотеке, а также зарегистрировать дополнительные компьютеры для конкретного пользователя.

Чтобы зарегистрировать пользователя нужно заполнить поля:

  • IP сервера SaliCAD - IP адрес сервера центральной библиотеки.
  • Индивидуальной имя - имя, под которым пользователь будет зарегистрирован в центральной библиотке. Это имя также будет записываться в поле автора для всех создаваемых пользователем объектов.
  • EMail - электронная почта для возможности восстановления ключа.

Поле Ключ компьютера оставьте пустым. При успешной регистрации здесь появится ключ, назначенный центральной библиотекой.

В поле Оставшиеся элементы показывается количество элементов, доступных для скачивания из центральной библиотеки.

Для регистрации нажмите кнопку Регистрация. Об успешной регистрации свидетельствует появление ключа в поле Ключ компьютера.

Чтобы зарегистрировать дополнительный компьютер для существующего пользователя нужно заполнить следующие поля:

  • IP сервера SaliCAD - IP адрес сервера центральной библиотеки.
  • Индивидуальной имя - имя, под которым пользователь будет зарегистрирован в центральной библиотке. Это имя также будет записываться в поле автора для всех создаваемых пользователем объектов.
  • EMail - электронная почта для возможности восстановления ключа.
  • Ключ компьютера - внесите сюда ключ любого зарегистрированного компьютера данного пользователя. При успешной регистрации этот ключ заменится на новый, выданный системой.

Для регистрации компьютера нажмите кнопку Добавить компьютер. При успешной регистрации в поле ключа появится новый ключ для данного компьютера.
