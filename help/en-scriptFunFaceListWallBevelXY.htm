faceListWallBevelXY

Content=>Script 3d=>Functions 3d=>faceListWallBevelXY

Prototype

FaceList faceListWallBevelXY( Face src, Face dst, Float radius, Float angleStep )

Description

This function builds chamfer walls between src and dst surfaces. The number of points in the source surfaces must be the same. It is recommended to get the dst surface using the faceBevelXY function. Different combinations of the sign in the faceBevelXY and faceListWallBevelXY parameters allow you to get different chamfer shapes.

Parameters

  • src - source surface
  • dst - the target surface obtained using the faceBevelXY function
  • radius - chamfer radius
  • angleStep - the size of the angle step for approximating the curve of the chamfer surface

Example

color1 = selectColor("#a00000")

c1 = faceCircleSide( 3, 56, matrix1 )
c2 = faceBevelXY( c1, 1 )

h1 = faceListWallBevelXY( c1, c2, 1, 2 )

partModel = model( color1, color1, color1, h1, matrix1 )
