Tool bar:

The SaliCAD system supports a descriptive system of pads. This means that for each conclusion of the case you can specify a description for the contact pads. Based on the description of the pad, the system builds an image of the pad. In some cases, you need to replace the default pad image with another one. For these purposes, is the Pads Association. For each pad description can be set any image of the pad in the plate design process. This mode opens the editor of these associations.
